# คำอธิบายและวิธีทำ
เราต้องการหาค่าของ $a - b$ เมื่อ $a$ คือสัมประสิทธิ์ของ $xy$ และ $b$ คือสัมประสิทธิ์ของ $y$ จากการกระจาย $(2x - y + 6)^2$

สูตรการกระจายกำลังสองของสามพจน์คือ:
$$(u + v + w)^2 = u^2 + v^2 + w^2 + 2uv + 2vw + 2wu$$

ให้:
- $u = 2x$
- $v = -y$
- $w = 6$

แทนค่าลงในสูตรการกระจาย:
$$(2x - y + 6)^2 = (2x)^2 + (-y)^2 + 6^2 + 2(2x)(-y) + 2(-y)(6) + 2(6)(2x)$$
$$= 4x^2 + y^2 + 36 - 4xy - 12y + 24x$$

จากผลการกระจาย:
1. พจน์ที่มี $xy$ คือ $-4xy$ ดังนั้น สัมประสิทธิ์ของ $xy$ คือ $a = -4$
2. พจน์ที่มี $y$ คือ $-12y$ ดังนั้น สัมประสิทธิ์ของ $y$ คือ $b = -12$

หาค่า $a - b$:
$$a - b = -4 - (-12)$$
$$a - b = -4 + 12 = 8$$

ดังนั้น ค่าของ $a - b$ คือ $8$
