# คำอธิบายและวิธีทำ

เมื่อพิจารณาการเคลื่อนที่ครบรอบ (1 รอบพอดี) บนลู่วิ่ง:

1. **ระยะทางการเคลื่อนที่ (Distance):**
   - คือ ความยาวตามแนวเส้นทางการเคลื่อนที่จริงทั้งหมดของวัตถุ
   - เนื่องจากโจทย์ระบุว่าการวิ่งในแต่ละรอบได้ระยะทาง $400\text{ เมตร}$ ดังนั้น ระยะทางการเคลื่อนที่ในแต่ละรอบจึงเท่ากับ $400\text{ เมตร}$

2. **การกระจัด (Displacement):**
   - คือ เวกเตอร์ที่ชี้จากตำแหน่งเริ่มต้นไปยังตำแหน่งสุดท้าย ซึ่งมีขนาดเท่ากับระยะห่างที่สั้นที่สุดระหว่างสองตำแหน่งนั้น
   - เนื่องจากการวิ่งครบรอบ (1 รอบพอดี) ตำแหน่งเริ่มต้นและตำแหน่งสุดท้ายคือ **จุดเดียวกัน** (ไม่มีการเปลี่ยนตำแหน่งสัมพัทธ์ระหว่างจุดเริ่มต้นกับจุดสิ้นสุด)
   - ดังนั้น ขนาดของการกระจัดจึงเท่ากับ $0\text{ เมตร}$

ดังนั้น ค่าที่ถูกต้องคือ:
- ระยะทางการเคลื่อนที่ = $400\text{ เมตร}$
- การกระจัด = $0\text{ เมตร}$

ซึ่งสอดคล้องกับตัวเลือก **4**
