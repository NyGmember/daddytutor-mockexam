# คำอธิบายและวิธีทำ
จากโจทย์ กำหนดให้ค่าของจำนวนในช่องสี่เหลี่ยมแต่ละช่องมีค่าเพิ่มขึ้นครั้งละเท่า ๆ กัน ซึ่งลักษณะเช่นนี้เรียกว่า **ลำดับเลขคณิต (Arithmetic Progression)**

ให้:
* $d$ แทนจำนวนที่เพิ่มขึ้นในแต่ละขั้นตอน (ผลต่างร่วม)
* $x_1, x_2, x_3, \dots$ แทนจำนวนในช่องสี่เหลี่ยมตามลำดับจากจุดเริ่มต้น

จากแผนภาพลำดับขั้นการเพิ่มขึ้น:
1. ช่องที่ 1: $x_1 = a$
2. ช่องที่ 2: $x_2$
3. ช่องที่ 3: $x_3$
4. ช่องที่ 4: $x_4 = -5$
5. ช่องที่ 5: $x_5 = -\frac{2}{3}$
6. ช่องที่ 6: $x_6$
7. ช่องที่ 7: $x_7$
8. ช่องที่ 8: $x_8 = \frac{37}{3}$
9. ช่องที่ 9: $x_9$
10. ช่องที่ 10: $x_{10} = b$

**ขั้นตอนที่ 1: หาค่าของผลต่างร่วม ($d$)**
จากช่องที่ 4 ($x_4 = -5$) ไปยังช่องที่ 5 ($x_5 = -\frac{2}{3}$) เป็นการเพิ่มขึ้น 1 ขั้นตอน ดังนั้น:
$$d = x_5 - x_4$$
$$d = -\frac{2}{3} - (-5)$$
$$d = -\frac{2}{3} + 5 = -\frac{2}{3} + \frac{15}{3}$$
$$d = \frac{13}{3}$$

*หมายเหตุเพื่อความแน่ใจ (ตรวจสอบ):*
จากช่องที่ 5 ($x_5 = -\frac{2}{3}$) ไปยังช่องที่ 8 ($x_8 = \frac{37}{3}$) เป็นการเพิ่มขึ้น 3 ขั้นตอน ($x_8 = x_5 + 3d$) 
$$x_8 - x_5 = \frac{37}{3} - \left(-\frac{2}{3}\right) = \frac{39}{3} = 13$$
ซึ่งเมื่อคำนวณจากผลต่างร่วมที่ได้:
$$3d = 3 \times \frac{13}{3} = 13$$
*(ค่าที่ได้ตรงกัน แสดงว่าความห่างของขั้นตอนถูกต้อง)*

**ขั้นตอนที่ 2: หาค่าของ $a$**
เนื่องจากช่องที่ 4 คือ $x_4 = x_1 + 3d$ และเราทราบว่า $x_1 = a$ และ $x_4 = -5$
$$-5 = a + 3\left(\frac{13}{3}\right)$$
$$-5 = a + 13$$
$$a = -5 - 13$$
$$a = -18$$

**ขั้นตอนที่ 3: หาค่าของ $b$**
เนื่องจากช่องที่ 10 คือ $x_{10} = x_8 + 2d$ และเราทราบว่า $x_8 = \frac{37}{3}$ และ $x_{10} = b$
$$b = \frac{37}{3} + 2\left(\frac{13}{3}\right)$$
$$b = \frac{37}{3} + \frac{26}{3}$$
$$b = \frac{63}{3}$$
$$b = 21$$

**ขั้นตอนที่ 4: คำนวณหาค่าของ $b - a$**
แทนค่า $a = -18$ และ $b = 21$:
$$b - a = 21 - (-18)$$
$$b - a = 21 + 18$$
$$b - a = 39$$

ดังนั้น ค่าของ $b - a$ คือ 39
