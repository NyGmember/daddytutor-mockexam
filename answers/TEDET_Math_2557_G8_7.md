# คำอธิบายและวิธีทำ
กำหนดระบบสมการเชิงเส้น:
1)  $$4x - 5y = -12$$
2)  $$5x + 3y = 22$$

นำสมการที่ 1) คูณด้วย 3:
3)  $$12x - 15y = -36$$

นำสมการที่ 2) คูณด้วย 5:
4)  $$25x + 15y = 110$$

นำสมการที่ 3) และ 4) มาบวกกันเพื่อกำจัดตัวแปร $y$:
$$(12x - 15y) + (25x + 15y) = -36 + 110$$
$$37x = 74$$
$$x = 2$$

ดังนั้น $a = 2$

แทนค่า $x = 2$ ลงในสมการที่ 2):
$$5(2) + 3y = 22$$
$$10 + 3y = 22$$
$$3y = 12$$
$$y = 4$$

ดังนั้น $b = 4$

หาค่าของ $a \times b$:
$$a \times b = 2 \times 4 = 8$$

ดังนั้น ค่าของ $a \times b$ เท่ากับ $8$
