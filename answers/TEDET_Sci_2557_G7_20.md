# คำอธิบายและวิธีทำ

โครงสร้างชั้นบรรยากาศของโลกแบ่งตามเกณฑ์การเปลี่ยนแปลงอุณหภูมิตามระดับความสูง ได้ดังนี้:

* **ชั้น A (โทรโพสเฟียร์ - Troposphere):** ตั้งแต่ระดับผิวโลกถึงความสูงประมาณ $10\text{ km}$ อุณหภูมิจะ **ลดลง** เมื่อระดับความสูงเพิ่มขึ้น (เนื่องจากได้รับความร้อนสะท้อนแผ่จากผิวโลกเป็นหลัก) ชั้นนี้มีไอน้ำหนาแน่นที่สุด ทำให้เกิดปรากฏการณ์ทางลมฟ้าอากาศ
* **ชั้น B (สตราโตสเฟียร์ - Stratosphere):** ความสูงประมาณ $10\text{ - } 50\text{ km}$ อุณหภูมิจะ **เพิ่มขึ้น** เมื่อความสูงเพิ่มขึ้น เนื่องจากชั้นนี้มี **แก๊สโอโซน** หนาแน่นคอยดูดกลืนรังสีอัลตราไวโอเลต (UV) จากดวงอาทิตย์
* **ชั้น C (มีโซสเฟียร์ - Mesosphere):** ความสูงประมาณ $50\text{ - } 80\text{ km}$ อุณหภูมิจะ **ลดลง** อีกครั้งเมื่อระดับความสูงเพิ่มขึ้น ชั้นนี้มีอุณหภูมิลดลงต่ำสุดได้ถึง $-90^\circ\text{C}$ เนื่องจากอุณหภูมิด้านล่างอุ่นกว่าด้านบน จึงเกิดการพาความร้อน (Convection) ลอยตัวขึ้น แต่เนื่องจากแทบไม่มีไอน้ำอยู่เลย จึงไม่เกิดปรากฏการณ์ลม ฟ้า อากาศ
* **ชั้น D (เทอร์โมสเฟียร์ - Thermosphere):** ความสูงตั้งแต่ $80\text{ - } 600\text{ km}$ อุณหภูมิจะ **เพิ่มขึ้น** อย่างรวดเร็วตามความสูงจนอาจถึงหลักพันองศาเซลเซียส เนื่องจากได้รับรังสีคลื่นสั้นจากดวงอาทิตย์โดยตรงในตอนกลางวัน

---

พิจารณาแต่ละตัวเลือก:
* **ตัวเลือก ① ถูกต้อง:** ชั้น B คือชั้นสตราโตสเฟียร์ ซึ่งเป็นแหล่งสะสมหลักของโอโซน
* **ตัวเลือก ② ถูกต้อง:** ชั้น D คือชั้นเทอร์โมสเฟียร์ ได้รับพลังงานรังสีตรงจากดวงอาทิตย์ ทำให้อุณหภูมิสูงมากในตอนกลางวัน
* **ตัวเลือก ③ ถูกต้อง:** ชั้น C คือชั้นมีโซสเฟียร์ อุณหภูมิที่ลดลงตามความสูงทำให้เกิดการหมุนเวียนและพาความร้อนของอากาศได้ แต่ไม่เกิดเมฆ ฝน หรือพายุ เพราะขาดไอน้ำ
* **ตัวเลือก ④ ไม่ถูกต้อง:** ในชั้น A (โทรโพสเฟียร์) อุณหภูมิของอากาศจะต้อง **ลดลง** (ไม่ใช่เพิ่มขึ้น) เมื่อระดับความสูงเพิ่มขึ้น
* **ตัวเลือก ⑤ ถูกต้อง:** เกณฑ์สากลที่ใช้ในการแบ่งชั้นบรรยากาศออกเป็น 4 ชั้นนี้คือ อัตราการเปลี่ยนแปลงอุณหภูมิตามระดับความสูง (Lapse Rate)

ดังนั้น ข้ออธิบายที่ไม่ถูกต้องคือข้อ **4**
