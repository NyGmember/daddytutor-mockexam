# คำอธิบายและวิธีทำ

ระบุและจำแนกสัญลักษณ์ทางไฟฟ้าของอุปกรณ์ประเภทต่าง ๆ ที่ใช้ในแผนภาพวงจรไฟฟ้า (Schematic Symbols):

1. **ถ่านไฟฉาย (เซลล์ไฟฟ้า - Electric Cell):**
   - สัญลักษณ์มาตรฐานประกอบด้วยเส้นคู่ขนานแนวตั้ง 2 เส้นที่มีความยาวต่างกัน:
     - เส้นยาวและบาง แทน ขั้วบวก ($+$)
     - เส้นสั้นและหนา แทน ขั้วลบ ($-$)
   - ในรูปตัวเลือกคือ สัญลักษณ์:
     ```
     -[|ı]-
     ```

2. **ตัวต้านทาน (Resistor):**
   - สัญลักษณ์มาตรฐานที่ใช้กันทั่วไปคือ เส้นซิกแซก (Zigzag line) เพื่อแสดงความกีดขวางหรือการต้านทานกระแสไฟฟ้า
   - ในรูปตัวเลือกคือ สัญลักษณ์:
     ```
     -\/\/\/-
     ```

3. **แอมมิเตอร์ (Ammeter):**
   - สัญลักษณ์คือ วงกลมที่มีอักษร **A** อยู่ด้านใน

4. **สวิตช์ (Switch):**
   - สัญลักษณ์คือ เส้นตรงที่มีจุดเชื่อมต่อแสดงการเปิดหรือปิดวงจร (ป้ายเฉียงเปิดขึ้นแสดงถึงสวิตช์เปิด/วงจรเปิด)

### สรุปการจับคู่ตามลำดับที่โจทย์กำหนด (ถ่านไฟฉาย &rarr; ตัวต้านทาน):
- **ถ่านไฟฉาย:** รูปซ้ายในตัวเลือก ①
- **ตัวต้านทาน:** รูปขวาในตัวเลือก ①

ซึ่งสอดคล้องกับตัวเลือก **1**
