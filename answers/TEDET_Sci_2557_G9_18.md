# คำอธิบายและวิธีทำ

วิเคราะห์วงจรไฟฟ้าและการไหลของกระแสไฟฟ้าผ่านแอมมิเตอร์แต่ละสาขา:

1. **ลักษณะการต่อวงจรในระบบ:**
   - วงจรนี้เป็น **วงจรขนาน (Parallel Circuit)** โดยแบ่งการไหลออกเป็น 2 สาขาหลัก:
     - **สาขาบน:** มีแอมมิเตอร์ A และหลอดไฟดวงที่ 1 ต่ออนุกรมกัน (ไม่มีสวิตช์ควบคุม แปลว่าจะมีกระแสไหลผ่านเสมอหากต่อครบวงจร)
     - **สาขาล่าง:** มีสวิตช์ไฟ แอมมิเตอร์ B และหลอดไฟดวงที่ 2 ต่ออนุกรมกัน (มีสวิตช์เป็นตัวควบคุมการปิด-เปิดการทำงานของสาขานี้)
   - หลังจากสองสาขานี้ไหลมารวมกันทางขวา กระแสไฟฟ้ารวมจะไหลผ่านแอมมิเตอร์ C ก่อนกลับขั้วบวกของแบตเตอรี่
   - ดังนั้น กระแสไฟฟ้าที่จุดต่าง ๆ คือ:
     - แอมมิเตอร์ A วัดกระแสไฟฟ้าในสาขาบน ($I_A$)
     - แอมมิเตอร์ B วัดกระแสไฟฟ้าในสาขาล่าง ($I_B$)
     - แอมมิเตอร์ C วัดกระแสไฟฟ้ารวมของวงจร ($I_C = I_A + I_B$)

2. **เมื่อยกสวิตช์ขึ้น (เปิดวงจร - Open Circuit):**
   - สาขาล่างจะขาด กระแสไฟฟ้าไม่ไหลผ่านสาขานี้เลย ($I_B = 0$) (ตามตัวเลือก ①)
   - แอมมิเตอร์ C จะวัดกระแสที่เหลือเพียงสาขาเดียว คือ $I_C = I_A + 0 = I_A$ ซึ่งทำให้กระแสที่ C มีค่าลดลงจากตอนปิดสวิตช์ (ตามตัวเลือก ②)
   - สำหรับสาขาบนที่ไม่มีสวิตช์: แรงดันไฟฟ้าคร่อมสาขาบนยังคงเท่ากับแรงดันไฟฟ้าของแบตเตอรี่ ($V$) และหลอดไฟมีความต้านทานเท่าเดิม ($R$)
   - ดังนั้น กระแสไฟฟ้า $I_A = \frac{V}{R}$ จะ **มีค่าคงที่เท่าเดิม** (ไม่ได้เพิ่มขึ้นหรือลดลง)

3. **เมื่อสับสวิตช์ลง (ปิดวงจร - Closed Circuit):**
   - กระแสไฟฟ้าไหลผ่านทั้งสองสาขา เนื่องจากหลอดไฟสองดวงเหมือนกันทุกประการ ($R$ เท่ากัน) และต่อขนานคร่อมขั้วแบตเตอรี่เดียวกัน แรงดันตกคร่อมจึงเท่ากัน
   - ทำให้กระแสไฟฟ้าที่ไหลผ่านสาขาบนและล่างเท่ากันด้วย ($I_A = I_B$) (ตามตัวเลือก ③)
   - กระแสไฟฟ้ารวมที่ผ่านแอมมิเตอร์ C จะเท่ากับ $I_C = I_A + I_B = 2 I_A$ (ตามตัวเลือก ⑤)

### พิจารณาตัวเลือก:
- **ตัวเลือก ① ถูกต้อง:** วงจรเปิดในสาขาล่างทำให้ไม่มีกระแสไฟฟ้าไหลผ่านแอมมิเตอร์ B
- **ตัวเลือก ② ถูกต้อง:** กระแสไฟฟ้ารวมที่ C ลดลงจาก $2I_A$ เหลือเพียง $I_A$
- **ตัวเลือก ③ ถูกต้อง:** กระแสไฟฟ้าในแต่ละสาขาขนานที่มีโหลดเท่ากันจะมีค่าเท่ากัน
- **ตัวเลือก ④ ไม่ถูกต้อง และเป็นคำตอบของข้อนี้:** ในวงจรขนานที่เชื่อมต่อกับแหล่งกำเนิดแรงดันคงที่ การเปิดหรือปิดสวิตช์ในสาขาหนึ่งจะ **ไม่ส่งผลต่อปริมาณกระแสไฟฟ้าในอีกสาขาหนึ่ง** ดังนั้นกระแสที่แอมมิเตอร์ A จะคงเดิม ไม่ได้เพิ่มขึ้น
- **ตัวเลือก ⑤ ถูกต้อง:** เมื่อสับสวิตช์ กระแสรวมที่แอมมิเตอร์ C เป็นผลรวมของสองสาขาที่เท่ากัน จึงมีค่าเป็น 2 เท่าของ A

ดังนั้น ข้อความที่อธิบายเกี่ยวกับวงจรไฟฟ้านี้ได้ไม่ถูกต้องคือ **ถ้ายกสวิตช์ (วงจรเปิด) กระแสไฟฟ้าที่ไหลไปที่แอมมิเตอร์ A จะเพิ่มขึ้น**

ซึ่งสอดคล้องกับตัวเลือก **4**
