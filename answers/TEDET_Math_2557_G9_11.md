# คำอธิบายและวิธีทำ

กำหนดให้:
* ความยาวด้านของสี่เหลี่ยมจัตุรัสรูปแรกคือ $a$ หน่วย (พื้นที่คือ $a^2$ ตารางหน่วย)
* ความยาวด้านของสี่เหลี่ยมจัตุรัสรูปที่สองคือ $b$ หน่วย (พื้นที่คือ $b^2$ ตารางหน่วย)

จากอัตราส่วน $a : b = 3 : 5$
เราสามารถสมมติให้:
* $a = 3k$
* $b = 5k$
เมื่อ $k$ เป็นจำนวนจริงบวก

---

### 1. ตั้งสมการจากผลบวกของพื้นที่:
ผลบวกของพื้นที่ของรูปสี่เหลี่ยมจัตุรัสทั้งสองรูปเท่ากับ 1224 ตารางหน่วย:
$$a^2 + b^2 = 1224$$

แทนค่า $a$ และ $b$ ในรูปของ $k$:
$$(3k)^2 + (5k)^2 = 1224$$
$$9k^2 + 25k^2 = 1224$$
$$34k^2 = 1224$$

---

### 2. แก้หาค่า $k$:
$$k^2 = \frac{1224}{34}$$
$$k^2 = 36$$

เนื่องจากความยาวด้านของสี่เหลี่ยมจัตุรัสต้องเป็นค่าบวก ($k > 0$) จะได้:
$$k = 6$$

---

### 3. หาค่าของ $a + b$:
$$a + b = 3k + 5k = 8k$$
แทนค่า $k = 6$:
$$a + b = 8 \times 6 = 48$$

*(หรือคิดแยกค่าจะได้ $a = 18$ และ $b = 30$ ซึ่งผลบวกคือ $18 + 30 = 48$)*

ดังนั้น ค่าของ $a + b$ คือ **48**
