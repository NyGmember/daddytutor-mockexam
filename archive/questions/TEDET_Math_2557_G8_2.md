2. จากการสำรวจน้ำหนักของนักเรียนจำนวน 20 คน ได้ผลดังตาราง

| น้ำหนัก $x$ (kg) | จำนวนนักเรียน (คน) |
| :---: | :---: |
| $30 \le x < 40$ | 1 |
| $40 \le x < 50$ | 2 |
| $50 \le x < 60$ | 8 |
| $60 \le x < 70$ | $A$ |
| $70 \le x < 80$ | 1 |
| รวม | 20 |

จงหาว่านักเรียนที่หนักตั้งแต่ 60 kg ขึ้นไป คิดเป็นร้อยละเท่าใดของนักเรียนทั้งหมด
