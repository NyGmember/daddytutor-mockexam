# คำถาม

26. จำนวนเต็ม $a, b$ และ $c$ สอดคล้องกับสมการที่กำหนดให้ต่อไปนี้

$$\text{(ก) } (x+1)(x+2)(x+3)(x+4) + 1 = (x^2 + ax + b)^2$$
$$\text{(ข) } \sqrt{21 \times 22 \times 23 \times 24 + 1} = c$$

จงหาค่าของ $a + b + c$

# คำอธิบายและวิธีทำ

### 1. หาค่าของ $a$ และ $b$ จากสมการ (ก):
กำหนดสมการ:
$$(x+1)(x+2)(x+3)(x+4) + 1 = (x^2 + ax + b)^2$$

จับคู่คูณพจน์ในฝั่งซ้ายใหม่เพื่อให้เกิดรูปแบบที่ง่ายต่อการแทนค่า:
* คู่ที่ 1: $(x+1)(x+4) = x^2 + 5x + 4$
* คู่ที่ 2: $(x+2)(x+3) = x^2 + 5x + 6$

แทนกลับลงในสมการ:
$$(x^2 + 5x + 4)(x^2 + 5x + 6) + 1 = (x^2 + ax + b)^2$$

ให้ $y = x^2 + 5x$ จะได้พจน์ทางฝั่งซ้ายจัดรูปใหม่เป็น:
$$(y + 4)(y + 6) + 1 = y^2 + 10y + 24 + 1$$
$$= y^2 + 10y + 25$$
$$= (y + 5)^2$$

แทนค่า $y = x^2 + 5x$ กลับเข้ามา:
$$(x^2 + 5x + 5)^2 = (x^2 + ax + b)^2$$

เปรียบเทียบสัมประสิทธิ์กับ $(x^2 + ax + b)^2$ จะได้:
* $a = 5$
* $b = 5$

---

### 2. หาค่าของ $c$ จากสมการ (ข):
กำหนดสมการ:
$$\sqrt{21 \times 22 \times 23 \times 24 + 1} = c$$

สังเกตว่าเทอมใต้รากมีโครงสร้างเดียวกับฝั่งซ้ายของสมการ (ก) คือผลคูณของจำนวนเต็ม 4 จำนวนที่เรียงติดกันแล้วบวกด้วย 1:
$$n(n+1)(n+2)(n+3) + 1$$

โดยมี $n = 21$ หรือหากเทียบตามรูปแบบในสมการ (ก) ให้ $x = 20$:
* $x + 1 = 21$
* $x + 2 = 22$
* $x + 3 = 23$
* $x + 4 = 24$

จากสมการ (ก) ที่เราพิสูจน์แล้วว่า:
$$(x+1)(x+2)(x+3)(x+4) + 1 = (x^2 + 5x + 5)^2$$

แทนค่า $x = 20$ จะได้:
$$21 \times 22 \times 23 \times 24 + 1 = (20^2 + 5(20) + 5)^2$$
$$= (400 + 100 + 5)^2$$
$$= (505)^2$$

ถอดรากที่สองเพื่อหาค่า $c$:
$$c = \sqrt{(505)^2} = 505$$

---

### 3. คำนวณหาค่า $a + b + c$:
$$a + b + c = 5 + 5 + 505 = 515$$

ดังนั้น ค่าของ $a + b + c$ เท่ากับ **515**
