# คำถาม

26. $a$ และ $b$ เป็นจำนวนนับที่มีสองหลัก ซึ่งลงท้ายด้วยเลขโดดตัวเดียวกัน และให้ $c = a \times b$ ถ้า $c$ เป็นจำนวนนับที่มีสามหลักและเลขโดดในแต่ละหลักเป็นตัวเดียวกัน จงหาค่าของ $c - a - b$

# คำอธิบายและวิธีทำ

เราทราบว่า $a$ และ $b$ เป็นจำนวนนับสองหลักที่ลงท้ายด้วยเลขโดดตัวเดียวกัน ให้เลขโดดตัวสุดท้ายเป็น $k$ ดังนั้น $a = 10x + k$ และ $b = 10y + k$ โดยที่ $x, y \in \{1, 2, \dots, 9\}$ และ $k \in \{0, 1, \dots, 9\}$ ผลคูณ $c = a \times b$ เป็นจำนวนนับสามหลักที่มีเลขโดดซ้ำกันทั้งสามหลัก (เช่น $111, 222, \dots, 999$) ซึ่งสามารถเขียนได้ในรูป $111n$ โดยที่ $n \in \{1, 2, \dots, 9\}$ จะได้ $c = 111n = 3 \times 37 \times n$ เนื่องจาก $37$ เป็นจำนวนเฉพาะ $a$ หรือ $b$ ต้องมีตัวประกอบคือ $37$ เนื่องจาก $a$ และ $b$ เป็นเลขสองหลัก ตัวคูณของ $37$ ที่เป็นไปได้คือ $37 \times 1 = 37, 37 \times 2 = 74$ ถ้าให้ $a = 37$ จะลงท้ายด้วย $7$ ดังนั้น $b$ ต้องลงท้ายด้วย $7$ เช่นกัน เลขสองหลักที่ลงท้ายด้วย $7$ ได้แก่ $17, 27, 37, 47, 57, 67, 77, 87, 97$ เราทดลองหาค่า $c$: ถ้า $a = 37$ และ $b = 17, c = 629$ (ไม่ใช่เลขซ้ำ), ถ้า $a = 37$ และ $b = 27, c = 999$ ซึ่งเป็นเลขสามหลักที่เลขโดดซ้ำกันทั้งหมดคือ $9, 9, 9$ ดังนั้น $a=37, b=27$ หรือ $a=27, b=37$ ตรวจสอบเงื่อนไข: $37$ ลงท้ายด้วย $7$, $27$ ลงท้ายด้วย $7$ ถูกต้อง ค่าของ $c - a - b = 999 - 37 - 27 = 999 - 64 = 935$
