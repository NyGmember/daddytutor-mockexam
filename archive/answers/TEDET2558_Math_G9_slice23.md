# คำถาม

14. ถ้า $x$ และ $y$ เป็นจำนวนนับที่ทำให้ $\sqrt{500-x} - \sqrt{200+y}$ เป็นจำนวนเต็มที่มีค่ามากที่สุดที่เป็นไปได้ จงหาค่าของ $x+y$

# คำอธิบายและวิธีทำ

โจทย์กำหนดให้ $x$ และ $y$ เป็นจำนวนนับ (คือจำนวนเต็มบวก 1, 2, 3, ...) และต้องการให้ค่าของ $\sqrt{500-x} - \sqrt{200+y}$ เป็นจำนวนเต็มที่มีค่ามากที่สุดที่เป็นไปได้ เราจะหาค่าของ $x+y$\n\nขั้นที่ 1: วิเคราะห์เงื่อนไขของนิพจน์\nเพื่อให้ $\sqrt{500-x}$ และ $\sqrt{200+y}$ มีค่าเป็นจำนวนเต็มได้นั้น ตัวเลขใต้เครื่องหมายรากที่สอง ($500-x$ และ $200+y$) จะต้องเป็นจำนวนเต็มบวกที่เป็นกำลังสองสมบูรณ์\n\nขั้นที่ 2: ทำให้ค่า $\sqrt{500-x}$ มีค่ามากที่สุด\nเนื่องจาก $x$ เป็นจำนวนนับ ดังนั้น $x \ge 1$\nดังนั้น $500-x \le 500-1 = 499$\nเราต้องการให้ $500-x$ เป็นกำลังสองสมบูรณ์ที่มากที่สุดแต่ไม่เกิน 499 (หรือ 500 เพื่อความง่ายในการหา)\nลองหาจำนวนกำลังสองสมบูรณ์ที่ใกล้เคียง 500:\n$20^2 = 400$\n$21^2 = 441$\n$22^2 = 484$\n$23^2 = 529$\nกำลังสองสมบูรณ์ที่มากที่สุดที่ไม่เกิน 500 คือ 484\nดังนั้น ให้ $500-x = 484$\nจะได้ $x = 500 - 484 = 16$\nค่า $x=16$ เป็นจำนวนนับ ซึ่งสอดคล้องกับเงื่อนไข\nเมื่อ $x=16$, $\sqrt{500-x} = \sqrt{484} = 22$\n\nขั้นที่ 3: ทำให้ค่า $\sqrt{200+y}$ มีค่าน้อยที่สุด\nเนื่องจาก $y$ เป็นจำนวนนับ ดังนั้น $y \ge 1$\nดังนั้น $200+y \ge 200+1 = 201$\nเราต้องการให้ $200+y$ เป็นกำลังสองสมบูรณ์ที่น้อยที่สุดแต่ไม่น้อยกว่า 201\nลองหาจำนวนกำลังสองสมบูรณ์ที่ใกล้เคียง 201:\n$14^2 = 196$\n$15^2 = 225$\nกำลังสองสมบูรณ์ที่น้อยที่สุดที่ไม่น้อยกว่า 201 คือ 225\nดังนั้น ให้ $200+y = 225$\nจะได้ $y = 225 - 200 = 25$\nค่า $y=25$ เป็นจำนวนนับ ซึ่งสอดคล้องกับเงื่อนไข\nเมื่อ $y=25$, $\sqrt{200+y} = \sqrt{225} = 15$\n\nขั้นที่ 4: คำนวณค่าของนิพจน์และตรวจสอบว่าเป็นค่ามากที่สุดหรือไม่\nเมื่อ $x=16$ และ $y=25$ ค่าของนิพจน์คือ\n$\sqrt{500-x} - \sqrt{200+y} = 22 - 15 = 7$\nเพื่อให้ค่าของนิพจน์ $\sqrt{500-x} - \sqrt{200+y}$ มีค่ามากที่สุดที่เป็นไปได้ เราต้องทำให้ $\sqrt{500-x}$ มีค่ามากที่สุด และ $\sqrt{200+y}$ มีค่าน้อยที่สุด ซึ่งเราได้ทำไปแล้วในขั้นตอนที่ 2 และ 3 ดังนั้น 7 จึงเป็นค่ามากที่สุดที่เป็นไปได้\n\nขั้นที่ 5: หาค่าของ $x+y$\nจากที่คำนวณได้ $x=16$ และ $y=25$\nดังนั้น $x+y = 16+25 = 41$\n
