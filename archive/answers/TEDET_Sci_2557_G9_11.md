# คำถาม

11. ใช้ด้ายผูกกับลูกตุ้มแล้วนำไปแขวนไว้ในตำแหน่งที่ไม่มีแรงต้านทานของอากาศดังภาพ ความสูงของการไปถึงที่จุด D ของลูกตุ้มเท่ากับที่จุด A

[![illustration](../images/TEDET_Sci_2557_G9_11_crop_0.png)]

ข้อใดอธิบายเกี่ยวกับการเคลื่อนที่ของลูกตุ้มนี้ไม่ถูกต้อง  
① พลังงานจลน์ที่จุด B มีมากที่สุด  
② พลังงานจลน์ที่จุด A และจุด D เป็น 0  
③ ลูกตุ้มจากจุด B ไปจุด C พลังงานกลเพิ่มขึ้น  
④ ลูกตุ้มขึ้นจากจุด C ไปจุด D พลังงานศักย์เพิ่มขึ้น  
⑤ ลูกตุ้มลงจากจุด A ไปจุด B พลังงานจลน์เพิ่มขึ้น

# คำอธิบายและวิธีทำ

วิเคราะห์พลังงานของลูกตุ้มที่แกว่งแบบฮาร์มอนิกอย่างง่ายโดยไม่มีความต้านทานของอากาศ (แรงเสียดทานเป็นศูนย์) ตามกฎการอนุรักษ์พลังงานกล (Law of Conservation of Mechanical Energy):

พลังงานกลรวม ($E_{\text{กล}}$) ของระบบจะคงที่ ณ ทุก ๆ ตำแหน่งที่ลูกตุ้มแกว่งผ่าน:
$$E_{\text{กล}} = E_p + E_k = \text{คงที่}$$

โดยที่:
- $E_p$ คือ พลังงานศักย์โน้มถ่วง (ขึ้นกับความสูง $h$ จากตำแหน่งต่ำสุด B)
- $E_k$ คือ พลังงานจลน์ (ขึ้นกับความเร็ว $v$ ของลูกตุ้ม)

### วิเคราะห์พลังงาน ณ ตำแหน่งต่าง ๆ:
- **ที่จุด A และ D (ตำแหน่งสูงสุดของการแกว่ง):**
   - ความสูงมากที่สุด $h_{\text{max}} \rightarrow$ พลังงานศักย์ ($E_p$) มีค่ามากที่สุด
   - ลูกตุ้มหยุดนิ่งชั่วขณะเพื่อเตรียมแกว่งกลับทิศทาง $v = 0 \rightarrow$ พลังงานจลน์มีค่าเป็นศูนย์ ($E_k = 0$) (สอดคล้องกับตัวเลือก ②)
- **ที่จุด B (ตำแหน่งต่ำสุดของการแกว่ง):**
   - ความสูงน้อยที่สุด $h = 0 \rightarrow$ พลังงานศักย์มีค่าน้อยที่สุด ($E_p = 0$)
   - ความเร็วในการเคลื่อนที่ของลูกตุ้มมีค่ามากที่สุด $v_{\text{max}} \rightarrow$ พลังงานจลน์ ($E_k$) มีค่ามากที่สุด (สอดคล้องกับตัวเลือก ①)
- **การแกว่งลง (จาก A &rarr; B):**
   - ความสูงลดลง ความเร็วเพิ่มขึ้น $\rightarrow$ พลังงานศักย์ลดลงและเปลี่ยนไปเป็นพลังงานจลน์ พลังงานจลน์จึงมีค่าเพิ่มขึ้น (สอดคล้องกับตัวเลือก ⑤)
- **การแกว่งขึ้น (จาก B &rarr; C &rarr; D):**
   - ความสูงเพิ่มขึ้น ความเร็วลดลง $\rightarrow$ พลังงานจลน์ลดลงและเปลี่ยนไปเป็นพลังงานศักย์ พลังงานศักย์จึงมีค่าเพิ่มขึ้น (สอดคล้องกับตัวเลือก ④)

### พิจารณาตัวเลือก:
- **ตัวเลือก ① ถูกต้อง:** ที่จุดต่ำสุด B ลูกตุ้มเคลื่อนที่เร็วที่สุด พลังงานจลน์จึงมากที่สุด
- **ตัวเลือก ② ถูกต้อง:** ที่จุดสูงสุด A และ D ลูกตุ้มหยุดนิ่งชั่วขณะก่อนจะแกว่งกลับ พลังงานจลน์จึงเป็น 0
- **ตัวเลือก ③ ไม่ถูกต้อง และเป็นคำตอบของข้อนี้:** เนื่องจากไม่มีแรงต้านทานของอากาศ พลังงานกลรวมของลูกตุ้ม ณ ทุก ๆ จุดตลอดแนวการแกว่ง (รวมถึงขณะแกว่งจาก B ไป C) **จะต้องมีค่าเท่าเดิมคงที่เสมอ** ไม่มีการเพิ่มขึ้นหรือลดลงของพลังงานกลรวม
- **ตัวเลือก ④ ถูกต้อง:** การแกว่งจาก C ขึ้นไป D มีความสูงเพิ่มขึ้น พลังงานศักย์จึงเพิ่มขึ้น
- **ตัวเลือก ⑤ ถูกต้อง:** การแกว่งลงจาก A ไป B มีการเปลี่ยนพลังงานศักย์เป็นพลังงานจลน์ ทำให้ความเร็วและพลังงานจลน์เพิ่มขึ้น

ดังนั้น ข้อความที่อธิบายเกี่ยวกับการเคลื่อนที่ของลูกตุ้มได้ไม่ถูกต้องคือ **ลูกตุ้มจากจุด B ไปจุด C พลังงานกลเพิ่มขึ้น**

ซึ่งสอดคล้องกับตัวเลือก **3**
