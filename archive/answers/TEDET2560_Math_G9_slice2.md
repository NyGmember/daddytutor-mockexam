# คำถาม

2. จำนวนนับ $x$ ที่สอดคล้องกับ $\sqrt{10} < x < \sqrt{50}$ มีทั้งหมดกี่จำนวน

# คำอธิบายและวิธีทำ

เราต้องการหาจำนวนนับ $x$ ทั้งหมดที่สอดคล้องกับอสมการ $\sqrt{10} < x < \sqrt{50}$ โดยที่จำนวนนับคือจำนวนเต็มบวก (1, 2, 3, ...)

ขั้นตอนที่ 1: ประมาณค่าของตัวเลขในรูปกรณฑ์
- เนื่องจาก $3^2 = 9$ และ $4^2 = 16$ จะได้ว่า $\sqrt{9} < \sqrt{10} < \sqrt{16}$ หรือ $3 < \sqrt{10} < 4$ ดังนั้น $\sqrt{10} \approx 3.16$
- เนื่องจาก $7^2 = 49$ และ $8^2 = 64$ จะได้ว่า $\sqrt{49} < \sqrt{50} < \sqrt{64}$ หรือ $7 < \sqrt{50} < 8$ ดังนั้น $\sqrt{50} \approx 7.07$

ขั้นตอนที่ 2: แทนค่าลงในอสมการ
อสมการกลายเป็น $3.16 < x < 7.07$

ขั้นตอนที่ 3: เลือกค่าจำนวนนับ $x$ ที่สอดคล้อง
ค่าของจำนวนนับ $x$ ที่มากกว่า 3.16 และน้อยกว่า 7.07 คือ 4, 5, 6, และ 7

สรุปจำนวนนับ $x$ ได้แก่ 4, 5, 6, 7 ซึ่งมีทั้งหมด 4 จำนวน
