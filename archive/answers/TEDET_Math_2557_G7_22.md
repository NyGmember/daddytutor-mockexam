# คำถาม

22. ยองซูตั้งรหัสลับของบัญชีเงินฝากเป็นจำนวนสี่หลักในรูป $AABB$ (เมื่อ $A$ และ $B$ เป็นเลขโดด) ซึ่งมีค่าเท่ากับผลบวกของสองจำนวนต่อไปนี้
- จำนวนแรกเป็นผลบวกของจำนวนเฉพาะทุกตัวที่มีหนึ่งหลัก
- จำนวนที่สองเป็นพหุคูณของผลคูณของจำนวนเฉพาะทุกตัวที่มีหนึ่งหลัก

จงหาค่าของ $2A + B$

# คำอธิบายและวิธีทำ

1. **หาจำนวนแรก (ผลบวกของจำนวนเฉพาะทุกตัวที่มีหนึ่งหลัก):**
   จำนวนเฉพาะที่มีหนึ่งหลัก ได้แก่ 2, 3, 5 และ 7
   $$\text{จำนวนแรก} = 2 + 3 + 5 + 7 = 17$$

2. **หาจำนวนที่สอง (พหุคูณของผลคูณของจำนวนเฉพาะทุกตัวที่มีหนึ่งหลัก):**
   ผลคูณของจำนวนเฉพาะที่มีหนึ่งหลักคือ:
   $$2 \times 3 \times 5 \times 7 = 210$$
   ดังนั้น จำนวนที่สองจะอยู่ในรูป $210k$ โดยที่ $k$ เป็นจำนวนเต็มบวก

3. **หารหัสลับ AABB:**
   รหัสลับมีค่าเท่ากับผลบวกของสองจำนวนนี้ ดังนั้น:
   $$\text{รหัสลับ} = 210k + 17$$
   เนื่องจากรหัสลับเป็นจำนวนสี่หลักในรูป $AABB$ (โดย $A$ และ $B$ เป็นเลขโดด) เราจึงมี:
   $$1000 \le 210k + 17 \le 9999$$
   $$983 \le 210k \le 9982$$
   $$4.68 \le k \le 47.53$$
   เนื่องจาก $k$ เป็นจำนวนเต็ม จะได้ว่า $k$ มีค่าตั้งแต่ $5$ ถึง $47$

   นอกจากนี้ จำนวนในรูป $AABB$ สามารถเขียนในรูปการกระจายได้เป็น:
   $$AABB = 1100A + 11B = 11(100A + B)$$
   ซึ่งหมายความว่า $AABB$ ต้องหารด้วย 11 ลงตัว
   ดังนั้น $210k + 17$ ต้องหารด้วย 11 ลงตัว:
   $$210k + 17 \equiv 0 \pmod{11}$$
   เนื่องจาก $210 = 19 \times 11 + 1 \equiv 1 \pmod{11}$ และ $17 \equiv 6 \pmod{11}$
   จะได้:
   $$1(k) + 6 \equiv 0 \pmod{11}$$
   $$k \equiv -6 \equiv 5 \pmod{11}$$

   ดังนั้น $k$ ที่เป็นไปได้ในช่วง $5 \le k \le 47$ คือ $5, 16, 27, 38$
   เราจะตรวจสอบแต่ละค่าของ $k$:
   * ถ้า $k = 5$: $210(5) + 17 = 1067$ (ไม่อยู่ในรูป $AABB$)
   * ถ้า $k = 16$: $210(16) + 17 = 3377$ (อยู่ในรูป $AABB$ โดยที่ $A = 3$ และ $B = 7$)
   * ถ้า $k = 27$: $210(27) + 17 = 5687$ (ไม่อยู่ในรูป $AABB$)
   * ถ้า $k = 38$: $210(38) + 17 = 7997$ (ไม่อยู่ในรูป $AABB$ เนื่องจาก $A \neq B$ แต่หลักสิบและหลักหน่วยต้องเหมือนกันเป็น $B$ ซึ่ง $9 \neq 7$)

   ดังนั้น รหัสลับคือ $3377$ ทำให้ได้ $A = 3$ และ $B = 7$

4. **คำนวณหาค่าของ $2A + B$:**
   $$2A + B = 2(3) + 7 = 6 + 7 = 13$$

ดังนั้น ค่าของ $2A + B$ คือ 13
