# คำถาม

20. มีจุด A, B, C, D, E และ F ที่เชื่อมโยงกัน ดังรูป ![illustration](../images/TEDET2559_Math_G9_slice20_crop_0.png) ให้ผลรวมของจำนวนจุดยอดและจุดพื้นฐานของจำนวนจุดยอด 3 จุด ซึ่งเชื่อมโยงกันจุดหนึ่ง ๆ โดยตรง ให้ x เป็นผลรวมของจำนวนจุดยอด A, B, C และ y เป็นผลรวมของจำนวนจุดยอด D, E, F จงหาค่าของ $$\frac{2x^2+4xy+3y^2}{3x^2-4xy+2y^2}$$

# คำอธิบายและวิธีทำ

ขั้นแรก เราต้องหาค่าของ x และ y จากความสัมพันธ์ของจุดยอดในรูป ดังนี้: 1) จุดยอด A เชื่อมต่อกับ B, C, D (รวม 3 จุด), จุดยอด B เชื่อมต่อกับ A, C, E (รวม 3 จุด), จุดยอด C เชื่อมต่อกับ A, B, F (รวม 3 จุด) ดังนั้น x ซึ่งเป็นผลรวมของจำนวนการเชื่อมต่อของ A, B, C คือ x = 3 + 3 + 3 = 9. 2) จุดยอด D เชื่อมต่อกับ A, E, F (รวม 3 จุด), จุดยอด E เชื่อมต่อกับ B, D, F (รวม 3 จุด), จุดยอด F เชื่อมต่อกับ C, D, E (รวม 3 จุด) ดังนั้น y ซึ่งเป็นผลรวมของจำนวนการเชื่อมต่อของ D, E, F คือ y = 3 + 3 + 3 = 9. 3) แทนค่า x = 9 และ y = 9 ลงในนิพจน์ที่กำหนด: \frac{2x^2+4xy+3y^2}{3x^2-4xy+2y^2} = \frac{2(9)^2+4(9)(9)+3(9)^2}{3(9)^2-4(9)(9)+2(9)^2}. ดึงตัวร่วม 9^2 ออกจากทั้งเศษและส่วนจะได้: \frac{9^2(2+4+3)}{9^2(3-4+2)} = \frac{2+4+3}{3-4+2} = \frac{9}{1} = 9
