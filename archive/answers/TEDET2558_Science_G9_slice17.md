# คำถาม

18. A, B, C เคลื่อนย้ายวัตถุ 5 อย่าง โดยแต่ละอย่างมีมวล 5 kg ไปไว้ที่ชั้นวางสูง 3 m A ใช้เวลา 1 นาที B ใช้เวลา 2 นาที และ C ใช้เวลา 3 นาที อัตราส่วนของอัตราการทำงานของทั้งสามคนคือข้อใด $P_A : P_B : P_C$ (1) 1 : 1 : 1 (2) 1 : 2 : 3 (3) 2 : 3 : 6 (4) 3 : 2 : 1 (5) 6 : 3 : 2

# คำอธิบายและวิธีทำ

จากนิยามของกำลัง (Power, P) คือ งาน (Work, W) ที่ทำได้ในหนึ่งหน่วยเวลา (t) โดยสมการคือ $P = \frac{W}{t}$ สำหรับงานที่ใช้ในการเคลื่อนย้ายวัตถุแต่ละครั้ง งานจะมีค่าคงที่เท่ากันสำหรับทั้งสามคนเพราะมวล ($m = 5$ kg) ระยะความสูง ($h = 3$ m) และแรงโน้มถ่วง ($g$) เท่ากัน ดังนั้นงานที่ทำ $W = mgh$ จึงเป็นค่าคงที่

โจทย์กำหนดให้:
เวลาที่ A ใช้ ($t_A$) = 1 นาที
เวลาที่ B ใช้ ($t_B$) = 2 นาที
เวลาที่ C ใช้ ($t_C$) = 3 นาที

อัตราการทำงานของแต่ละคนคือ:
$P_A = \frac{W}{1}$
$P_B = \frac{W}{2}$
$P_C = \frac{W}{3}$

หาอัตราส่วน $P_A : P_B : P_C$:
$P_A : P_B : P_C = \frac{W}{1} : \frac{W}{2} : \frac{W}{3}$
เมื่อหารด้วย $W$ ทั้งหมดจะได้:
$= 1 : \frac{1}{2} : \frac{1}{3}$
นำ 6 (ครน. ของ 1, 2, 3) คูณตลอดเพื่อทำเป็นจำนวนเต็ม:
$= (1 \times 6) : (\frac{1}{2} \times 6) : (\frac{1}{3} \times 6)$
$= 6 : 3 : 2$
