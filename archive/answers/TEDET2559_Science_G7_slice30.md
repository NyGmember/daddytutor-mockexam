# คำอธิบายและวิธีทำ

หลักการทำงานของแถบโลหะคู่ (bimetallic strip) อาศัยสมบัติการขยายตัวด้วยความร้อนที่แตกต่างกันของโลหะสองชนิดที่เชื่อมติดกัน เมื่ออุณหภูมิเปลี่ยนไป โลหะที่ขยายตัวได้มากกว่าจะยืดออกได้ยาวกว่า ทำให้แถบโลหะโค้งงอเข้าหาด้านที่ขยายตัวน้อยกว่า ในภาพ (ก) แสดงระดับการขยายตัวเรียงจากมากไปน้อยคือ ทองเหลือง > ทองแดง > เหล็ก เมื่อทำให้แถบโลหะเย็นตัวลง จะเกิดการหดตัวในลักษณะตรงข้ามกับการขยายตัว โลหะที่ขยายตัวได้มากกว่าก็จะหดตัวได้มากกว่าเช่นกัน ในภาพ (ข) แถบโลหะคู่ A-B โค้งงอเข้าหา A แสดงว่า A หดตัวน้อยกว่า B (A มีสัมประสิทธิ์การขยายตัวน้อยกว่า B) ส่วนแถบ B-C โค้งงอเข้าหา B แสดงว่า B หดตัวน้อยกว่า C (B มีสัมประสิทธิ์การขยายตัวน้อยกว่า C) ดังนั้นลำดับการขยายตัวน้อยไปมากคือ A < B < C จากกราฟ (ก) ลำดับจากน้อยไปมากคือ เหล็ก < ทองแดง < ทองเหลือง ดังนั้น A คือเหล็ก, B คือทองแดง และ C คือทองเหลือง
