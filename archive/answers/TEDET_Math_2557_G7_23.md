# คำอธิบายและวิธีทำ

กำหนดให้ $a = 2^{15}$ และ $b = 5^{10} + 1$
เราต้องการหาจำนวนหลักของผลคูณ $a \times b$ ในระบบเลขฐานสิบ:

1. **เขียนผลคูณ $a \times b$ ในรูปสัญลักษณ์คณิตศาสตร์:**
   $$a \times b = 2^{15} \times (5^{10} + 1)$$
   $$a \times b = (2^{15} \times 5^{10}) + 2^{15}$$

2. **จัดรูปพจน์แรกให้อยู่ในรูปของเลขฐานสิบ ($10^n$):**
   $$2^{15} \times 5^{10} = 2^5 \times 2^{10} \times 5^{10}$$
   $$2^{15} \times 5^{10} = 2^5 \times (2 \times 5)^{10}$$
   $$2^{15} \times 5^{10} = 32 \times 10^{10}$$

3. **คำนวณพจน์ที่สอง ($2^{15}$):**
   $$2^{10} = 1024$$
   $$2^{15} = 2^5 \times 2^{10} = 32 \times 1024 = 32,768$$

4. **หาผลรวม:**
   $$a \times b = (32 \times 10^{10}) + 32,768$$
   $$a \times b = 320,000,000,000 + 32,768 = 320,000,032,768$$

   * $32 \times 10^{10}$ คือ เลข 32 ที่ตามด้วยเลข 0 อีก 10 ตัว (รวมเป็นจำนวน 12 หลัก)
   * เมื่อบวกกับ $32,768$ (ซึ่งเป็นจำนวน 5 หลัก) จะมีผลต่อค่าในหลักหน่วยถึงหลักหมื่นเท่านั้น
   * ดังนั้น จำนวนหลักของผลรวมนี้จึงยังคงเท่ากับ 12 หลัก

ดังนั้น เมื่อแสดงค่าของ $a \times b$ ในระบบตัวเลขฐานสิบ จะได้จำนวนที่มี **12** หลัก
