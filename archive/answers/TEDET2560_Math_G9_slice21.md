# คำถาม

21. นักเรียนสามคน คือ A, B และ C เลือกแผ่นวงกลมคนละแผ่น โดยวงกลมจะแบ่งเป็น 2 ส่วนเท่า ๆ กัน และมีจำนวนเขียนกำกับแต่ละส่วนไว้ ดังรูป ![illustration](../images/TEDET2560_Math_G9_slice21_crop_0.png) นักเรียนทั้งสามคนเล่นเกม โดยแต่ละคนจะหมุนแผ่นวงกลมที่ตนเลือกไว้ แล้วปาลูกดอกไปปักบนแผ่นวงกลมของตน เมื่อปาลูกดอกครบทุกคน ผู้ที่ปาลูกดอกไปปักที่บริเวณซึ่งจำนวนที่เขียนกำกับไว้มีค่ามากที่สุดจะเป็นผู้ชนะ (ทุกคนปาลูกดอกปักแผ่นวงกลม แต่ไม่ปักที่เส้นแบ่งครึ่ง) ข้อใดต่อไปนี้ถูกต้อง
(1) A ได้เปรียบที่สุดในเกมนี้
(2) B ได้เปรียบที่สุดในเกมนี้
(3) C ได้เปรียบที่สุดในเกมนี้
(4) B ได้เปรียบ C ในเกมนี้
(5) เกมนี้ยุติธรรมต่อ A, B และ C

# คำอธิบายและวิธีทำ

เกมนี้ตัดสินผู้ชนะจากค่าของตัวเลขที่ปาลูกดอกลงไปในแผ่นวงกลมที่แบ่งครึ่งเท่ากัน (แต่ละส่วนมีโอกาสเกิด 0.5 หรือ 50%) เราต้องพิจารณาว่าโอกาสที่จะได้ตัวเลขที่ 'มากที่สุด' ของแต่ละคนเป็นอย่างไร: 

1. นักเรียน A: มีตัวเลข 3 และ 8 โอกาสชนะขึ้นอยู่กับการที่คู่แข่งได้ตัวเลขที่น้อยกว่า 8 
2. นักเรียน B: มีตัวเลข 7 และ 4 
3. นักเรียน C: มีตัวเลข 6 และ 5 

พิจารณาผู้ที่ได้ตัวเลขสูงสุดในแต่ละรอบ:
- A จะชนะถ้าได้ 8 (โอกาส 0.5)
- B จะชนะถ้า A ไม่ได้ 8 และ B ได้ 7 (โอกาส 0.5 * 0.5 = 0.25)
- C จะชนะถ้า A ไม่ได้ 8, B ไม่ได้ 7 และ C ได้ 6 (โอกาส 0.5 * 0.5 * 0.5 = 0.125)

ในการเปรียบเทียบความได้เปรียบ เราดูว่าใครมีโอกาสชนะรวมสูงที่สุด:
- โอกาสที่ A จะชนะ = P(A=8) = 0.5
- โอกาสที่ B จะชนะ = P(A=3) * P(B=7) = 0.5 * 0.5 = 0.25
- โอกาสที่ C จะชนะ = P(A=3) * P(B=4) * P(C=6) = 0.5 * 0.5 * 0.5 = 0.125

สรุปได้ว่านักเรียน A มีโอกาสชนะสูงที่สุด (50%) เนื่องจากมีตัวเลขที่มากที่สุดคือ 8 ซึ่งชนะตัวเลขอื่นทั้งหมดในเกม ดังนั้น A จึงได้เปรียบที่สุด
