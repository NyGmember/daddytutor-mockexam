# คำถาม

26. เมืองฮันยองต้องการคัดเลือกพื้นที่สำหรับก่อสร้างศูนย์ฝึกอบรมเยาวชน โดยมีพื้นที่ให้เลือกทั้งหมด 5 แห่ง ซึ่งระบุด้วยตัวอักษร A, B, C, D และ E ในรูปต่อไปนี้

![illustration](../images/TEDET2559_Math_G7_slice26_crop_0.png)

และมีข้อมูลความลาดชันของพื้นที่ในหน่วยองศาดังรูปต่อไปนี้

![illustration](../images/TEDET2559_Math_G7_slice26_crop_1.png)

ความลาดชัน (°)

ในการประเมินคะแนนของแต่ละพื้นที่ จะใช้ระยะห่างจากถนน ระยะห่างจากแม่น้ำ และความลาดชันเป็นเกณฑ์การประเมินตามตารางต่อไปนี้

![illustration](../images/TEDET2559_Math_G7_slice26_crop_2.png)

* วัดระยะห่างที่ใกล้ที่สุดจากถนนหรือแม่น้ำไปถึงจุดกึ่งกลางของรูปสี่เหลี่ยมจัตุรัสแต่ละรูป

![illustration](../images/TEDET2559_Math_G7_slice26_crop_3.png)

พื้นที่ที่มีคะแนนประเมินจากเกณฑ์ทั้ง 3 ข้อ รวมกันสูงสุดจะได้รับเลือกเป็นพื้นที่ก่อสร้าง

จงหาว่าพื้นที่ในข้อใดจะได้รับเลือกเป็นพื้นที่ก่อสร้าง

(1) A
(2) B
(3) C
(4) D
(5) E

# คำอธิบายและวิธีทำ

เราสามารถคำนวณคะแนนของแต่ละพื้นที่ได้โดยดูจากเกณฑ์ระยะห่างจากถนน ระยะห่างจากแม่น้ำ และความลาดชัน ดังนี้: 1. พื้นที่ A: ถนนอยู่ห่าง 2 ช่อง (200-300 ม.) -> ให้คะแนน 2, แม่น้ำอยู่ห่าง 0.5 ช่อง (0-100 ม.) -> ให้คะแนน 0, ความลาดชัน 2 องศา -> ให้คะแนน 4. รวม = 2+0+4 = 6 คะแนน 2. พื้นที่ B: ถนนอยู่ห่าง 1.5 ช่อง (100-200 ม.) -> ให้คะแนน 4, แม่น้ำอยู่ห่าง 2.5 ช่อง (200-300 ม.) -> ให้คะแนน 2, ความลาดชัน 1 องศา -> ให้คะแนน 4. รวม = 4+2+4 = 10 คะแนน 3. พื้นที่ C: ถนนอยู่ห่าง 3.5 ช่อง (300-400 ม.) -> ให้คะแนน 2, แม่น้ำอยู่ห่าง 1.5 ช่อง (100-200 ม.) -> ให้คะแนน 2, ความลาดชัน 2 องศา -> ให้คะแนน 4. รวม = 2+2+4 = 8 คะแนน 4. พื้นที่ D: ถนนอยู่ห่าง 2.5 ช่อง (200-300 ม.) -> ให้คะแนน 2, แม่น้ำอยู่ห่าง 2.5 ช่อง (200-300 ม.) -> ให้คะแนน 2, ความลาดชัน 3 องศา -> ให้คะแนน 2. รวม = 2+2+2 = 6 คะแนน 5. พื้นที่ E: ถนนอยู่ห่าง 2.5 ช่อง (200-300 ม.) -> ให้คะแนน 2, แม่น้ำอยู่ห่าง 3.5 ช่อง (300-400 ม.) -> ให้คะแนน 4, ความลาดชัน 5 องศา -> ให้คะแนน 1. รวม = 2+4+1 = 7 คะแนน ดังนั้น พื้นที่ B มีคะแนนรวมสูงสุด คือ 10 คะแนน
