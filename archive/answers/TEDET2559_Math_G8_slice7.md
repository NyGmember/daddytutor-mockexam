# คำถาม

7. เขียนจำนวนทั้งหมดที่สร้างได้จากเลขโดด 0, 1, 2, 6 เรียงตามลำดับดังนี้ ![illustration](../images/TEDET2559_Math_G8_slice7_crop_0.png) จงหาว่าจำนวนในลำดับที่ 64 คือจำนวนใด

# คำอธิบายและวิธีทำ

ลำดับนี้คือการนำเลขโดด {0, 1, 2, 6} มาเรียงสับเปลี่ยนกัน โดยเลขโดดที่มีให้มี 4 ตัว เราสามารถพิจารณาจำนวนหลักของตัวเลขได้ดังนี้: 1) จำนวน 1 หลัก: ได้แก่ 0, 1, 2, 6 (รวม 4 จำนวน) แต่ปกติเลข 0 ไม่ขึ้นต้น ยกเว้นกรณีที่เป็นจำนวนหลักเดียวโดยเฉพาะ ในที่นี้โจทย์กำหนดมาแล้วว่า 0, 1, 2, 6 ดังนั้นลำดับที่ 1-4 คือ 0, 1, 2, 6. 2) จำนวน 2 หลัก: เลขหลักสิบเป็น {1, 2, 6} (3 ตัว) และหลักหน่วยเป็น {0, 1, 2, 6} (4 ตัว) รวมเป็น 3 × 4 = 12 จำนวน. 3) จำนวน 3 หลัก: เลขหลักร้อยเป็น {1, 2, 6}, หลักสิบเป็น {0, 1, 2, 6}, หลักหน่วยเป็น {0, 1, 2, 6} รวมเป็น 3 × 4 × 4 = 48 จำนวน. เมื่อรวมจำนวนที่มี 1, 2, และ 3 หลัก จะได้ 4 + 12 + 48 = 64 จำนวนพอดี ซึ่งหมายความว่าลำดับที่ 64 คือจำนวนที่มากที่สุดที่เป็นเลข 3 หลัก ที่สร้างจากเลขโดด {0, 1, 2, 6} โดยใช้เลขได้ซ้ำ. การสร้างเลข 3 หลักให้มีค่ามากที่สุด เราต้องเลือกเลขหลักร้อย หลักสิบ และหลักหน่วยให้มากที่สุดเท่าที่จะทำได้ คือ 6, 6, 6 ตามลำดับ. ดังนั้น จำนวนลำดับที่ 64 คือ 666
