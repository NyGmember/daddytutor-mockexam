# คำอธิบายและวิธีทำ

จากเงื่อนไขที่โจทย์กำหนดให้:
$$\frac{4^{a+b}}{2^{5b}} = 32 \quad \text{และ} \quad \frac{9^a}{3^{a+b}} = 27$$

สามารถแยกคิดทีละสมการเพื่อหาค่าของตัวแปร $a$ และ $b$ ได้ดังนี้:

### 1. พิจารณาสมการแรก:
$$\frac{4^{a+b}}{2^{5b}} = 32$$

แปลงฐานของเลขยกกำลังให้เป็นฐาน 2:
* $4 = 2^2 \implies 4^{a+b} = (2^2)^{a+b} = 2^{2(a+b)} = 2^{2a+2b}$
* $32 = 2^5$

แทนค่าลงในสมการ:
$$\frac{2^{2a+2b}}{2^{5b}} = 2^5$$
$$2^{(2a+2b) - 5b} = 2^5$$
$$2^{2a-3b} = 2^5$$

เนื่องจากฐานเท่ากัน จะได้สมการเลขชี้กำลัง:
$$2a - 3b = 5 \quad \text{--- (สมการที่ 1)}$$

---

### 2. พิจารณาสมการที่สอง:
$$\frac{9^a}{3^{a+b}} = 27$$

แปลงฐานของเลขยกกำลังให้เป็นฐาน 3:
* $9 = 3^2 \implies 9^a = (3^2)^a = 3^{2a}$
* $27 = 3^3$

แทนค่าลงในสมการ:
$$\frac{3^{2a}}{3^{a+b}} = 3^3$$
$$3^{2a - (a+b)} = 3^3$$
$$3^{a-b} = 3^3$$

เนื่องจากฐานเท่ากัน จะได้สมการเลขชี้กำลัง:
$$a - b = 3 \quad \text{--- (สมการที่ 2)}$$

---

### 3. แก้ระบบสมการหาค่า $a$ และ $b$:
จาก (สมการที่ 2) จะได้:
$$a = b + 3$$

แทนค่า $a = b + 3$ ลงใน (สมการที่ 1):
$$2(b + 3) - 3b = 5$$
$$2b + 6 - 3b = 5$$
$$-b + 6 = 5$$
$$b = 1$$

แทนค่า $b = 1$ กลับลงใน $a = b + 3$:
$$a = 1 + 3 = 4$$

ตรวจสอบคำตอบใน (สมการที่ 1):
$$2(4) - 3(1) = 8 - 3 = 5$$ (ถูกต้อง)

---

### 4. หาค่าของ $3ab$:
$$3ab = 3 \times 4 \times 1 = 12$$

ดังนั้น ค่าของ $3ab$ คือ **12**
