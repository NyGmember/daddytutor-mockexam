# คำอธิบายและวิธีทำ

จากที่โจทย์กำหนด:
$$x = \frac{\sqrt{3}-1}{\sqrt{3}+1} \quad \text{และ} \quad y = \frac{\sqrt{3}+1}{\sqrt{3}-1}$$

เราต้องการหาค่าของ $2x^2 - xy + 2y^2$
ซึ่งเราสามารถจัดรูปเพื่อคำนวณได้ง่ายขึ้นโดยการหาค่าของ $x + y$ และ $xy$:

---

### 1. หาค่าของ $xy$:
$$xy = \left( \frac{\sqrt{3}-1}{\sqrt{3}+1} \right) \times \left( \frac{\sqrt{3}+1}{\sqrt{3}-1} \right) = 1$$

---

### 2. หาค่าของ $x + y$:
$$x + y = \frac{\sqrt{3}-1}{\sqrt{3}+1} + \frac{\sqrt{3}+1}{\sqrt{3}-1}$$

หาตัวส่วนร่วมคือ $(\sqrt{3}+1)(\sqrt{3}-1) = (\sqrt{3})^2 - 1^2 = 3 - 1 = 2$:
$$x + y = \frac{(\sqrt{3}-1)^2 + (\sqrt{3}+1)^2}{2}$$
$$x + y = \frac{(3 - 2\sqrt{3} + 1) + (3 + 2\sqrt{3} + 1)}{2}$$
$$x + y = \frac{8}{2} = 4$$

---

### 3. หาค่าของ $x^2 + y^2$:
จากความสัมพันธ์กำลังสองสมบูรณ์ $x^2 + y^2 = (x+y)^2 - 2xy$:
$$x^2 + y^2 = 4^2 - 2(1)$$
$$x^2 + y^2 = 16 - 2 = 14$$

---

### 4. คำนวณหาค่าของ $2x^2 - xy + 2y^2$:
จัดรูปพจน์ที่ต้องการหาค่า:
$$2x^2 - xy + 2y^2 = 2(x^2 + y^2) - xy$$
แทนค่า $x^2 + y^2 = 14$ และ $xy = 1$:
$$2(14) - 1 = 28 - 1 = 27$$

ดังนั้น ค่าของ $2x^2 - xy + 2y^2$ คือ **27**
