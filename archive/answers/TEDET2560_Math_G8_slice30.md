# คำถาม

30. ตารางด้านล่างแสดงอุณหภูมิที่จุดเยือกแข็งและจุดเดือดของน้ำในหน่วยองศาเซลเซียสและองศาฟาเรนไฮต์

| อุณหภูมิ | จุดเยือกแข็ง | จุดเดือด |
| :--- | :--- | :--- |
| องศาเซลเซียส (°C) | 0 | 100 |
| องศาฟาเรนไฮต์ (°F) | 32 | 212 |

โดยอุณหภูมิในหน่วยองศาเซลเซียสกับองศาฟาเรนไฮต์มีความสัมพันธ์กันดังแสดงด้วยกราฟต่อไปนี้

![illustration](../images/TEDET2560_Math_G8_slice30_crop_0.png)

ข้อใดต่อไปนี้เป็นอุณหภูมิที่เทอร์โมมิเตอร์อ่านค่าในหน่วยองศาเซลเซียสและองศาฟาเรนไฮต์ได้เป็นค่าเดียวกัน

(1) $-250°C, -250°F$ (2) $-40°C, -40°F$

(3) $60°C, 60°F$ (4) $130°C, 130°F$

(5) $450°C, 450°F$

# คำอธิบายและวิธีทำ

เราสามารถสร้างความสัมพันธ์ระหว่างอุณหภูมิในหน่วยองศาเซลเซียส ($C$) และองศาฟาเรนไฮต์ ($F$) ได้โดยใช้สมการเส้นตรงในรูป $F = mC + c$ จากโจทย์ จุดสองจุดที่ผ่านกราฟคือ $(0, 32)$ และ $(100, 212)$ ขั้นแรกหาความชัน ($m$): $m = \frac{212 - 32}{100 - 0} = \frac{180}{100} = 1.8$ หรือเขียนเป็นเศษส่วนคือ $\frac{9}{5}$ เนื่องจากจุดตัดแกน $y$ คือ $32$ ดังนั้นสมการคือ $F = \frac{9}{5}C + 32$ โจทย์ต้องการหาอุณหภูมิที่ค่าของ $C$ และ $F$ เท่ากัน ให้แทนค่า $F$ ด้วย $C$ ในสมการ: $C = \frac{9}{5}C + 32$ ย้ายข้างเพื่อแก้สมการ: $C - \frac{9}{5}C = 32$ จะได้ $-\frac{4}{5}C = 32$ คูณด้วย $-\frac{5}{4}$ ทั้งสองข้าง: $C = 32 \times (-\frac{5}{4}) = -40$ ดังนั้น ที่อุณหภูมิ $-40^\circ C$ จะมีค่าเท่ากับ $-40^\circ F$
