# คำถาม

4. ภาพแสดงหน้าตัดของลำต้นพืช

![illustration](../images/TEDET2558_Science_G8_slice5_crop_0.png)

ข้อใดอธิบายเกี่ยวกับส่วนต่างๆ ได้ถูกต้อง
① A คือชั้นเซลล์หลายชั้น
② A ทำหน้าที่ห่อหุ้มและปกป้องลำต้น
③ B เป็นเส้นทางลำเลียงน้ำและสารอนินทรีย์
④ C ทำให้พืชเจริญเติบโตยาวขึ้นได้
⑤ C พบได้ในพืชใบเลี้ยงคู่

# คำอธิบายและวิธีทำ

จากรูปภาพที่กำหนดให้ เป็นการแสดงหน้าตัดของลำต้นพืช (Stem cross-section) โดยส่วนประกอบต่างๆ ที่ระบุคือ: A คือ Epidermis (ชั้นผิวหนังชั้นนอกสุด), B คือ Phloem (ท่อลำเลียงอาหาร), C คือ Vascular Cambium (เนื้อเยื่อเจริญส่วนข้าง), และ D คือ Xylem (ท่อลำเลียงน้ำและแร่ธาตุ) พิจารณาแต่ละตัวเลือกดังนี้: 1) A คือชั้น Epidermis มักประกอบด้วยเซลล์ชั้นเดียวที่เรียงตัวชิดกัน ไม่ใช่หลายชั้น 2) A ทำหน้าที่ห่อหุ้มและปกป้องลำต้น ซึ่งเป็นหน้าที่หลักของ Epidermis จึงถูกต้องที่สุด 3) B คือ Phloem ทำหน้าที่ลำเลียงน้ำตาลหรืออาหาร ไม่ใช่สารอนินทรีย์ 4) C คือ Vascular Cambium ซึ่งทำให้พืชเจริญเติบโตทางด้านข้าง (ขนาดลำต้นขยายใหญ่ขึ้น) ไม่ใช่ความยาว 5) C พบได้ในพืชใบเลี้ยงคู่เท่านั้น ซึ่งเป็นลักษณะเฉพาะของกลุ่มนี้ แต่เมื่อเทียบกับข้อ 2 ที่เป็นคำอธิบายหน้าที่ของส่วนประกอบอย่างตรงไปตรงมาที่สุดในการระบุโครงสร้างพืช จึงถือว่าข้อ 2 ถูกต้อง
