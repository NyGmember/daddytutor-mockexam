# คำถาม

1. ถ้า $\sqrt{0.27} = k\sqrt{3}$ แล้ว $100k$ มีค่าเท่าใด

# คำอธิบายและวิธีทำ

เริ่มต้นจากสมการที่โจทย์กำหนดให้คือ $\sqrt{0.27} = k\sqrt{3}$ โดยเราสามารถแปลงทศนิยมให้เป็นเศษส่วนได้ดังนี้: $0.27 = \frac{27}{100}$ ดังนั้นสมการคือ $\sqrt{\frac{27}{100}} = k\sqrt{3}$ โดยอาศัยสมบัติของรากที่สอง $\sqrt{\frac{a}{b}} = \frac{\sqrt{a}}{\sqrt{b}}$ เราจะได้ $\frac{\sqrt{27}}{\sqrt{100}} = k\sqrt{3}$ เนื่องจาก $\sqrt{27} = \sqrt{9 \times 3} = \sqrt{9} \times \sqrt{3} = 3\sqrt{3}$ และ $\sqrt{100} = 10$ สมการจึงกลายเป็น $\frac{3\sqrt{3}}{10} = k\sqrt{3}$ นำ $\sqrt{3}$ หารทั้งสองข้างของสมการ จะได้ $k = \frac{3}{10} = 0.3$ สุดท้ายโจทย์ต้องการหาค่าของ $100k$ ดังนั้น $100k = 100(0.3) = 30$
