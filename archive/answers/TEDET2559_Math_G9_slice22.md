# คำถาม

22. รูปแบบพื้นที่แสดงการไหลของน้ำในระบบทางน้ำและท่อที่เชื่อมต่อกัน
เพื่อให้การไหลของน้ำมีความต่อเนื่อง ศึกษารูปแบบการไหลที่ระบุตำแหน่งต่างๆ
ประกอบด้วยรูปภาพดังนี้
![illustration](../images/TEDET2559_Math_G9_slice22_crop_0.png)
กำหนดให้ปริมาณน้ำ 512 คิว ซึ่งไหลผ่านในพื้นที่ประตู A 
และผ่านไปสู่ท่อและพื้นที่ออกประตู D

# คำอธิบายและวิธีทำ

พิจารณาการไหลของน้ำในระบบตามแผนภาพ โดยกำหนดให้ปริมาณน้ำเข้าที่ประตู A คือ 512 หน่วย ซึ่งน้ำจะไหลผ่านระบบท่อที่แยกออกเป็นสาขาต่างๆ ตามลูกศร เพื่อหาปริมาณน้ำที่ออกประตู D เราต้องวิเคราะห์การแบ่งน้ำในแต่ละจุดแยก ดังนี้: 1. ที่ประตู A น้ำจำนวน 512 ไหลเข้าสู่จุดแยกแรก น้ำจะถูกแบ่งออกเป็น 2 ทางเท่าๆ กัน (ตามสมมาตรของภาพ) คือ 512 / 2 = 256 หน่วยต่อสาย 2. สังเกตจากลูกศรในภาพ: สายหลักที่ผ่านไปทางขวาของระบบจะผ่านจุดแยกย่อยอีกหลายจุด ซึ่งทุกจุดแยกน้ำจะถูกแบ่งครึ่งเสมอ (สมมติฐานตามรูปแบบการไหลในท่อมาตรฐานที่โจทย์มักกำหนด) 3. จากจุดเริ่ม A เข้าสู่ระบบทางหลักด้านล่าง 512 หน่วย ถูกแบ่งขึ้นไปด้านบน 256 และไปทางขวา 256 4. พิจารณาเส้นทางที่ไปสู่ D: น้ำที่ผ่านการแยกตัวมาเรื่อยๆ จนถึงประตู D จะพบว่าปริมาณน้ำที่รวมเข้าสู่ทางออก D มาจากจุดแยกสุดท้ายของระบบขวาบน ตามผังนี้ระบบมีการแบ่งครึ่งปริมาณน้ำรวมทั้งหมด 4 ครั้ง (2^4 = 16) เพื่อให้เข้าสู่ทางออก D ดังนั้นปริมาณน้ำที่ออกประตู D คือ 512 / 2^4 = 512 / 16 = 32 หน่วย
