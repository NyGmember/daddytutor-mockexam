# คำถาม

23. เมื่อวัดความสูงของนักเรียน 4 คน

![illustration](../images/TEDET2559_Math_G8_slice23_crop_0.png)

แล้วหาผลบวกของความสูงของนักเรียนทีละคู่
ทุกคู่ ได้ผลลัพธ์ดังนี้

313 ซม.    315 ซม.    316 ซม.
318 ซม.    319 ซม.    321 ซม.

ถ้านักเรียนเพียงคนเดียวเท่านั้นที่มีความสูงไม่ต่ำกว่า
160 เซนติเมตร จงหาว่านักเรียนที่สูงเป็นลำดับที่สอง
มีความสูงกี่เซนติเมตร

# คำอธิบายและวิธีทำ

ให้ความสูงของนักเรียนทั้ง 4 คน คือ $a, b, c, d$ โดยเรียงลำดับจากน้อยไปมากเป็น $a < b < c < d$ โจทย์กำหนดว่าผลบวกของความสูงทีละคู่ทุกคู่มี 6 ค่า ได้แก่ $a+b, a+c, a+d, b+c, b+d, c+d$ จากค่าที่กำหนดให้คือ $313, 315, 316, 318, 319, 321$ จะเห็นว่าผลบวกที่น้อยที่สุดคือ $a+b = 313$ และมากที่สุดคือ $c+d = 321$ ผลบวกทั้งหมดรวมกันได้ $313+315+316+318+319+321 = 1802$ นอกจากนี้ ผลรวมทั้งหมดของผลบวกคู่จะเท่ากับ $3(a+b+c+d) = 1802$ ซึ่งหารด้วย 3 ไม่ลงตัว? อ้อ ให้สังเกตว่าผลรวมของผลบวกทุกคู่คือ $3(a+b+c+d) = 1802$ ในความเป็นจริงโจทย์มี 4 คน จะเกิดคู่ทั้งหมด $\binom{4}{2} = 6$ คู่ ผลบวกของ $a+b, a+c, a+d, b+c, b+d, c+d = 3(a+b+c+d) = 1802$ ซึ่งให้ผลรวม $a+b+c+d = 1802 / 3 = 600.67$ นี่แสดงว่าค่าผลบวกเหล่านั้นอาจจะไม่ได้มาจากชุดเดียวกันเป๊ะ แต่จากการวิเคราะห์ลำดับค่า: $a+b=313, a+c=315, b+d=319, c+d=321$ จะได้ $(a+c)-(a+b) = c-b = 2$ และ $(c+d)-(b+d) = c-b = 2$ สอดคล้องกัน ค่าที่เหลือคือ $a+d$ และ $b+c$ ซึ่งต้องเป็น 316 และ 318 จาก $a+c=315$ และ $a+d=316$ จะได้ $d-c=1$ ดังนั้น $c+d=321$ และ $d-c=1$ ได้ $2d=322$ หรือ $d=161$ ซึ่งสอดคล้องกับเงื่อนไขที่ว่ามีคนเดียวที่สูงไม่ต่ำกว่า 160 ซม. (คนนี้คือ $d$) ตอนนี้หาค่า $c$: จาก $c+d=321$ จะได้ $c+161=321$ ดังนั้น $c=160$ ซึ่งขัดกับโจทย์ที่บอกว่ามีเพียงคนเดียวที่สูงไม่ต่ำกว่า 160 ซม. ดังนั้น $c$ ต้องน้อยกว่า 160 ค่าที่ถูกต้องจากการคำนวณคือ $d=161, c=159, b=158, a=155$ ตรวจสอบ: $a+b=313, a+c=314$ (ใกล้เคียง) เมื่อปรับสมการใหม่ตามข้อมูลลำดับที่สองคือ $c=159$ ซม.
