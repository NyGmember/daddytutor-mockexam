# คำถาม

16. นักเรียน 8 คน คือ A, B, C, D, E, F, G และ H สอบวิชาคณิตศาสตร์ที่มีคะแนนเต็ม 100 คะแนน และแสดงคะแนนสำหรับบางคนในตารางต่อไปนี้ ![illustration](../images/TEDET2560_Math_G8_slice16_crop_0.png) ถ้าคะแนนเฉลี่ยของนักเรียนทั้ง 8 คน คือ 65 คะแนน โดยที่ F ได้คะแนนสูงสุดและได้คะแนนเป็น 2 เท่าของใครคนใดคนหนึ่งจาก 7 คนที่เหลือ จงหาว่า C ได้กี่คะแนน

# คำอธิบายและวิธีทำ

1. คำนวณผลรวมคะแนนทั้งหมดของนักเรียน 8 คน: จากค่าเฉลี่ย 65 คะแนนต่อคน ผลรวมคะแนนทั้งหมดคือ $65 \times 8 = 520$ คะแนน
2. หาผลรวมคะแนนของนักเรียน 6 คนที่ทราบคะแนน (A, B, D, E, G, H): $75 + 48 + 92 + 35 + 58 + 80 = 388$ คะแนน
3. ให้ผลรวมคะแนนของ C และ F คือ $X = 520 - 388 = 132$ คะแนน
4. โจทย์ระบุว่า F มีคะแนนสูงสุดและเป็น 2 เท่าของนักเรียนอีกคนหนึ่งจาก 7 คนที่เหลือ (A=75, B=48, C, D=92, E=35, G=58, H=80)
5. ลองพิจารณาค่าที่เป็นไปได้ของ F: เนื่องจาก F เป็นคะแนนสูงสุด F ต้องมากกว่า 92 และ F ต้องเป็น 2 เท่าของตัวเลขในกลุ่ม {75, 48, C, 92, 35, 58, 80}
   - ถ้า F = 2 * 75 = 150 (เป็นไปได้ เพราะ F > 92 แต่คะแนนเต็ม 100 ดังนั้น 150 ไม่ได้)
   - ถ้า F = 2 * 48 = 96 (เป็นไปได้ เพราะ F > 92)
   - ถ้า F = 2 * 35 = 70 (ไม่ได้ เพราะ F ต้อง > 92)
   - ถ้า F = 2 * 58 = 116 (ไม่ได้ เพราะคะแนนเต็ม 100)
   - ถ้า F = 2 * 80 = 160 (ไม่ได้ เพราะคะแนนเต็ม 100)
   - กรณี F เป็น 2 เท่าของ C: หาก F = 2C ดังนั้น C + F = C + 2C = 3C = 132 จะได้ C = 44 ซึ่งทำให้ F = 88 (ไม่ได้ เพราะ F ต้องมากกว่า 92)
6. ดังนั้น กรณีที่เป็นไปได้คือ F = 96 (จาก 2 * 48) และจากผลรวม C + F = 132 จะได้ C + 96 = 132 ดังนั้น C = 132 - 96 = 36 คะแนน
