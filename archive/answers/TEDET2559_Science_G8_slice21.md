# คำถาม

17. ภาพแสดงชุดอุปกรณ์แยกสารโซเดียมไฮโดรเจนคาร์บอเนต (ก) และขวดแก้วที่มีเอทานอลเผาไหม้อยู่ด้านใน (ข)

![illustration](../images/TEDET2559_Science_G8_slice21_crop_0.png)

จากกการทดลอง ข้อใดแสดงสูตรทางเคมีของแก๊สที่เกิดขึ้นเหมือนกันทั้งสองการทดลองและเปรียบเทียบน้ำหนักของแก๊สนั้นกับอากาศได้ถูกต้อง

| ชนิดของแก๊ส | เปรียบเทียบกับอากาศ |
| :--- | :--- |
| ① $\text{CO}_2$ | เบากว่า |
| ② $\text{CO}$ | หนักกว่า |
| ③ $\text{CO}_2$ | หนักกว่า |
| ④ $\text{H}_2\text{O}$ | เบากว่า |
| ⑤ $\text{O}_2$ | หนักกว่า |

# คำอธิบายและวิธีทำ

การทดลองทั้งสองชุดผลิตแก๊สชนิดเดียวกัน คือ แก๊สคาร์บอนไดออกไซด์ ($\text{CO}_2$) โดยมีรายละเอียดดังนี้: 1. ชุดที่ (ก) เป็นการสลายตัวด้วยความร้อนของโซเดียมไฮโดรเจนคาร์บอเนต (เบกกิ้งโซดา) ซึ่งมีสมการเคมีคือ $2\text{NaHCO}_3(s) \xrightarrow{\Delta} \text{Na}_2\text{CO}_3(s) + \text{H}_2\text{O}(g) + \text{CO}_2(g)$ ซึ่งจะให้แก๊สคาร์บอนไดออกไซด์ออกมา 2. ชุดที่ (ข) เป็นการเผาไหม้เอทานอล ($\text{C}_2\text{H}_5\text{OH}$) ในอากาศ ซึ่งเป็นการเผาไหม้อย่างสมบูรณ์ มีสมการเคมีคือ $\text{C}_2\text{H}_5\text{OH}(l) + 3\text{O}_2(g) \rightarrow 2\text{CO}_2(g) + 3\text{H}_2\text{O}(g)$ ซึ่งก็ให้แก๊สคาร์บอนไดออกไซด์ออกมาเช่นกัน สำหรับการเปรียบเทียบน้ำหนักกับอากาศ สามารถพิจารณาได้จากมวลโมเลกุล: มวลโมเลกุลของอากาศมีค่าเฉลี่ยประมาณ 29 กรัม/โมล ในขณะที่มวลโมเลกุลของ $\text{CO}_2 = 12 + (2 \times 16) = 44$ กรัม/โมล เนื่องจาก 44 มากกว่า 29 แก๊สคาร์บอนไดออกไซด์จึงมีความหนาแน่นมากกว่าอากาศ หรือกล่าวได้ว่า 'หนักกว่า' อากาศนั่นเอง ดังนั้นตัวเลือกที่ถูกต้องคือข้อ (3)
