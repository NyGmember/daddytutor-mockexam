# คำถาม

19. ถ้าให้ $\langle p, q, r \rangle$ แทนความสัมพันธ์ $p \times q = r$ เมื่อกำหนด $\langle a, b, -10 \rangle$, $\langle a, b + c, 45 \rangle$ และ $\langle a, c, x \rangle$ มาให้ จงหาค่าของ $x$

# คำอธิบายและวิธีทำ

จากโจทย์ กำหนดให้ $\langle p, q, r \rangle$ แทนความสัมพันธ์ $p \times q = r$ ดังนั้นเราสามารถเขียนสมการจากสิ่งที่โจทย์กำหนดให้ได้ดังนี้:\n1. จาก $\langle a, b, -10 \rangle$ จะได้สมการ: $a \times b = -10$ (สมการที่ 1)\n2. จาก $\langle a, b + c, 45 \rangle$ จะได้สมการ: $a \times (b + c) = 45$ (สมการที่ 2)\n3. จาก $\langle a, c, x \rangle$ จะได้สมการ: $a \times c = x$ (เราต้องการหาค่า $x$)\n\nจากสมการที่ 2 เราสามารถกระจายตัวคูณ $a$ เข้าไปในวงเล็บได้ดังนี้:\n$a \times (b + c) = (a \times b) + (a \times c) = 45$\n\nแทนค่าจากสมการที่ 1 ($a \times b = -10$) ลงในสมการข้างต้น จะได้:\n$-10 + (a \times c) = 45$\n\nนำ 10 มาบวกทั้งสองข้างของสมการ:\n$a \times c = 45 + 10$\n$a \times c = 55$\n\nเนื่องจากเรากำหนดให้ $\langle a, c, x \rangle$ คือ $a \times c = x$ ดังนั้น $x = 55$
