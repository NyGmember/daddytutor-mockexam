# คำถาม

8. ถ้า $2\sqrt{0.2} + \frac{12}{\sqrt{80}} + \sqrt{605} = k\sqrt{5}$  
จงหาค่าของ $k$

# คำอธิบายและวิธีทำ

ในการแก้โจทย์นี้ เราต้องจัดรูปพจน์แต่ละพจน์ให้อยู่ในรูปของ $\sqrt{5}$ เพื่อให้สามารถบวกกันได้ โดยใช้สมบัติของเลขยกกำลังและรากที่สอง ดังนี้: 
1. พจน์แรก: $2\sqrt{0.2} = 2\sqrt{\frac{2}{10}} = 2\sqrt{\frac{1}{5}} = 2\frac{1}{\sqrt{5}} = 2\frac{\sqrt{5}}{5} = \frac{2}{5}\sqrt{5}$
2. พจน์ที่สอง: $\frac{12}{\sqrt{80}} = \frac{12}{\sqrt{16 \times 5}} = \frac{12}{4\sqrt{5}} = \frac{3}{\sqrt{5}} = \frac{3\sqrt{5}}{5} = \frac{3}{5}\sqrt{5}$
3. พจน์ที่สาม: $\sqrt{605} = \sqrt{121 \times 5} = 11\sqrt{5}$
เมื่อนำพจน์ทั้งหมดมารวมกัน: $\frac{2}{5}\sqrt{5} + \frac{3}{5}\sqrt{5} + 11\sqrt{5} = (\frac{2}{5} + \frac{3}{5} + 11)\sqrt{5} = (1 + 11)\sqrt{5} = 12\sqrt{5}$ ดังนั้นจะได้ $k\sqrt{5} = 12\sqrt{5}$ ทำให้ $k = 12$
