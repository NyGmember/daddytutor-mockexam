# คำถาม

27. จากรูป จุด $B(10, 10k)$ อยู่บนกราฟ $y = kx$ จุด $A(p, 10)$ ได้จากการหมุนจุด $B$ ด้วยมุม $45^\circ$ ในทิศทวนเข็มนาฬิกา โดยมีจุดกำเนิดเป็นจุดหมุน

[![illustration](../images/TEDET_Math_2557_G8_27_crop_0.png)]

ถ้า $\overline{AB}$ มีความยาวเท่ากับ $ak$ จงหาค่าของ $a$

# คำอธิบายและวิธีทำ

จากรายละเอียดที่โจทย์กำหนดให้:
* จุด $B(10, 10k)$ อยู่บนเส้นตรง $y = kx$
* จุด $A(p, 10)$ เกิดจากการหมุนจุด $B$ รอบจุดกำเนิด $O(0, 0)$ ด้วยมุม $45^\circ$ ในทิศทวนเข็มนาฬิกา
* ความยาวของส่วนของเส้นตรง $\overline{AB}$ เท่ากับ $ak$

---

### 1. วิเคราะห์ด้วยตรีโกณมิติและเวกเตอร์ของจุดหมุน:
ให้มุมที่เส้นตรง $OB$ ทำกับแกน $x$ ในทิศบวกคือ $\theta$
เนื่องจากจุด $B$ มีพิกัดเป็น $(10, 10k)$ ดังนั้น:
* $\tan\theta = \frac{10k}{10} = k$
* ความยาว $OB = \sqrt{10^2 + (10k)^2} = 10\sqrt{1+k^2}$

เนื่องจากการหมุนรอบจุดกำเนิดเป็นการอนุรักษ์ระยะทาง (Distance-preserving transformation) จะได้ว่า:
* ความยาว $OA = OB = 10\sqrt{1+k^2}$
* จุด $A$ เกิดจากการหมุนจุด $B$ ไป $45^\circ$ ดังนั้น มุมที่ $OA$ ทำกับแกน $x$ คือ $\theta + 45^\circ$

---

### 2. หาค่าของ $k$:
จากพิกัดของจุด $A(p, 10)$ จะได้พิกัดแกน $y$ ของ $A$ คือ $10$:
$$y_A = OA \sin(\theta + 45^\circ) = 10$$

จากสูตรการบวกมุมของฟังก์ชันไซน์:
$$\sin(\theta + 45^\circ) = \sin\theta\cos 45^\circ + \cos\theta\sin 45^\circ$$
$$\sin(\theta + 45^\circ) = \frac{\sqrt{2}}{2}(\sin\theta + \cos\theta)$$

แทนค่าลงในสมการ $OA \sin(\theta + 45^\circ) = 10$:
$$OA \left( \frac{\sqrt{2}}{2}(\sin\theta + \cos\theta) \right) = 10$$
$$\frac{\sqrt{2}}{2} (OA \sin\theta + OA \cos\theta) = 10$$

สังเกตว่า:
* $OB \sin\theta = y_B = 10k$
* $OB \cos\theta = x_B = 10$
เนื่องจาก $OA = OB$ จะได้:
$$\frac{\sqrt{2}}{2} (10k + 10) = 10$$
$$5\sqrt{2}(k + 1) = 10$$
$$\sqrt{2}(k + 1) = 2$$
$$k + 1 = \sqrt{2}$$
$$k = \sqrt{2} - 1$$

---

### 3. หาความยาวของ $\overline{AB}$:
ในสามเหลี่ยม $OAB$ เป็นสามเหลี่ยมหน้าจั่วที่มี $OA = OB = R$ และมีมุมยอด $\angle AOB = 45^\circ$
โดยกฎของโคไซน์ (Law of Cosines) ในสามเหลี่ยม $OAB$:
$$AB^2 = OA^2 + OB^2 - 2(OA)(OB)\cos 45^\circ$$
$$AB^2 = R^2 + R^2 - 2R^2\left(\frac{\sqrt{2}}{2}\right) = R^2(2 - \sqrt{2})$$

คำนวณหาค่า $R^2$:
$$R^2 = OB^2 = 100(1+k^2)$$
แทนค่า $k = \sqrt{2} - 1$ จะได้:
$$1+k^2 = 1 + (\sqrt{2}-1)^2 = 1 + (2 - 2\sqrt{2} + 1) = 4 - 2\sqrt{2}$$
ดังนั้น:
$$R^2 = 100(4 - 2\sqrt{2}) = 200(2 - \sqrt{2})$$

แทนค่า $R^2$ ลงในสมการของ $AB^2$:
$$AB^2 = 200(2 - \sqrt{2})(2 - \sqrt{2}) = 200(2 - \sqrt{2})^2$$

ถอดรากที่สองของทั้งสองข้าง (เนื่องจาก $AB > 0$):
$$AB = \sqrt{200}(2 - \sqrt{2}) = 10\sqrt{2}(2 - \sqrt{2})$$
$$AB = 20\sqrt{2} - 20 = 20(\sqrt{2} - 1)$$

เนื่องจาก $k = \sqrt{2} - 1$ จะได้:
$$AB = 20k$$

---

### 4. หาค่าของ $a$:
จากโจทย์ $\overline{AB} = ak$ เมื่อเทียบกับ $AB = 20k$
จะได้:
$$a = 20$$

ดังนั้น ค่าของ $a$ คือ **20**
