# คำอธิบายและวิธีทำ

ในระบบเลขฐานสอง (Binary System) เลขโดดแต่ละหลักจะมีค่าประจำหลักเป็นเลขยกกำลังของ 2 ตามตำแหน่งจากขวาไปซ้าย โดยเริ่มจากตำแหน่งที่ 0 ดังนี้: หลักที่ 6, 5, 4, 3, 2, 1, 0. พิจารณาเลขฐานสอง $1000100_2$:\n1. ตำแหน่งของเลขโดดแต่ละตัวจากขวาไปซ้ายคือ:\n- หลักที่ 0 คือเลข 0 ($2^0$)\n- หลักที่ 1 คือเลข 0 ($2^1$)\n- หลักที่ 2 คือเลข 1 (ตำแหน่ง B, ค่าประจำหลักคือ $2^2 = 4$)\n- หลักที่ 3 คือเลข 0 ($2^3$)\n- หลักที่ 4 คือเลข 0 ($2^4$)\n- หลักที่ 5 คือเลข 0 ($2^5$)\n- หลักที่ 6 คือเลข 1 (ตำแหน่ง A, ค่าประจำหลักคือ $2^6 = 64$)\n\n2. เลขโดด 1 ที่ตำแหน่ง A มีค่าคือ $1 \times 2^6 = 64$\n3. เลขโดด 1 ที่ตำแหน่ง B มีค่าคือ $1 \times 2^2 = 4$\n4. โจทย์ถามว่าเลขโดด 1 ที่ตำแหน่ง A เป็นกี่เท่าของเลขโดด 1 ที่ตำแหน่ง B ดังนั้นคำนวณได้จาก $\frac{64}{4} = 16$.
