# คำถาม

11. จากภาพหน้าตัดของดินที่อุดมสมบูรณ์ ข้อใดอธิบายเกี่ยวกับดินได้ถูกต้อง

[![illustration](../images/TEDET_Sci_2557_G8_11_crop_0.png)]

① A เป็นชั้นที่ถูกสร้างขึ้นโดยการย่อยสลาย B  
② B เป็นชั้นที่ถูกสร้างขึ้นโดยการย่อยสลาย C  
③ D เป็นชั้นที่ถูกสร้างขึ้นหลังสุด  
④ A มีบทบาทสำคัญต่อการเจริญเติบโตของพืช เพราะเป็นชั้นที่มีการทับถมของซากพืชซากสัตว์  
⑤ เนื่องจากเวลาที่ใช้ในการเกิด D ถึง A เป็นช่วงระยะเวลาสั้น จึงไม่เกี่ยวข้องกับการพัดพาหน้าดินหรือการเกิดมลภาวะ

# คำอธิบายและวิธีทำ

พิจารณาโครงสร้างหน้าตัดดิน (Soil Profile) ซึ่งแบ่งออกเป็นชั้นต่าง ๆ จากบนลงล่างดังนี้:

* **ชั้น A (ดินชั้นบน / Topsoil):** เป็นชั้นที่มีการสะสมและทับถมของสารอินทรียวัตถุ (ฮิวมัส) ที่เกิดจากการย่อยสลายของซากพืชและซากสัตว์ ดินในชั้นนี้จึงมีสีเข้มและมีธาตุอาหารอุดมสมบูรณ์ที่สุด ซึ่งมีบทบาทสำคัญอย่างมากต่อการเจริญเติบโตของพืช
* **ชั้น B (ดินชั้นล่าง / Subsoil):** เป็นชั้นดินที่สะสมตะกอนและแร่ธาตุต่าง ๆ ที่ถูกชะล้างลงมาจากดินชั้นบน
* **ชั้น C (ชั้นการผุพังของหิน / Weathered Parent Material):** เป็นชั้นที่ประกอบด้วยเศษหินและกรวดที่ผุพังมาจากหินดินดานด้านล่าง
* **ชั้น D (หินพื้น / Bedrock):** เป็นชั้นหินแข็งดั้งเดิมที่เป็นวัตถุต้นกำเนิดของดินชั้นบนทั้งหมด ถือเป็นชั้นแรกสุดที่เกิดขึ้น ไม่ใช่ชั้นสุดท้าย

พิจารณาตัวเลือกต่าง ๆ:
- **ตัวเลือก ① และ ② ไม่ถูกต้อง:** ลำดับการเกิดชั้นดินจะเริ่มจากหินพื้น (D) ผุพังเป็นวัตถุต้นกำเนิดดิน (C) แล้วค่อย ๆ พัฒนาขึ้นไปเป็นชั้น B และ A ตามลำดับ ไม่ใช่ชั้นบนย่อยสลายเป็นชั้นล่าง
- **ตัวเลือก ③ ไม่ถูกต้อง:** ชั้น D (หินพื้น) เกิดขึ้นเป็นลำดับแรกสุดในกระบวนการเกิดดิน
- **ตัวเลือก ④ ถูกต้อง:** ชั้น A เป็นดินชั้นบนที่มีฮิวมัสจากการทับถมของซากพืชซากสัตว์ ซึ่งเป็นประโยชน์โดยตรงต่อการเจริญเติบโตของพืช
- **ตัวเลือก ⑤ ไม่ถูกต้อง:** กระบวนการเกิดดินจากชั้น D ถึง A ใช้เวลาเวลายาวนานมาก (นับร้อยหรือพันปี) และมีความเกี่ยวข้องโดยตรงกับการพัดพาหน้าดินและการสะสมมลภาวะ

ดังนั้น อธิบายเกี่ยวกับดินได้ถูกต้องคือ **A มีบทบาทสำคัญต่อการเจริญเติบโตของพืช เพราะเป็นชั้นที่มีการทับถมของซากพืชซากสัตว์**

ซึ่งสอดคล้องกับตัวเลือก **4**
