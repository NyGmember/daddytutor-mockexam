# คำถาม

12. ถังเปล่าทรงกระบอกใบหนึ่งมีความสูงเท่ากับความยาวเส้นผ่านศูนย์กลางของฐานนำภาชนะครึ่งทรงกลมที่มีรัศมียาวเท่ากับรัศมีของฐานของถังทรงกระบอก มาตักทรายใส่ถังจงหาว่าต้องตักทรายทั้งหมดกี่ครั้ง ทรายถึงจะเต็มถัง

# คำอธิบายและวิธีทำ

กำหนดให้รัศมีของฐานของถังทรงกระบอกคือ r หน่วย
ดังนั้น ความยาวเส้นผ่านศูนย์กลางของฐานถังทรงกระบอกคือ 2r หน่วย
จากโจทย์ ความสูงของถังทรงกระบอก (h) เท่ากับความยาวเส้นผ่านศูนย์กลางของฐาน
ดังนั้น h = 2r
ปริมาตรของถังทรงกระบอก (V_cylinder) = พื้นที่ฐาน × ความสูง = πr^2h
แทนค่า h = 2r จะได้ V_cylinder = πr^2(2r) = 2πr^3 ลูกบาศก์หน่วย

ภาชนะที่ใช้ตักทรายเป็นครึ่งทรงกลม มีรัศมีเท่ากับรัศมีของฐานของถังทรงกระบอก
ดังนั้น รัศมีของภาชนะครึ่งทรงกลมคือ r หน่วย
ปริมาตรของทรงกลม = (4/3)πr^3 ลูกบาศก์หน่วย
ปริมาตรของครึ่งทรงกลม (V_hemisphere) = (1/2) × (4/3)πr^3 = (2/3)πr^3 ลูกบาศก์หน่วย

จำนวนครั้งที่ต้องตักทรายจนเต็มถัง = ปริมาตรของถังทรงกระบอก / ปริมาตรของครึ่งทรงกลม
จำนวนครั้ง = (2πr^3) / ((2/3)πr^3)
จำนวนครั้ง = 2 / (2/3)
จำนวนครั้ง = 2 × (3/2)
จำนวนครั้ง = 3
ดังนั้น ต้องตักทรายทั้งหมด 3 ครั้ง ทรายถึงจะเต็มถัง
