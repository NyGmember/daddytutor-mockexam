# คำถาม

25. แขวนลูกบอลที่ปลายเชือกที่ยาว 80 เซนติเมตร แล้วดึงขึ้นมาในแนวระดับดังภาพ ถ้าปล่อยเชือกแล้วลูกบอลเคลื่อนที่ตามส่วนโค้งของวงกลมจากจุด P และกระทบกับปลายไม้ทำให้ลูกบอลเคลื่อนที่เป็นวงกลมจนถึงจุด Q โดยมีรัศมี 20 เซนติเมตร

จงหาว่าความเร็วเมื่อลูกบอลมาถึงจุด Q เป็นกี่เมตร/วินาที (ความเร่งโน้มถ่วงคือ $g = 9.8 \text{ m/s}^2$)

(1) 1

# คำอธิบายและวิธีทำ

หลักการสำคัญในการแก้โจทย์ข้อนี้คือการใช้กฎการอนุรักษ์พลังงานกล (Law of Conservation of Mechanical Energy) โดยพิจารณาพลังงานศักย์โน้มถ่วงที่เปลี่ยนไปเป็นพลังงานจลน์ 

1. กำหนดตำแหน่ง P เป็นจุดเริ่มต้น: ณ จุดนี้ ลูกบอลอยู่นิ่ง ความเร็ว v_P = 0
2. กำหนดตำแหน่ง Q เป็นจุดสิ้นสุด: ณ จุดนี้ลูกบอลเคลื่อนที่เป็นวงกลมรัศมี r = 20 ซม. (0.2 เมตร)
3. พิจารณาระดับความสูง (h): 
   - จุด P อยู่ในแนวระดับเดียวกับจุดที่เชือกแขวน ดังนั้นถ้าให้ระดับแขวนเป็น h = 0 จุด P จะอยู่ที่ระดับความสูง h_P = 0 เมตร
   - เมื่อเชือกยาว L = 80 ซม. (0.8 เมตร) แกว่งลงมาถึงจุด Q ลูกบอลจะอยู่ต่ำกว่าระดับจุดแขวนเป็นระยะ L ดังนั้น h_Q = -0.8 เมตร
   - ความแตกต่างของความสูงระหว่างจุด P และจุด Q คือ Delta h = h_P - h_Q = 0 - (-0.8) = 0.8 เมตร
4. ใช้กฎการอนุรักษ์พลังงาน: พลังงานศักย์ที่ลดลงเท่ากับพลังงานจลน์ที่เพิ่มขึ้น
   mgh = (1/2)mv^2
   gh = (1/2)v^2
   v^2 = 2gh
   v^2 = 2 * 9.8 * 0.8
   v^2 = 15.68
   v = sqrt(15.68) ≈ 3.96 เมตร/วินาที

เมื่อพิจารณาตัวเลือกที่มี (ใกล้เคียงที่สุดคือ 3.92 ซึ่งมาจากการคำนวณที่ใช้ค่า g หรือการประมาณความสูงที่แตกต่างกันเล็กน้อย แต่หลักการทางฟิสิกส์คือ v = sqrt(2gh))
