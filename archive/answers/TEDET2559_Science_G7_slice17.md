# คำถาม

14. แบมซื้อต้นไม้มาปลูกในกระถาง รดน้ำต้นไม้ทุกวัน จนต้นไม้โตขึ้น แต่อีกไม่กี่วันต่อมาต้นไม้เหี่ยวเฉา แบมตั้งสมมติฐานว่า ‘รดน้ำต้นไม้มากเกินไปจนต้นไม้เหี่ยวเฉา’ แบมจึงทำการทดลองเพื่อพิสูจน์สมมติฐาน

ข้อใดจับคู่ตัวแปรควบคุมและตัวแปรต้นได้ถูกต้อง

| | ตัวแปรควบคุม | ตัวแปรต้น |
|---|---|---|
| ① | อุณหภูมิ | ความเข้มของแสง |
| ② | ขนาดของต้นไม้ | ชนิดของต้นไม้ |
| ③ | ชนิดของต้นไม้ | ปริมาณของน้ำที่รดต้นไม้ |
| ④ | ปริมาณของน้ำที่รดต้นไม้ | อุณหภูมิ |
| ⑤ | ระดับความเหี่ยวเฉาของต้นไม้ | ปริมาณปุ๋ยที่ใส่ให้ต้นไม้ |

# คำอธิบายและวิธีทำ

ในการทดลองทางวิทยาศาสตร์ ตัวแปรที่สำคัญประกอบด้วย 3 ประเภทคือ: 1. ตัวแปรต้น (Independent Variable) คือสิ่งที่เราต้องการศึกษาหรือสิ่งที่ทำให้เกิดผลการทดลองที่แตกต่างกัน ซึ่งในกรณีนี้คือ 'ปริมาณของน้ำที่รดต้นไม้' เพื่อดูว่าส่งผลให้ต้นไม้เหี่ยวหรือไม่ 2. ตัวแปรตาม (Dependent Variable) คือผลที่เกิดขึ้นจากการทดลอง ซึ่งก็คือ 'ระดับความเหี่ยวเฉาของต้นไม้' 3. ตัวแปรควบคุม (Controlled Variable) คือปัจจัยอื่นๆ ที่อาจส่งผลกระทบต่อผลการทดลอง จึงต้องทำให้เหมือนกันทุกกลุ่มตัวอย่างเพื่อไม่ให้เป็นตัวแปรแทรกซ้อน เช่น 'ชนิดของต้นไม้' ขนาดของกระถาง หรือสภาพแวดล้อมที่ตั้งต้นไม้ ดังนั้น ตัวเลือกที่ถูกต้องที่สุดคือตัวเลือกที่ระบุชนิดของต้นไม้เป็นตัวแปรควบคุม (เพื่อให้มั่นใจว่าผลที่ได้เกิดจากปริมาณน้ำ ไม่ใช่เพราะพันธุ์ไม้ต่างกัน) และปริมาณน้ำเป็นตัวแปรต้นตามสมมติฐานที่ตั้งไว้
