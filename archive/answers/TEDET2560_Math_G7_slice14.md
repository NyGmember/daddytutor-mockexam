# คำถาม

14. พิจารณาการเขียนจำนวนบนด้านของรูปสามเหลี่ยมด้านเท่า ดังรูป ![illustration](../images/TEDET2560_Math_G7_slice14_crop_0.png) ถ้าผลบวกของจำนวนสามจำนวนบนแต่ละด้านมีค่าเท่ากันหมด จงหาค่าของ $A \times B$

# คำอธิบายและวิธีทำ

กำหนดให้ผลรวมของจำนวนบนแต่ละด้านของรูปสามเหลี่ยมมีค่าเท่ากับ S ตามภาพ เราสามารถเขียนสมการจากผลรวมของแต่ละด้านได้ดังนี้: 1) ด้านบน-ซ้าย: 3 + A + B = S, 2) ด้านล่าง: B + 2 + (-2) = S ซึ่งจะได้ B + 0 = S ดังนั้น B = S, 3) ด้านบน-ขวา: 3 + (-5) + (-2) = S ซึ่งจะได้ -4 = S. เมื่อเราทราบว่า S = -4 แล้ว เราสามารถแทนค่าในสมการที่ 1 และ 2 เพื่อหาค่า A และ B: จาก B = S จะได้ B = -4. จาก 3 + A + B = S แทนค่า B และ S จะได้ 3 + A + (-4) = -4, นั่นคือ A - 1 = -4, ดังนั้น A = -3. โจทย์ถามหาค่าของ A * B ดังนั้นจะได้ (-3) * (-4) = 12.
