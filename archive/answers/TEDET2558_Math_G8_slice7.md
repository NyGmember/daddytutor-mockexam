# คำถาม

7. จากรูป จงหาว่า $\angle a + \angle b + \angle c + \angle d + \angle e + \angle f$ เท่ากับกี่องศา

![illustration](../images/TEDET2558_Math_G8_slice7_crop_0.png)

# คำอธิบายและวิธีทำ

พิจารณารูปสามเหลี่ยมทั้งหมดที่เกี่ยวข้องในรูปนี้: 1. ให้รูปสามเหลี่ยมใหญ่คือ $\triangle ABC$ ผลรวมของมุมภายในคือ $\angle A + \angle B + \angle C = 180^\circ$ 2. ด้านในรูปสามเหลี่ยมใหญ่มีรูปห้าเหลี่ยมอยู่หนึ่งรูป และมีรูปสามเหลี่ยมเล็กๆ เกาะอยู่รอบๆ เราสามารถหาผลรวมมุมได้โดยใช้คุณสมบัติของมุมภายในรูปสามเหลี่ยม: - จากรูปสามเหลี่ยมที่มียอดเป็น $A$ และมุมฐานคือ $a$ และ $f$ จะได้ผลรวมมุมในสามเหลี่ยมนี้คือ $180^\circ - (a+f)$ (มุมที่ฐาน $B$ และ $C$ ส่วนที่เหลือ) - อย่างไรก็ตาม วิธีที่ง่ายกว่าคือการมองที่รูปสามเหลี่ยม 3 รูปที่อยู่รอบนอกรูปห้าเหลี่ยมภายใน: รูปสามเหลี่ยมที่ 1 มีมุมยอดคือ $A$ และมุมฐานคือ $a, f$ (หมายเหตุ: จากรูป $a$ และ $f$ คือมุมภายในสามเหลี่ยมเล็กที่ยอด $A$), รูปสามเหลี่ยมที่ 2 มีมุมยอดคือ $B$ และมุมฐานคือมุมที่เหลือของรูปห้าเหลี่ยม, รูปสามเหลี่ยมที่ 3 มีมุมยอดคือ $C$ และมุมฐานคือมุมที่เหลือของรูปห้าเหลี่ยม ในที่นี้ ผลรวม $\angle a + \angle b + \angle c + \angle d + \angle e + \angle f$ คือผลรวมมุมของสามเหลี่ยมสองรูปที่ซ้อนทับกันภายในรูปสามเหลี่ยมใหญ่ $ABC$ หรือใช้หลักการผลรวมมุมภายในรูปสามเหลี่ยมสามรูปประกอบกัน กล่าวคือผลรวมมุมเหล่านี้จะเท่ากับผลรวมของมุมภายในของรูปสามเหลี่ยมสองรูป ซึ่งเท่ากับ $180^\circ + 180^\circ = 360^\circ$
