# คำถาม

15. ถ้า $x^2-7x+1=0$ จงหาค่าของ $x^2+x+\frac{1}{x}+\frac{1}{x^2}$

# คำอธิบายและวิธีทำ

โจทย์กำหนดให้สมการ $x^2-7x+1=0$ และต้องการหาค่าของ $x^2+x+rac{1}{x}+rac{1}{x^2}$

ขั้นตอนที่ 1: จัดรูปสมการที่กำหนดให้
จากสมการที่กำหนด $x^2-7x+1=0$
เนื่องจาก $x=0$ จะทำให้ $1=0$ ซึ่งเป็นไปไม่ได้ ดังนั้น $x \neq 0$ เราสามารถหารสมการตลอดด้วย $x$ ได้:
$\frac{x^2}{x} - \frac{7x}{x} + \frac{1}{x} = \frac{0}{x}$
$x - 7 + \frac{1}{x} = 0$
ย้ายข้างตัวเลขไปอีกฝั่ง:
$x + \frac{1}{x} = 7$ (สมการที่ 1)

ขั้นตอนที่ 2: จัดรูปนิพจน์ที่ต้องการหาค่า
นิพจน์ที่ต้องการหาค่าคือ $x^2+x+rac{1}{x}+rac{1}{x^2}$
เราสามารถจัดกลุ่มพจน์ใหม่ได้ดังนี้:
$(x^2 + \frac{1}{x^2}) + (x + \frac{1}{x})$

จากสมการที่ 1 เรารู้ค่าของ $(x + \frac{1}{x})$ แล้ว นั่นคือ 7

ขั้นตอนที่ 3: หาค่าของ $(x^2 + \frac{1}{x^2})$
จากสมการที่ 1: $x + \frac{1}{x} = 7$
ยกกำลังสองทั้งสองข้างของสมการ:
$(x + \frac{1}{x})^2 = 7^2$
ใช้สูตร $(a+b)^2 = a^2 + 2ab + b^2$ โดยให้ $a=x$ และ $b=\frac{1}{x}$:
$x^2 + 2(x)(\frac{1}{x}) + (\frac{1}{x})^2 = 49$
$x^2 + 2 + \frac{1}{x^2} = 49$
ย้ายข้าง 2 ไปอีกฝั่ง:
$x^2 + \frac{1}{x^2} = 49 - 2$
$x^2 + \frac{1}{x^2} = 47$ (สมการที่ 2)

ขั้นตอนที่ 4: นำค่าที่ได้ไปแทนในนิพจน์
แทนค่าจากสมการที่ 1 และสมการที่ 2 ลงในนิพจน์ที่จัดรูปไว้:
$(x^2 + \frac{1}{x^2}) + (x + \frac{1}{x}) = 47 + 7$
$= 54$

ดังนั้น ค่าของ $x^2+x+rac{1}{x}+rac{1}{x^2}$ คือ 54
