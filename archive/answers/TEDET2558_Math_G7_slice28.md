# คำอธิบายและวิธีทำ

To solve this problem, we need to determine the number of coins at each position (A, B, C, D, E) by analyzing the two given projections (views). Let the number of coins at each position be represented by variables $n_A, n_B, n_C, n_D, n_E$. 1. The view (a) represents the projection from the top-left perspective, showing the side view of rows. Looking at the illustration of view (a), we see 4 stacks of coins in total height. The left part is 2 stacks high (height 3 and 2), and the right part is 1 stack high (height 4). Mapping these to positions A, B, C, D, E based on the lines projected, we can deduce: $n_A=3, n_B=2, n_E=4$ (for the vertical column), $n_C=2$ (right), $n_D=1$ (top). 2. Let's verify with view (b), which is the view from the bottom perspective: The columns correspond to C and A on the sides, and D, E, B in the center. The heights are: Left column height is 3, middle column height is 4, right column height is 2. Following the visual projection lines: Position A has 3 coins, Position B has 2 coins, Position C has 2 coins, Position D has 1 coin, Position E has 4 coins. Adding these up: $3 + 2 + 2 + 1 + 4 = 12$ coins. Each coin is worth 10 baht. Therefore, the total value is $12 \times 10 = 120$ baht.
