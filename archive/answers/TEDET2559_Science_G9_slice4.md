# คำอธิบายและวิธีทำ

การวิเคราะห์ผลการทดลอง: 1) ถุงเซลโลเฟนทำหน้าที่เป็นเยื่อเลือกผ่าน (semi-permeable membrane) ซึ่งรูขนาดเล็กยอมให้สารโมเลกุลเล็กผ่านได้ แต่สารโมเลกุลใหญ่ผ่านไม่ได้ 2) การทดสอบด้วยเบเนดิกต์ (สีเหลืองปนแดง = ตรวจพบน้ำตาลโมเลกุลเดี่ยว/กลูโคส) และไอโอดีน (สีน้ำเงิน = ตรวจพบแป้ง) 3) ชุด C: กลูโคสพบทั้งในและนอกถุง แสดงว่ากลูโคสผ่านถุงได้ (สรุป ง ถูก) 4) ชุด A: แป้งพบแค่ในถุง ไม่พบนอกถุง แสดงว่าแป้งผ่านถุงไม่ได้ (สรุป ค ถูก) 5) ชุด D: สารละลายแป้ง + น้ำลาย พบทั้งน้ำตาลนอกถุงและในถุง แสดงว่าน้ำลายย่อยแป้งให้กลูโคสและน้ำตาลเหล่านี้แพร่ออกนอกถุงได้ (สรุป ก ถูก) 6) จากข้อเท็จจริง: น้ำตาล (กลูโคส) ผ่านถุงได้ แต่แป้งผ่านไม่ได้ หมายความว่า รูของถุงเซลโลเฟนมีขนาดเล็กกว่าโมเลกุลแป้ง แต่ใหญ่กว่าโมเลกุลของกลูโคส/สารที่ได้จากการย่อยแป้ง ดังนั้นการเรียงลำดับขนาดคือ 'รูถุงเซลโลเฟน < แป้ง' และ 'สารที่ได้จากการย่อยแป้ง < รูถุงเซลโลเฟน' ซึ่งสรุปการเปรียบเทียบใน (A) คือ 'สารที่ได้จากการย่อยแป้ง < รูถุงเซลโลเฟน < แป้ง' นั้นถูกต้อง 7) ตรวจสอบความถูกต้องของข้อความ: (จ) มอลต์ไม่ใช่เอนไซม์ที่ใช้ย่อยแป้ง แต่มอลต์เป็นน้ำตาลมอลโตส การบอกว่ามอลต์เปลี่ยนแป้งเป็นกลูโคสจึงเป็นข้อความที่ไม่ถูกต้อง 8) ดังนั้น คู่ที่ 'ไม่ถูกต้อง' คือ (จ) และ (A) ซึ่งตรงกับตัวเลือกที่ 5
