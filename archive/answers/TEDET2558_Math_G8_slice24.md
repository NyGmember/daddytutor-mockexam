# คำถาม

18. ถ้า $2^{a+1} + 2^b = 80$ และ $2^{1+b} = 32$ จงหาค่าของ $a + b$

# คำอธิบายและวิธีทำ

จากโจทย์ที่กำหนดให้เรามีสมการสองสมการคือ: (1) $2^{a+1} + 2^b = 80$ และ (2) $2^{1+b} = 32$. ขั้นแรก เราจะหาค่าของ $b$ จากสมการที่ (2): เนื่องจาก $32 = 2^5$ ดังนั้น $2^{1+b} = 2^5$ ซึ่งจะได้ว่า $1 + b = 5$ ทำให้ได้ $b = 4$. ขั้นที่สอง นำค่า $b = 4$ ไปแทนในสมการที่ (1): $2^{a+1} + 2^4 = 80$. เนื่องจาก $2^4 = 16$ ดังนั้น $2^{a+1} + 16 = 80$. ย้ายข้างสมการจะได้ $2^{a+1} = 80 - 16 = 64$. เนื่องจาก $64 = 2^6$ ดังนั้น $2^{a+1} = 2^6$ ซึ่งจะได้ $a + 1 = 6$ ทำให้ได้ $a = 5$. โจทย์ต้องการหาค่าของ $a + b$ ดังนั้น $a + b = 5 + 4 = 9$.
