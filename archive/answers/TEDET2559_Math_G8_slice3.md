# คำถาม

3. กำหนดให้ A, B, C และ D เป็นจำนวนจริงสี่จำนวน ที่มีความสัมพันธ์ดังต่อไปนี้

![illustration](../images/TEDET2559_Math_G8_slice3_crop_0.png)

จงเรียงลำดับความยาวของจำนวนจริงทั้งสี่จำนวนนี้ จากมากไปน้อย

(1) A, B, C, D
(2) B, A, C, D
(3) C, B, A, D
(4) C, B, D, A
(5) B, C, A, D

# คำอธิบายและวิธีทำ

เราจะคำนวณค่าของ A, B, C และ D ทีละตัวเพื่อนำมาเรียงลำดับกัน:

1. จากโจทย์กำหนดให้ C = 250% ของ 6 จะได้:
C = 2.5 * 6 = 15

2. จากโจทย์กำหนดให้ D = 0.3 เท่าของ C จะได้:
D = 0.3 * 15 = 4.5

3. จากโจทย์กำหนดให้ A = 60% ของผลรวม C กับ D จะได้:
A = 0.6 * (C + D) = 0.6 * (15 + 4.5) = 0.6 * 19.5 = 11.7

4. จากโจทย์กำหนดให้ B มากกว่าผลต่างของ A กับ D จะได้:
ผลต่างของ A กับ D คือ A - D = 11.7 - 4.5 = 7.2
โจทย์ระบุว่า B มากกว่าผลต่างนี้ แต่ไม่ได้กำหนดค่าที่แน่นอน อย่างไรก็ตามในบริบทของข้อสอบลักษณะนี้ เราพิจารณาว่า B มากกว่าค่าที่ได้ (ในที่นี้คือ 7.2) หากลองสังเกตจากตัวเลือก B จะต้องอยู่ระหว่าง C และ A หรือมากกว่านั้น แต่จากตัวเลือกที่มี B มักจะมีค่าสูงสุด หรือเป็นไปตามลำดับที่สัมพันธ์กับ A, C, D หากลองเรียงค่าที่มีคือ C=15, A=11.7, D=4.5 ซึ่งเรียงได้เป็น C > A > D

เมื่อพิจารณาตัวเลือกที่ให้มา:
(1) A, B, C, D
(2) B, A, C, D
(3) C, B, A, D
(4) C, B, D, A
(5) B, C, A, D

หาก B มีค่ามากที่สุด จะได้ลำดับ B > C (15) > A (11.7) > D (4.5) ซึ่งตรงกับตัวเลือกที่ (5)
