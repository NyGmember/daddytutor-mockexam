# คำถาม

25. ถ้า $x = \frac{2}{\sqrt{3} - \sqrt{2}}$ และ $y = \frac{2}{\sqrt{3} + \sqrt{2}}$

จงหาค่าของ $\left(\frac{2}{x} + \frac{2}{y}\right)^2 + \left(\frac{2}{x} - \frac{2}{y}\right)^2$

# คำอธิบายและวิธีทำ

เราสามารถแก้โจทย์นี้ได้โดยการหาค่าของ \(\frac{1}{x}\) และ \(\frac{1}{y}\) ก่อน:
1. จาก \(x = \frac{2}{\sqrt{3} - \sqrt{2}}\)， จะได้ \(\frac{1}{x} = \frac{\sqrt{3} - \sqrt{2}}{2}\)
2. จาก \(y = \frac{2}{\sqrt{3} + \sqrt{2}}\)， จะได้ \(\frac{1}{y} = \frac{\sqrt{3} + \sqrt{2}}{2}\)

3. หาค่า \(\frac{2}{x}\) และ \(\frac{2}{y}\):
   \(\frac{2}{x} = 2 \cdot \frac{\sqrt{3} - \sqrt{2}}{2} = \sqrt{3} - \sqrt{2}\)
   \(\frac{2}{y} = 2 \cdot \frac{\sqrt{3} + \sqrt{2}}{2} = \sqrt{3} + \sqrt{2}\)

4. คำนวณนิพจน์แรก \(\left(\frac{2}{x} + \frac{2}{y}\right)^2\):
   \((\sqrt{3} - \sqrt{2} + \sqrt{3} + \sqrt{2})^2 = (2\sqrt{3})^2 = 4 \cdot 3 = 12\)

5. คำนวณนิพจน์ที่สอง \(\left(\frac{2}{x} - \frac{2}{y}\right)^2\):
   \((\sqrt{3} - \sqrt{2} - (\sqrt{3} + \sqrt{2}))^2 = (-2\sqrt{2})^2 = 4 \cdot 2 = 8\)

6. นำผลลัพธ์มารวมกัน: \(12 + 8 = 20\)
