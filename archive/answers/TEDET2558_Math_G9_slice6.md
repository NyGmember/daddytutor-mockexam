# คำถาม

6. จงหาจำนวนเต็ม $a$ ที่มีค่าอยู่ระหว่าง 400 กับ 500 และทำให้ $\frac{\sqrt{a}}{\sqrt{3}}$ มีค่าเป็นจำนวนเต็ม

# คำอธิบายและวิธีทำ

จากโจทย์ที่กำหนดให้ $\frac{\sqrt{a}}{\sqrt{3}}$ เป็นจำนวนเต็ม เราสามารถเขียนใหม่ได้เป็น $\sqrt{\frac{a}{3}} = k$ โดยที่ $k$ เป็นจำนวนเต็ม จะได้ว่า $\frac{a}{3} = k^2$ หรือ $a = 3k^2$ โจทย์กำหนดให้ $400 < a < 500$ ดังนั้นจะได้อสมการคือ $400 < 3k^2 < 500$ หารด้วย 3 ทั้งอสมการ จะได้ $\frac{400}{3} < k^2 < \frac{500}{3}$ ซึ่งคำนวณได้ประมาณ $133.33 < k^2 < 166.66$ เราต้องหาจำนวนเต็ม $k$ ที่ทำให้ $k^2$ อยู่ในช่วงนี้ ค่าที่เหมาะสมคือ $k^2$ ที่เป็นเลขกำลังสองสมบูรณ์ ได้แก่ $12^2 = 144$ และ $13^2 = 169$ ทดสอบทีละค่า: 1) ถ้า $k^2 = 144$ จะได้ $a = 3 \times 144 = 432$ ซึ่ง $400 < 432 < 500$ เป็นจริง 2) ถ้า $k^2 = 169$ จะได้ $a = 3 \times 169 = 507$ ซึ่งไม่อยู่ในช่วง $400 < a < 500$ ดังนั้นค่า $a$ ที่เป็นไปได้เพียงค่าเดียวคือ $432$
