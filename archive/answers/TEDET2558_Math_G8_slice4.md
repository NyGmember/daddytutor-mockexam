# คำถาม

4. ถ้าเขียน $2.4\dot{6}$ ในรูปเศษส่วนจะได้เป็น $\frac{x}{90}$ ซึ่งเมื่อเขียนในรูปเศษส่วนอย่างต่ำ จะได้เป็น $\frac{37}{y}$ จงหาค่าของ $x + y$

# คำอธิบายและวิธีทำ

ขั้นตอนที่ 1: แปลงทศนิยมซ้ำ $2.4\dot{6}$ ให้เป็นเศษส่วน โดยใช้หลักการเปลี่ยนทศนิยมซ้ำเป็นเศษส่วน ดังนี้ $2.4\dot{6} = 2 + 0.4\dot{6}$ และ $0.4\dot{6} = \frac{46 - 4}{90} = \frac{42}{90}$ ดังนั้น $2.4\dot{6} = 2 + \frac{42}{90} = \frac{180 + 42}{90} = \frac{222}{90}$ จะได้ $x = 222$ 
ขั้นตอนที่ 2: ทำให้ $\frac{x}{90}$ เป็นเศษส่วนอย่างต่ำ โดยที่ $x = 222$ จะได้ $\frac{222}{90}$ นำแม่ 6 มาหารทั้งเศษและส่วน: $222 \div 6 = 37$ และ $90 \div 6 = 15$ ดังนั้น เศษส่วนอย่างต่ำคือ $\frac{37}{15}$ 
ขั้นตอนที่ 3: เทียบค่ากับ $\frac{37}{y}$ จะได้ $y = 15$ 
ขั้นตอนที่ 4: หาค่า $x + y = 222 + 15 = 237$
