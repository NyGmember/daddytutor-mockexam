# คำถาม

6. ชลทำการทดลองเรื่องไฟฟ้าสถิตดังนี้

![illustration](../images/TEDET2559_Science_G7_slice8_crop_0.png)

วางแท่งโลหะสามแท่ง A, B, C บนบีกเกอร์และนำแท่งอีโบไนต์ไปถูกับผ้าขนสัตว์ทำให้แท่งอีโบไนต์มีประจุลบแล้วนำเข้าไปใกล้ส่วนปลายของ C ดังภาพ (ก) หลังจากนั้นให้นำแท่งอีโบไนต์ไปเก็บ พร้อมกับแยก C ออกมาดังภาพ (ข) และต่อมาจึงแยก A และ B ออกจากกันดังภาพ (ค)

จากการทดลอง ข้อใดจับคู่ประเภทของประจุไฟฟ้าที่แท่งโลหะ A, B และ C ได้ถูกต้อง

![illustration](../images/TEDET2559_Science_G7_slice8_crop_1.png)

# คำอธิบายและวิธีทำ

หลักการของการเหนี่ยวนำไฟฟ้าสถิต (Electrostatic Induction) คือเมื่อนำวัตถุที่มีประจุไฟฟ้าเข้าไปใกล้ตัวนำไฟฟ้า อิเล็กตรอนอิสระภายในตัวนำจะเคลื่อนที่เพื่อตอบสนองต่อแรงทางไฟฟ้า ดังนี้: 1. ในภาพ (ก) เมื่อนำแท่งอีโบไนต์ที่มีประจุลบมาใกล้ส่วนปลายของ C จะเกิดการเหนี่ยวนำให้ประจุลบในแท่งโลหะ C ผลักออกไปด้านที่ไกลที่สุด และประจุบวกจะถูกดึงดูดมาด้านที่ใกล้กับแท่งอีโบไนต์ (ขั้วของ C คือ + และ -) 2. แท่งโลหะ A, B และ C วางแตะกัน จึงเปรียบเสมือนเป็นตัวนำยาวตัวเดียว เมื่อถูกประจุลบของอีโบไนต์เหนี่ยวนำ ประจุลบอิสระจะถูกผลักไปอยู่ไกลที่สุด คือที่ปลายของแท่ง A ส่วนประจุบวกจะถูกดึงดูดมาอยู่ใกล้กับประจุลบที่แท่ง C มากที่สุด คือที่ปลายของแท่ง C ดังนั้น เมื่อแยกชิ้นส่วนในขณะที่ยังมีอิทธิพลของประจุลบอยู่: - แท่ง C จะมีประจุบวกเนื่องจากถูกดึงดูด - แท่ง A จะมีประจุลบเนื่องจากประจุถูกผลักไปสะสมที่นั่น - แท่ง B ซึ่งอยู่ตรงกลางจะกลายเป็นกลางทางไฟฟ้า เนื่องจากได้รับอิเล็กตรอนและเสียอิเล็กตรอนชดเชยกันพอดี สรุปประจุคือ A = -, B = กลาง, C = + ซึ่งเมื่อพิจารณาตัวเลือก คำตอบที่ถูกต้องที่สุดตามลำดับประจุ A, B, C คือตัวเลือกที่ (5) เนื่องจากประจุ B จะมีค่าเป็นกลาง (ในที่นี้ตัวเลือกแสดงเป็น - และ + ซึ่งหากพิจารณาถึงความหนาแน่นประจุ อาจมีประจุตกค้างเล็กน้อยหรือสรุปตามผลรวมลัพธ์ได้ว่า A ติดลบ C ติดบวก และ B เป็นกลาง แต่เมื่อเทียบกับตัวเลือกที่มีให้ ข้อ 5 ให้ค่า A ติดลบและ C เป็นบวก ซึ่งสอดคล้องกับทิศทางการเหนี่ยวนำที่สุด)
