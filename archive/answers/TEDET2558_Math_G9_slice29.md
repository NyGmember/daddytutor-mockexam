# คำถาม

29. จำนวนนับที่มีสามหลักซึ่งเขียนในระบบตัวเลขฐานสิบได้เป็น $abc$ มีทั้งหมดกี่จำนวน เมื่อ $a, b, c$ เป็นเลขโดดที่ไม่เป็นศูนย์ และ $a = \frac{b + c}{2}$

# คำอธิบายและวิธีทำ

เราต้องการหาจำนวนนับ 3 หลัก $abc$ โดยที่ $a, b, c \in \{1, 2, 3, 4, 5, 6, 7, 8, 9\}$ และ $a = \frac{b+c}{2}$ ซึ่งเขียนได้อีกรูปแบบหนึ่งคือ $2a = b+c$ โดยที่ $a, b, c$ ไม่เป็นศูนย์\n\nจากการสังเกตสมการ $2a = b+c$ จะเห็นได้ว่าผลบวกของ $b+c$ ต้องเป็นเลขคู่ ดังนั้น $b$ และ $c$ ต้องมีความเป็นคู่เหมือนกัน (คือเป็นเลขคู่ทั้งคู่ หรือเลขคี่ทั้งคู่) เพื่อให้ผลรวมเป็นเลขคู่เสมอ\n\nเราจะแบ่งกรณีพิจารณาตามค่าของ $a$ ตั้งแต่ 1 ถึง 9:\n1. ถ้า $a = 1$: $b+c = 2$ คู่ลำดับ $(b, c)$ ที่เป็นไปได้คือ $(1, 1)$ [รวม 1 จำนวน]\n2. ถ้า $a = 2$: $b+c = 4$ คู่ลำดับ $(b, c)$ ได้แก่ $(1, 3), (2, 2), (3, 1)$ [รวม 3 จำนวน]\n3. ถ้า $a = 3$: $b+c = 6$ คู่ลำดับ $(b, c)$ ได้แก่ $(1, 5), (2, 4), (3, 3), (4, 2), (5, 1)$ [รวม 5 จำนวน]\n4. ถ้า $a = 4$: $b+c = 8$ คู่ลำดับ $(b, c)$ ได้แก่ $(1, 7), (2, 6), (3, 5), (4, 4), (5, 3), (6, 2), (7, 1)$ [รวม 7 จำนวน]\n5. ถ้า $a = 5$: $b+c = 10$ คู่ลำดับ $(b, c)$ ได้แก่ $(1, 9), (2, 8), ..., (9, 1)$ [เลขโดดที่ใช้ได้คือ 1-9 ดังนั้นมีทั้งหมด 9 จำนวน คือ $(1,9), (2,8), (3,7), (4,6), (5,5), (6,4), (7,3), (8,2), (9,1)$]\n6. ถ้า $a = 6$: $b+c = 12$ คู่ลำดับ $(b, c)$ ได้แก่ $(3, 9), (4, 8), (5, 7), (6, 6), (7, 5), (8, 4), (9, 3)$ [รวม 7 จำนวน]\n7. ถ้า $a = 7$: $b+c = 14$ คู่ลำดับ $(b, c)$ ได้แก่ $(5, 9), (6, 8), (7, 7), (8, 6), (9, 5)$ [รวม 5 จำนวน]\n8. ถ้า $a = 8$: $b+c = 16$ คู่ลำดับ $(b, c)$ ได้แก่ $(7, 9), (8, 8), (9, 7)$ [รวม 3 จำนวน]\n9. ถ้า $a = 9$: $b+c = 18$ คู่ลำดับ $(b, c)$ ได้แก่ $(9, 9)$ [รวม 1 จำนวน]\n\nรวมจำนวนทั้งหมดคือ $1 + 3 + 5 + 7 + 9 + 7 + 5 + 3 + 1 = 41$ จำนวน
