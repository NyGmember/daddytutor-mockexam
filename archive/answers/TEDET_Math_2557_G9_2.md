# คำถาม

2. กำหนดให้ $a$ และ $b$ เป็นจำนวนตรรกยะที่ทำให้$$\frac{\sqrt{54}}{\sqrt{2}} - \frac{4}{\sqrt{8}} + \frac{12\sqrt{3}}{\sqrt{6}} - \frac{18}{\sqrt{3}} = a\sqrt{2} + b\sqrt{3}$$จงหาค่าของ $2a+b$

# คำอธิบายและวิธีทำ

ในการแก้ปัญหานี้ เราต้องทำให้แต่ละพจน์ของสมการที่กำหนดอยู่ในรูปอย่างง่ายที่สุด โดยมีตัวส่วนเป็นจำนวนตรรกยะ จากนั้นรวมพจน์ที่เหมือนกันเพื่อเทียบสัมประสิทธิ์กับรูปแบบ $a\sqrt{2} + b\sqrt{3}$ เพื่อหาค่า $a$ และ $b$ แล้วจึงคำนวณค่าของ $2a+b$

สมการที่กำหนดคือ:
$$\frac{\sqrt{54}}{\sqrt{2}} - \frac{4}{\sqrt{8}} + \frac{12\sqrt{3}}{\sqrt{6}} - \frac{18}{\sqrt{3}} = a\sqrt{2} + b\sqrt{3}$$

เราจะทำการทำให้แต่ละพจน์ทางซ้ายมือ (LHS) อยู่ในรูปอย่างง่ายทีละพจน์:

**พจน์ที่ 1:** $\frac{\sqrt{54}}{\sqrt{2}}$
เรารู้ว่า $\sqrt{54} = \sqrt{9 \times 6} = 3\sqrt{6}$ หรือสามารถรวมรากที่สองได้เลย:
$$\frac{\sqrt{54}}{\sqrt{2}} = \sqrt{\frac{54}{2}} = \sqrt{27} = \sqrt{9 \times 3} = 3\sqrt{3}$$

**พจน์ที่ 2:** $-\frac{4}{\sqrt{8}}$
เรารู้ว่า $\sqrt{8} = \sqrt{4 \times 2} = 2\sqrt{2}$:
$$-\frac{4}{\sqrt{8}} = -\frac{4}{2\sqrt{2}} = -\frac{2}{\sqrt{2}}$$
จากนั้นทำให้ตัวส่วนเป็นจำนวนตรรกยะโดยการคูณด้วย $\frac{\sqrt{2}}{\sqrt{2}}$:
$$-\frac{2}{\sqrt{2}} \times \frac{\sqrt{2}}{\sqrt{2}} = -\frac{2\sqrt{2}}{2} = -\sqrt{2}$$

**พจน์ที่ 3:** $\frac{12\sqrt{3}}{\sqrt{6}}$
สามารถแยกตัวประกอบหรือรวมรากที่สองได้:
$$\frac{12\sqrt{3}}{\sqrt{6}} = 12\sqrt{\frac{3}{6}} = 12\sqrt{\frac{1}{2}} = \frac{12}{\sqrt{2}}$$
จากนั้นทำให้ตัวส่วนเป็นจำนวนตรรกยะโดยการคูณด้วย $\frac{\sqrt{2}}{\sqrt{2}}$:
$$\frac{12}{\sqrt{2}} \times \frac{\sqrt{2}}{\sqrt{2}} = \frac{12\sqrt{2}}{2} = 6\sqrt{2}$$

**พจน์ที่ 4:** $-\frac{18}{\sqrt{3}}$
ทำให้ตัวส่วนเป็นจำนวนตรรกยะโดยการคูณด้วย $\frac{\sqrt{3}}{\sqrt{3}}$:
$$-\frac{18}{\sqrt{3}} \times \frac{\sqrt{3}}{\sqrt{3}} = -\frac{18\sqrt{3}}{3} = -6\sqrt{3}$$

**รวมทุกพจน์เข้าด้วยกัน:**
แทนค่าพจน์ที่ทำให้ง่ายแล้วกลับเข้าไปในสมการ:
$$3\sqrt{3} - \sqrt{2} + 6\sqrt{2} - 6\sqrt{3}$$ 
จัดเรียงพจน์ตามรากที่เหมือนกัน:
$$(- \sqrt{2} + 6\sqrt{2}) + (3\sqrt{3} - 6\sqrt{3})$$
$$(5\sqrt{2}) + (-3\sqrt{3})$$
$$5\sqrt{2} - 3\sqrt{3}$$

**เทียบสัมประสิทธิ์:**
เรามี $5\sqrt{2} - 3\sqrt{3} = a\sqrt{2} + b\sqrt{3}$
เมื่อเปรียบเทียบสัมประสิทธิ์ของ $\sqrt{2}$ และ $\sqrt{3}$ เราจะได้:
$a = 5$
$b = -3$

**หาค่าของ $2a+b$:**
$$2a+b = 2(5) + (-3)$$
$$2a+b = 10 - 3$$
$$2a+b = 7$$

ดังนั้น ค่าของ $2a+b$ คือ $7$ ท่านสามารถยืนยันคำตอบโดยตรวจสอบการคำนวณแต่ละขั้นตอนอย่างละเอียด.
