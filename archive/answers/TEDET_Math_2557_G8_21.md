# คำอธิบายและวิธีทำ

จากเงื่อนไขในโจทย์ เราสามารถคำนวณหาปริมาตรของรูปทรงที่เกิดจากการหมุนกราฟ (รูปทรงกรวย) ได้ดังนี้:

### 1. หาปริมาตร $V_1$ (หมุนรอบแกน $x$):
* กราฟคือเส้นตรง $y = ax$ บนช่วง $0 \le x \le 2$ เมื่อหมุนรอบแกน $x$ จะเกิดเป็นกรวยตันในแนวนอน
* **พารามิเตอร์ของกรวย $V_1$:**
  * ส่วนสูงของกรวย ($h_1$) = $2$ (จาก $x=0$ ถึง $x=2$)
  * รัศมีที่ฐานกรวย ($r_1$) = ค่าของ $y$ ที่จุดปลาย $x=2$ คือ $a(2) = 2a$
* **คำนวณหาปริมาตร $V_1$:**
  $$V_1 = \frac{1}{3}\pi r_1^2 h_1$$
  $$V_1 = \frac{1}{3}\pi (2a)^2 (2) = \frac{8}{3}\pi a^2$$

---

### 2. หาปริมาตร $V_2$ (หมุนรอบแกน $y$):
* กราฟคือเส้นตรง $y = 3ax$ บนช่วง $0 \le x \le 3$ หมุนรอบแกน $y$ จะเกิดเป็นรูปทรงกรวยคว่ำในแนวตั้ง (มีจุดยอดที่จุดกำเนิด $O$)
* **พารามิเตอร์ของกรวย $V_2$:**
  * ส่วนสูงของกรวย ($h_2$) = ค่าของ $y$ ที่จุดปลาย $x=3$ คือ $3a(3) = 9a$
  * รัศมีที่ฐานกรวย ($r_2$) = $3$ (ซึ่งคือระยะห่างจากแกนหมุน $y$ ถึงเส้นขอบ $x=3$)
* **คำนวณหาปริมาตร $V_2$:**
  $$V_2 = \frac{1}{3}\pi r_2^2 h_2$$
  $$V_2 = \frac{1}{3}\pi (3)^2 (9a) = 27\pi a$$

---

### 3. หาอัตราส่วน $\frac{V_1}{V_2}$ และเปรียบเทียบค่าเพื่อหา $m + n$:
$$\frac{V_1}{V_2} = \frac{\frac{8}{3}\pi a^2}{27\pi a}$$
$$\frac{V_1}{V_2} = \frac{8a^2}{3 \times 27a} = \frac{8}{81}a$$

เทียบรูปแบบกับที่โจทย์ระบุคือ $\frac{n}{m}a$:
$$\frac{n}{m} = \frac{8}{81}$$

เนื่องจาก ห.ร.ม. ของ 8 และ 81 เท่ากับ 1 (อยู่ในรูปเศษส่วนอย่างต่ำแล้ว) และ $a > 0$
จะได้:
* $n = 8$
* $m = 81$

หาค่า $m + n$:
$$m + n = 81 + 8 = 89$$

ดังนั้น ค่าของ $m + n$ คือ **89**
