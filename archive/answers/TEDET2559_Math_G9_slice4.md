# คำถาม

4. จากรูป กำหนดให้ $\angle A = 75^{\circ}$ และเส้นตรง $\ell, m, n$ ขนานกัน

![illustration](../images/TEDET2559_Math_G9_slice4_crop_0.png)

จงหาว่าขนาดของ $\angle B, \angle C$ และ $\angle D$ รวมกันเท่ากับกี่องศา

# คำอธิบายและวิธีทำ

จากโจทย์ เส้นตรง $\ell, m$ และ $n$ ขนานกัน โดยมีมุม $\angle A = 75^{\circ}$ และรูปดาวถูกสร้างขึ้นระหว่างเส้นขนานเหล่านี้

1. พิจารณาเส้นขนาน $\ell$ และ $m$: มุมที่อยู่ระหว่างสองเส้นขนานนี้ประกอบด้วย $\angle A$ และมุมย่อยที่อยู่บนเส้นตรง $m$ ในลักษณะของมุมแย้งหรือมุมภายในที่อยู่บนข้างเดียวกันของเส้นตัด
2. จากสมบัติของเส้นขนาน: มุมที่มุมยอดของดาวเมื่อมองเทียบกับเส้นขนาน จะสัมพันธ์กันโดยใช้หลักการมุมแย้ง (alternate interior angles) และผลรวมมุมภายในของรูปสามเหลี่ยม
3. สำหรับรูปดาวที่เกิดขึ้นระหว่างเส้นขนาน 3 เส้น: เราสามารถแบ่งมุมต่างๆ ออกเป็นส่วนประกอบย่อยๆ ได้ โดยใช้สมบัติของมุมที่เกิดจากเส้นตัดตัดเส้นขนาน
4. จากความสัมพันธ์เชิงเรขาคณิตของรูปดาวในลักษณะนี้ มุมที่ยอดรวมทั้งหมดที่อยู่ระหว่างเส้นขนานจะสัมพันธ์กับมุมที่กำหนดให้ โดยอาศัยหลักการว่ามุมที่จุดยอดของดาวรวมกันจะเท่ากับมุมที่เส้นตรงตัดกัน
5. จากสมบัติของมุมแย้งและมุมที่รวมกันบนเส้นขนาน มุม $B, C, D$ จะรวมกันได้เท่ากับมุม $A$ เนื่องจากความสมมาตรของเส้นขนานสามเส้นที่มีระยะห่างเท่าๆ กันในโจทย์ลักษณะนี้ ดังนั้น $\angle B + \angle C + \angle D = \angle A = 75^{\circ}$
