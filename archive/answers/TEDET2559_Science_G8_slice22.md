# คำถาม

18. กราฟแสดงการเปลี่ยนแปลงปริมาตรตามเวลาที่แก๊ส A และแก๊ส B ทำปฏิกิริยากันในขณะที่อุณหภูมิและความดันคงที่แล้วได้เป็นแก๊ส C เมื่อแก๊ส A และแก๊ส B เป็นโมเลกุลที่ประกอบด้วยอะตอมอย่างละ 2 อะตอม

![illustration](../images/TEDET2559_Science_G8_slice22_crop_0.png)

ข้อใดแสดงจำนวนอะตอมที่ประกอบอยู่ในหนึ่งโมเลกุลของแก๊ส C และสมการเคมีได้ถูกต้อง

| จำนวนอะตอมของแก๊ส C | สมการเคมี |
| :--- | :--- |
| ① 2 | A + 3B $\rightarrow$ 2C |
| ② 4 | A + 3B $\rightarrow$ 2C |
| ③ 2 | A + 3B $\rightarrow$ 4C |
| ④ 4 | A + 3B $\rightarrow$ 4C |
| ⑤ 6 | 3A + 9B $\rightarrow$ 4C |

# คำอธิบายและวิธีทำ

จากกราฟการเปลี่ยนแปลงปริมาตรของแก๊สที่อุณหภูมิและความดันคงที่ ตามกฎของอาโวกาโดร ปริมาตรของแก๊สที่ทำปฏิกิริยากันจะแปรผันตรงกับจำนวนโมลของแก๊ส พิจารณาการเปลี่ยนแปลงปริมาตรจากกราฟ: แก๊ส A ลดลงจาก 30 mL เหลือ 10 mL (เปลี่ยนแปลง -20 mL), แก๊ส B ลดลงจาก 90 mL เหลือ 30 mL (เปลี่ยนแปลง -60 mL), และแก๊ส C เกิดขึ้นจาก 0 mL เป็น 40 mL (เปลี่ยนแปลง +40 mL) ดังนั้น อัตราส่วนโดยปริมาตรคือ A : B : C = 20 : 60 : 40 หรือทอนเป็นอัตราส่วนอย่างต่ำได้เป็น 1 : 3 : 2 ดังนั้นสมการเคมีคือ 1A + 3B -> 2C. โจทย์กำหนดว่า A และ B เป็นโมเลกุลที่ประกอบด้วย 2 อะตอม (เช่น A2 และ B2) พิจารณาจำนวนอะตอมรวม: ฝั่งซ้ายมีอะตอมของ A คือ 1(2) = 2 อะตอม และอะตอมของ B คือ 3(2) = 6 อะตอม รวมเป็น 8 อะตอม เมื่อเกิดเป็นสาร C จำนวน 2 โมเลกุล ตามกฎทรงมวล อะตอมรวมฝั่งขวาต้องเป็น 8 อะตอม ดังนั้นใน 1 โมเลกุลของ C จะมี 8 / 2 = 4 อะตอม
