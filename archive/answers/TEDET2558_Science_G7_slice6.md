# คำอธิบายและวิธีทำ

การเปลี่ยนแปลงรูปร่างปรากฏของดวงจันทร์ (Moon Phases) เกิดจากการที่ดวงจันทร์โคจรรอบโลก โดยตำแหน่งของดวงจันทร์ที่เปลี่ยนไปทำให้สัดส่วนของพื้นที่สว่างที่เรามองเห็นจากโลกเปลี่ยนไปในวัฏจักรประมาณ 29.5 วัน

1. วัฏจักรปกติเริ่มจากเดือนดับ (New Moon - มืดสนิท) -> จันทร์เสี้ยวข้างขึ้น (Waxing Crescent) -> จันทร์ครึ่งดวงข้างขึ้น (First Quarter) -> ดวงจันทร์ค่อนดวงข้างขึ้น (Waxing Gibbous) -> จันทร์เต็มดวง (Full Moon) -> ดวงจันทร์ค่อนดวงข้างแรม (Waning Gibbous) -> จันทร์ครึ่งดวงข้างแรม (Last Quarter) -> จันทร์เสี้ยวข้างแรม (Waning Crescent) -> กลับสู่เดือนดับ
2. วิเคราะห์ตัวเลือก:
- ตัวเลือก (1), (2), (4), และ (5) แสดงลำดับการเปลี่ยนแปลงของเสี้ยวสว่างที่เพิ่มขึ้นหรือลดลงอย่างต่อเนื่องและเป็นธรรมชาติ
- ตัวเลือก (3) เริ่มต้นจากดวงจันทร์เกือบเต็มดวง แล้วค่อยๆ กลายเป็นจันทร์เต็มดวง ซึ่งตามหลักการแล้ว หากเป็นช่วงที่ดวงจันทร์สว่างมากขึ้นเรื่อยๆ จนเต็มดวง สัดส่วนของพื้นที่สว่างที่แสดงในภาพนั้นไม่มีความสมเหตุสมผลในเชิงลำดับการโคจรต่อเนื่องเมื่อเปรียบเทียบกับภาพแรกๆ ที่แสดงเสี้ยวที่ค่อนข้างสว่างมากอยู่แล้ว การเปลี่ยนแปลงในภาพที่ 3 ไม่ได้แสดงรูปแบบการเปลี่ยนแปลงเสี้ยวที่สอดคล้องกับการเคลื่อนที่จริงของดวงจันทร์ในรอบข้างขึ้น-ข้างแรมที่ถูกต้องตามลำดับวัฏจักรธรรมชาติ
