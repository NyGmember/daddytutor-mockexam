# คำอธิบายและวิธีทำ

เรามี 3 ทีม คือ ไทย (T), มาเลเซีย (M), และอินเดีย (I) โดยมีการแข่งขันแบบพบกันหมด ทีมละ 1 นัด รวมทั้งหมด 3 นัด ได้แก่ (T vs M), (T vs I), และ (M vs I) กำหนดให้ $T_{M}, T_{I}$ คือประตูที่ไทยยิงมาเลเซียและอินเดียตามลำดับ และ $M_{T}, M_{I}$ คือประตูที่มาเลเซียยิงไทยและอินเดีย และ $I_{T}, I_{M}$ คือประตูที่อินเดียยิงไทยและมาเลเซีย
1. ข้อมูลไทย: ชนะ 2 นัด (ยิงรวม) และเสีย 2 ประตู หมายความว่าผลรวมประตูเสียของไทย $M_{T} + I_{T} = 2$
2. ข้อมูลมาเลเซีย: ได้ 4 ประตู เสีย 5 ประตู หมายความว่า $M_{T} + M_{I} = 4$ และ $T_{M} + I_{M} = 5$
3. ข้อมูลอินเดีย: เสมอ 1 นัด (น่าจะเป็นกับไทยหรือมาเลเซีย) ได้ 2 ประตู เสีย 8 ประตู หมายความว่า $I_{T} + I_{M} = 2$ และ $T_{I} + M_{I} = 8$

จากสมการประตูเสียของอินเดีย $T_{I} + M_{I} = 8$ และจากข้อมูลมาเลเซีย $M_{T} + M_{I} = 4$ เราพบว่า $M_{I} = 4 - M_{T}$
แทนค่าในสมการ $I_{T} + I_{M} = 2$ จะได้ $I_{T} + (5 - T_{M}) = 2$ หรือ $I_{T} - T_{M} = -3$
จากผลรวมประตูเสียของไทย $M_{T} + I_{T} = 2$ ดังนั้น $I_{T} = 2 - M_{T}$ แทนค่าลงไป: $(2 - M_{T}) - T_{M} = -3 \Rightarrow T_{M} + M_{T} = 5$
หากพิจารณาผลรวมประตูได้-เสีย ของทุกทีม รวมกันต้องเท่ากัน (ประตูได้รวม = ประตูเสียรวม) ซึ่งคือ $2+4+2 = 8$ และประตูเสียรวม $2+5+8 = 15$ ไม่เท่ากัน แสดงว่าต้องมีประตูที่ยิงได้เกินมา
จากการวิเคราะห์สมการ $I_{T} + I_{M} = 2$ ถ้าอินเดียได้ 2 ประตู และเสีย 8 ประตู โดยเสมอ 1 นัด (สมมติเสมอไทย $T_{I}=I_{T}$) จะได้ $I_{T} + I_{M} = 2$ และ $T_{I} = I_{T}$ ซึ่งถ้าให้ $I_{T}=1$ จะได้ $I_{M}=1$ ดังนั้นไทยเสีย 2 ประตูจากการที่มาเลเซียยิง 1 และอินเดียยิง 1 ซึ่งสอดคล้องกับ $I_{T}=1$
