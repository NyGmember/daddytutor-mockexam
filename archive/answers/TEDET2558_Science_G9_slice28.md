# คำถาม

27. ภาพแสดงเนฟรอนของไตและตารางแสดงความเข้มข้นของสาร (ก) (ข) (ค) และ (ง) ที่ตรวจสอบได้จากจุดสามจุด A, B และ C จากขั้นตอนการผลิตปัสสาวะ (C หมายถึงหลอดไต และ (ก), (ข), (ค), (ง) หมายถึงยูเรีย, เม็ดเลือดแดง, กลูโคส, กรดอะมิโน อย่างหนึ่งอย่างใดจากที่กล่าวมานี้)

![illustration](../images/TEDET2558_Science_G9_slice28_crop_0.png)

(หน่วย : g/100 mL)

| สาร / จุด | A | B | C |
| :--- | :--- | :--- | :--- |
| (ก) | 0.10 | 0.10 | 0.00 |
| (ข) | 0.05 | 0.05 | 0.00 |
| (ค) | 8.00 | 0.00 | 0.00 |
| (ง) | 0.03 | 0.03 | 1.80 |

ข้อใดกล่าวไม่ถูกต้อง

① (ค) ไม่ถูกกรองจาก A ไปยัง B

② ผู้ป่วยโรคโปรตีนสูงในปัสสาวะจะตรวจสอบพบ (ก) ที่ C

③ ความเข้มข้นของ (ง) ที่หลอดเลือดดำไตสูงกว่าที่หลอดเลือดแดงไต

④ ในปัสสาวะปกติจะไม่พบ (ก), (ข), (ค)

⑤ (ก), (ข), (ค) เป็นสารที่ถูกดูดซึมได้ 100 %

# คำอธิบายและวิธีทำ

หลักการทำงานของเนฟรอน (Nephron) ในไตมีดังนี้: จุด A คือบริเวณโกลเมอรูลัส (Glomerulus) ซึ่งเกิดการกรองสารผ่านหลอดเลือด, จุด B คือบริเวณโบว์แมนแคปซูล (Bowman's capsule) ซึ่งได้รับสารที่ผ่านการกรองมา, และ C คือหลอดไต (Tubule) ซึ่งเกิดการดูดกลับ (Reabsorption) และการหลั่งสาร (Secretion). พิจารณาจากตาราง: สาร (ค) คือโปรตีนขนาดใหญ่ (พบที่ A 8.00 แต่ไม่พบที่ B หรือ C) สาร (ง) คือยูเรีย (เพิ่มความเข้มข้นขึ้นมากที่ C) สาร (ก) และ (ข) คือสารที่ถูกกรองได้แต่ถูกดูดกลับจนหมดในคนปกติ. วิเคราะห์ตัวเลือก: 1) (ค) เป็นโปรตีนโมเลกุลใหญ่จึงไม่ถูกกรอง เป็นข้อความที่ถูกต้อง. 2) หากป่วยด้วยโรคไต (โปรตีนรั่ว) จะพบโปรตีนที่ C เป็นข้อความที่ถูกต้อง. 3) ที่หลอดเลือดดำไต (หลังผ่านการกรอง) ความเข้มข้นของยูเรียจะลดลงเมื่อเทียบกับหลอดเลือดแดงไต เพราะยูเรียส่วนหนึ่งถูกขับออกเป็นปัสสาวะ ดังนั้นข้อ 3 จึงเป็นข้อความที่ผิด เพราะที่หลอดเลือดดำไตควรมีความเข้มข้นต่ำกว่า. 4) ในปัสสาวะปกติจะไม่พบโปรตีนและสารอาหารสำคัญ เป็นข้อความที่ถูกต้อง. 5) (ก), (ข), (ค) คือสารที่ร่างกายปกติจะดูดกลับหรือป้องกันไม่ให้หลุดออกไปในปัสสาวะ (โปรตีนถูกกรองไม่ได้, กลูโคส/กรดอะมิโนถูกดูดกลับ 100%) เป็นข้อความที่ถูกต้อง
