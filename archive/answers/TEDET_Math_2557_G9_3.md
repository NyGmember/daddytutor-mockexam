# คำถาม

3. ให้ $\sqrt{800} = a\sqrt{2}$ และ $\sqrt{0.005} = b\sqrt{2}$
จงหาค่าของ $a^2 - 2ab + \frac{1}{b^2}$

# คำอธิบายและวิธีทำ

ในการแก้ปัญหาข้อนี้ เราจะต้องเริ่มจากการหาสมการของตัวแปร $a$ และ $b$ จากสมการที่กำหนดให้ จากนั้นจึงนำค่าของ $a$ และ $b$ ที่ได้ไปแทนค่าในนิพจน์ที่โจทย์ต้องการหาคำตอบ

**ขั้นที่ 1: หาสมการของตัวแปร $a$**
โจทย์กำหนดให้ $\sqrt{800} = a\sqrt{2}$ เราจะทำให้อยู่ในรูปที่สามารถเปรียบเทียบค่า $a$ ได้โดยการจัดรูปพจน์ทางซ้ายมือ:
$\sqrt{800} = \sqrt{400 \times 2}$
$= \sqrt{400} \times \sqrt{2}$
$= 20\sqrt{2}$
เมื่อนำไปเทียบกับ $a\sqrt{2}$ จะได้:
$20\sqrt{2} = a\sqrt{2}$
ดังนั้น $a = 20$

**ขั้นที่ 2: หาสมการของตัวแปร $b$**
โจทย์กำหนดให้ $\sqrt{0.005} = b\sqrt{2}$ เราจะจัดรูปพจน์ทางซ้ายมือเช่นเดียวกัน:
$\sqrt{0.005} = \sqrt{\frac{5}{1000}}$
$= \sqrt{\frac{1}{200}}$
$= \frac{\sqrt{1}}{\sqrt{200}}$
$= \frac{1}{\sqrt{100 \times 2}}$
$= \frac{1}{10\sqrt{2}}$
เพื่อทำให้ส่วนไม่ติดกรณฑ์ เราจะคูณด้วย $\frac{\sqrt{2}}{\sqrt{2}}$:
$= \frac{1}{10\sqrt{2}} \times \frac{\sqrt{2}}{\sqrt{2}}$
$= \frac{\sqrt{2}}{10 \times 2}$
$= \frac{\sqrt{2}}{20}$
เมื่อนำไปเทียบกับ $b\sqrt{2}$ จะได้:
$\frac{\sqrt{2}}{20} = b\sqrt{2}$
ดังนั้น $b = \frac{1}{20}$

**ขั้นที่ 3: แทนค่า $a$ และ $b$ ในนิพจน์ $a^2 - 2ab + \frac{1}{b^2}$**
เราทราบว่า $a = 20$ และ $b = \frac{1}{20}$ เราจะคำนวณแต่ละพจน์ในนิพจน์:
1.  $a^2 = (20)^2 = 400$
2.  $2ab = 2 \times (20) \times \left(\frac{1}{20}\right) = 2 \times 1 = 2$
3.  $\frac{1}{b^2} = \frac{1}{\left(\frac{1}{20}\right)^2} = \frac{1}{\frac{1}{400}} = 400$

นำค่าที่ได้ทั้งหมดมาแทนในนิพจน์:
$a^2 - 2ab + \frac{1}{b^2} = 400 - 2 + 400$
$= 798$

ดังนั้น ค่าของนิพจน์ $a^2 - 2ab + \frac{1}{b^2}$ คือ 798
