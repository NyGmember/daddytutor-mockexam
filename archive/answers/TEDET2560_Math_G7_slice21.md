# คำถาม

21. พิจารณาจำนวนนับตั้งแต่ 1 ถึง 100 ให้ $m$ เป็นผลบวกของจำนวนคี่ทุกตัว และ $n$ เป็นผลบวกของจำนวนคู่ทุกตัว

จงหาว่า $m - n$ มีค่าเท่าใด

(1) 50 (2) 25 (3) 0

(4) $-25$ (5) $-50$

# คำอธิบายและวิธีทำ

เราพิจารณาจำนวนนับตั้งแต่ 1 ถึง 100 ซึ่งประกอบด้วยตัวเลข 1, 2, 3, 4, ..., 99, 100 โดยแบ่งเป็น:

1. จำนวนคี่ คือ 1, 3, 5, ..., 99
2. จำนวนคู่ คือ 2, 4, 6, ..., 100

โจทย์กำหนดให้ $m$ คือผลบวกของจำนวนคี่ และ $n$ คือผลบวกของจำนวนคู่ ดังนั้นเราต้องการหาค่าของ $m - n$:
$m - n = (1 + 3 + 5 + ... + 99) - (2 + 4 + 6 + ... + 100)$

เราสามารถจัดกลุ่มตัวเลขโดยนำตัวเลขคี่แต่ละตัวมาลบกับจำนวนคู่ที่อยู่ถัดจากมัน:
$m - n = (1 - 2) + (3 - 4) + (5 - 6) + ... + (99 - 100)$

สังเกตว่าแต่ละวงเล็บจะมีค่าเท่ากับ $-1$ ดังนี้:
$(1 - 2) = -1$
$(3 - 4) = -1$
$(5 - 6) = -1$
...
$(99 - 100) = -1$

เนื่องจากมีจำนวนนับตั้งแต่ 1 ถึง 100 ทั้งหมด 100 ตัว แบ่งเป็นจำนวนคี่ 50 ตัว และจำนวนคู่ 50 ตัว ดังนั้นจะมีทั้งหมด 50 คู่ (วงเล็บ)

ดังนั้น $m - n = (-1) + (-1) + ... + (-1)$ (ทั้งหมด 50 ตัว)
$m - n = 50 \times (-1) = -50$
