# คำถาม

20. จากการสำรวจสมาชิกในครอบครัวของนักเรียนในโรงเรียนแห่งหนึ่ง ได้ผลดังต่อไปนี้

* นักเรียนที่มีพี่ชาย มี 34% ของนักเรียนทั้งหมด
* นักเรียนที่มีพี่สาว มี 27% ของนักเรียนทั้งหมด
* นักเรียนที่มีทั้งพี่ชายและพี่สาว มี 12% ของนักเรียนทั้งหมด

จงหาว่านักเรียนที่ไม่มีทั้งพี่ชายและพี่สาวนั้นคิดเป็นร้อยละเท่าใดของนักเรียนทั้งหมด

# คำอธิบายและวิธีทำ

เราสามารถใช้หลักการของเซต (Set Theory) พื้นฐานเพื่อแก้โจทย์ข้อนี้ได้ โดยกำหนดให้จำนวนนักเรียนทั้งหมดคือ 100% ให้ $A$ แทนเซตของนักเรียนที่มีพี่ชาย และ $B$ แทนเซตของนักเรียนที่มีพี่สาว จากโจทย์เราทราบค่าดังนี้:

1. จำนวนนักเรียนที่มีพี่ชาย $n(A) = 34\%$
2. จำนวนนักเรียนที่มีพี่สาว $n(B) = 27\%$
3. จำนวนนักเรียนที่มีทั้งพี่ชายและพี่สาว $n(A \cap B) = 12\%$

ขั้นแรก เราต้องหาจำนวนนักเรียนที่มีพี่ชายหรือพี่สาวอย่างน้อยหนึ่งคน โดยใช้สูตรการหาจำนวนสมาชิกของยูเนียนของสองเซต:
$n(A \cup B) = n(A) + n(B) - n(A \cap B)$
แทนค่าลงในสมการ:
$n(A \cup B) = 34\% + 27\% - 12\%$
$n(A \cup B) = 61\% - 12\% = 49\%$

ดังนั้น นักเรียนที่มีพี่ชายหรือพี่สาวอย่างน้อยหนึ่งคนคิดเป็น 49% ของนักเรียนทั้งหมด

สุดท้าย หาจำนวนนักเรียนที่ไม่มีทั้งพี่ชายและพี่สาว โดยนำนักเรียนทั้งหมด (100%) ลบด้วยจำนวนนักเรียนที่มีพี่ชายหรือพี่สาวอย่างน้อยหนึ่งคน:
จำนวนนักเรียนที่ไม่มีพี่ชายและพี่สาว $= 100\% - 49\% = 51\%$
