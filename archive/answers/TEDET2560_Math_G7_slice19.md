# คำถาม

19. ในการเดินทางท่องเที่ยวด้วยรถยนต์ของครอบครัวหนึ่ง วันแรกเดินทางได้น้อยกว่าครึ่งหนึ่งของระยะทางทั้งหมดอยู่ 50 กิโลเมตร วันที่สองเดินทางได้มากกว่าครึ่งหนึ่งของระยะทางที่เหลืออยู่ 20 กิโลเมตร และวันที่สามเดินทางได้ 100 กิโลเมตร ถ้ายงเหลือระยะทางที่ต้องเดินทางอีก 50 กิโลเมตร จงหาว่าระยะทางทั้งหมดเป็นกี่กิโลเมตร

# คำอธิบายและวิธีทำ

ให้ระยะทางทั้งหมดเท่ากับ $x$ กิโลเมตร

1. วันแรกเดินทางได้น้อยกว่าครึ่งหนึ่งของระยะทางทั้งหมดอยู่ 50 กม. หมายความว่าวันแรกเดินทางได้: $\frac{x}{2} - 50$ กม.
ดังนั้น ระยะทางที่เหลือหลังจบวันแรกคือ $x - (\frac{x}{2} - 50) = \frac{x}{2} + 50$ กม.

2. วันที่สองเดินทางได้มากกว่าครึ่งหนึ่งของระยะทางที่เหลืออยู่ 20 กม. หมายความว่าวันที่สองเดินทางได้: $\frac{1}{2}(\frac{x}{2} + 50) + 20 = \frac{x}{4} + 25 + 20 = \frac{x}{4} + 45$ กม.
ระยะทางที่เหลือหลังจบวันที่สองคือ $(\frac{x}{2} + 50) - (\frac{x}{4} + 45) = \frac{x}{4} + 5$ กม.

3. วันที่สามเดินทางได้ 100 กม. และยังมีระยะทางเหลืออีก 50 กม. รวมเป็นส่วนที่เหลือสุดท้ายคือ $100 + 50 = 150$ กม.

ดังนั้น เราตั้งสมการจากระยะที่เหลือหลังวันที่สองได้ว่า:
$\frac{x}{4} + 5 = 150$
$\frac{x}{4} = 145$
$x = 145 \times 4$
$x = 580$

ตรวจสอบคำตอบ:
วันแรก: $580/2 - 50 = 240$ กม. (เหลือ $580 - 240 = 340$ กม.)
วันที่สอง: $(340/2) + 20 = 170 + 20 = 190$ กม. (เหลือ $340 - 190 = 150$ กม.)
วันที่สามเดินทาง 100 กม. จะเหลือทางอีก $150 - 100 = 50$ กม. ซึ่งตรงกับโจทย์กำหนด
