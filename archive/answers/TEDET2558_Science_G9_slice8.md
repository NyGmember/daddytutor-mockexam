# คำอธิบายและวิธีทำ

เรามาวิเคราะห์แต่ละปรากฏการณ์ในโจทย์กันครับ:

1. ปรากฏการณ์ 'มิราจ' (Mirage) หรือการที่เห็นพื้นถนนในวันที่ร้อนจัดเหมือนมีน้ำขัง (ข้อ a): เกิดจากการหักเหของแสงผ่านอากาศที่มีอุณหภูมิและดัชนีหักเหต่างกัน ทำให้แสงหักเหโค้งขึ้นและเกิดการสะท้อนกลับหมดเมื่อแสงเดินทางจากอากาศที่มีความหนาแน่นมากไปยังอากาศที่มีความหนาแน่นน้อย (อากาศร้อนใกล้พื้น) ดังนั้นข้อ a จึงถูกต้อง

2. การมองเห็นปลาในอ่างเลี้ยงปลาใหญ่กว่าความเป็นจริง (ข้อ b): เป็นปรากฏการณ์การหักเหของแสงเมื่อเดินทางจากน้ำ (ความหนาแน่นมาก) ไปสู่อากาศ (ความหนาแน่นน้อย) ทำให้รังสีของแสงเบนออกจากเส้นปกติ ผู้สังเกตจึงมองเห็นภาพเสมือนของปลาที่ดูใหญ่กว่าและอยู่ตื้นกว่าตำแหน่งจริง ข้อ b จึงเกี่ยวข้องกับการหักเห

3. การใช้ไฟหน้ารถยนต์ (ข้อ c): นี่คือหลักการสะท้อนของแสงปกติผ่านกระจกเงาโค้งพาราโบลาเพื่อให้แสงขนานกัน ไม่ใช่การสะท้อนกลับหมดหรือการหักเห ข้อ c จึงไม่ถูกต้อง

4. การมองเห็นเหรียญหลังจากเทน้ำออก (ข้อ d): ขณะที่มีน้ำ แสงจากเหรียญจะหักเหทำให้เห็นเหรียญในตำแหน่งที่สูงขึ้น แต่เมื่อเทน้ำออก แสงจากเหรียญเดินทางเป็นเส้นตรงในตัวกลางเดียว (อากาศ) หากมุมมองไม่เหมาะสม แสงจะไม่สามารถเดินทางมายังตาผู้สังเกตได้ (ถูกขอบแก้วบัง) ซึ่งเกี่ยวข้องกับการหักเหของแสงโดยตรง ข้อ d จึงถูกต้อง

สรุป: ปรากฏการณ์ที่เกี่ยวข้องกับการหักเหและการสะท้อนกลับหมด ได้แก่ a, b และ d
