# คำถาม

3. จงหาผลลัพธ์ของ $110_{สอง} - 11_{สอง} + 1100_{สอง}$ ในระบบตัวเลขฐานสิบ

# คำอธิบายและวิธีทำ

โจทย์ข้อนี้ต้องการให้เราคำนวณผลลัพธ์ของนิพจน์ที่อยู่ในระบบเลขฐานสอง แล้วแปลงผลลัพธ์นั้นให้เป็นระบบเลขฐานสิบ

ขั้นตอนที่ 1: แปลงเลขฐานสองเป็นเลขฐานสิบ
เราจะแปลงแต่ละจำนวนจากฐานสองเป็นฐานสิบ โดยใช้หลักการกระจายค่าประจำหลัก:

*   $110_{สอง}$ = $(1 \times 2^2) + (1 \times 2^1) + (0 \times 2^0)$ = $(1 \times 4) + (1 \times 2) + (0 \times 1)$ = $4 + 2 + 0$ = $6_{สิบ}$
*   $11_{สอง}$ = $(1 \times 2^1) + (1 \times 2^0)$ = $(1 \times 2) + (1 \times 1)$ = $2 + 1$ = $3_{สิบ}$
*   $1100_{สอง}$ = $(1 \times 2^3) + (1 \times 2^2) + (0 \times 2^1) + (0 \times 2^0)$ = $(1 \times 8) + (1 \times 4) + (0 \times 2) + (0 \times 1)$ = $8 + 4 + 0 + 0$ = $12_{สิบ}$

ขั้นตอนที่ 2: คำนวณนิพจน์ในระบบเลขฐานสิบ
เมื่อเราแปลงจำนวนทั้งหมดเป็นฐานสิบแล้ว เราสามารถคำนวณนิพจน์ได้ดังนี้:

$110_{สอง} - 11_{สอง} + 1100_{สอง}$ ในฐานสิบ คือ $6 - 3 + 12$

$6 - 3 = 3$
$3 + 12 = 15$

ดังนั้น ผลลัพธ์คือ $15_{สิบ}$
