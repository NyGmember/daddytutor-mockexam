# คำถาม

11. ให้ $A = 3x - y, B = 4x + 3y$ และ $C = -2x + 5y$
จากประโยคสัญลักษณ์ต่อไปนี้
$$2A - \{B + 2C - (A + 2B)\} + 3C$$
เมื่อเขียนให้อยู่ในรูปผลสำเร็จจะได้เป็น $mx + ny$
จงหาค่าของ $m + n$

# คำอธิบายและวิธีทำ

ขั้นตอนการแก้โจทย์พีชคณิตนี้ มีลำดับดังนี้:\n\n1. จัดรูปนิพจน์ที่กำหนดให้:\nกำหนดให้ $E = 2A - \{B + 2C - (A + 2B)\} + 3C$ \n\n2. ถอดวงเล็บในส่วนที่ซ้อนกันก่อน (วงเล็บกลม):\n$E = 2A - \{B + 2C - A - 2B\} + 3C$ \n\n3. รวมพจน์ที่คล้ายกันในปีกกา:\n$E = 2A - \{-A - B + 2C\} + 3C$ \n\n4. กระจายเครื่องหมายลบเข้าในปีกกา:\n$E = 2A + A + B - 2C + 3C$ \n\n5. รวมพจน์ทั้งหมด:\n$E = 3A + B + C$ \n\n6. แทนค่า $A = 3x-y$, $B = 4x+3y$ และ $C = -2x+5y$ ลงในนิพจน์ที่ลดรูปแล้ว:\n$E = 3(3x-y) + (4x+3y) + (-2x+5y)$ \n$E = (9x - 3y) + (4x + 3y) + (-2x + 5y)$ \n\n7. รวมกลุ่มพจน์ $x$ และ $y$:\n$x: 9x + 4x - 2x = 11x$ \n$y: -3y + 3y + 5y = 5y$ \nจะได้ $E = 11x + 5y$ \n\n8. โจทย์กำหนดว่าผลสำเร็จอยู่ในรูป $mx + ny$ จะได้ $m = 11$ และ $n = 5$ \nดังนั้น ค่าของ $m + n = 11 + 5 = 16$
