# คำถาม

5. ให้ $a = \sqrt{200} + 7$ และ $b = \sqrt{50} - 8$ 
จงหาค่าของ $a - 2b$

# คำอธิบายและวิธีทำ

เราต้องการหาค่าของ $a - 2b$ โดยกำหนดให้ $a = \sqrt{200} + 7$ และ $b = \sqrt{50} - 8$ 

ขั้นตอนแรก เราควรทำให้อยู่ในรูปอย่างง่ายโดยการถอดรากที่สอง (Simplify the radicals):
$\sqrt{200} = \sqrt{100 \times 2} = 10\sqrt{2}$
$\sqrt{50} = \sqrt{25 \times 2} = 5\sqrt{2}$

ดังนั้น:
$a = 10\sqrt{2} + 7$
$b = 5\sqrt{2} - 8$

ต่อไป เราจะแทนค่าในสมการ $a - 2b$:
$a - 2b = (10\sqrt{2} + 7) - 2(5\sqrt{2} - 8)$

กระจายเลข 2 เข้าไปในวงเล็บของ $b$:
$a - 2b = 10\sqrt{2} + 7 - (10\sqrt{2} - 16)$
$a - 2b = 10\sqrt{2} + 7 - 10\sqrt{2} + 16$

นำพจน์ที่เหมือนกันมาคำนวณ:
$(10\sqrt{2} - 10\sqrt{2}) + (7 + 16) = 0 + 23 = 23$
