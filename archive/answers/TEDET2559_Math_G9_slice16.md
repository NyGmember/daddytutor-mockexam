# คำถาม

16. กำหนดให้ $a, b$ และ $c$ เป็นความยาวด้านของรูปสามเหลี่ยม

ข้อสรุปในข้อใดต่อไปนี้ถูกต้อง

(1) ถ้า $a^2 + b^2 = c^2$ แล้วรูปสามเหลี่ยมนี้เป็นรูปสามเหลี่ยมมุมฉาก

(2) ถ้า $a^2 + b^2 > c^2$ แล้วรูปสามเหลี่ยมนี้เป็นรูปสามเหลี่ยมมุมแหลม

(3) ถ้า $a^2 + b^2 < c^2$ แล้วรูปสามเหลี่ยมนี้เป็นรูปสามเหลี่ยมมุมป้าน

(4) ถ้า $c > b > a$ แล้ว $a + b > c > b - a$

(5) ถ้ารูปสามเหลี่ยมนี้เป็นรูปสามเหลี่ยมมุมฉากที่มี $b$ เป็นด้านตรงข้ามมุมฉากแล้ว $a^2 + c^2 = b^2$

# คำอธิบายและวิธีทำ

โจทย์ข้อนี้เป็นการทดสอบความเข้าใจเกี่ยวกับสมบัติของรูปสามเหลี่ยม โดยพิจารณาทีละข้อดังนี้: 1) ทฤษฎีบทพีทาโกรัสกล่าวว่า ถ้าสามเหลี่ยมที่มีด้านยาว a, b และ c โดย c เป็นด้านที่ยาวที่สุด และ a^2 + b^2 = c^2 แล้วจะเป็นสามเหลี่ยมมุมฉาก ข้อนี้จึงถูกต้อง 2) หาก a^2 + b^2 > c^2 แสดงว่าด้าน c สั้นกว่าด้านตรงข้ามมุมฉากในสามเหลี่ยมมุมฉาก ทำให้มุมตรงข้ามด้าน c มีขนาดน้อยกว่า 90 องศา จึงเป็นสามเหลี่ยมมุมแหลม ข้อนี้จึงถูกต้อง 3) หาก a^2 + b^2 < c^2 แสดงว่าด้าน c ยาวกว่าด้านตรงข้ามมุมฉาก ทำให้มุมตรงข้ามด้าน c มีขนาดมากกว่า 90 องศา จึงเป็นสามเหลี่ยมมุมป้าน ข้อนี้จึงถูกต้อง 4) อสมการของรูปสามเหลี่ยมระบุว่าผลรวมของความยาวด้านสองด้านใดๆ ต้องมากกว่าด้านที่สามเสมอ หาก c เป็นด้านที่ยาวที่สุด a + b > c เป็นจริง และจากความสัมพันธ์ของอสมการในรูปสามเหลี่ยม c - b < a ซึ่งก็คือ c < a + b และ c - a < b เป็นจริงเสมอ ข้อนี้จึงถูกต้อง 5) หาก b เป็นด้านตรงข้ามมุมฉาก ทฤษฎีบทพีทาโกรัสคือ a^2 + c^2 = b^2 ซึ่งก็ถูกต้องเช่นกัน เนื่องจากทุกข้อเป็นสมบัติที่ถูกต้องตามหลักเรขาคณิต (โดยเฉพาะในระดับมัธยมศึกษาตอนต้น) จึงสรุปได้ว่าทุกข้อเป็นข้อความที่ถูกต้องตามหลักการ
