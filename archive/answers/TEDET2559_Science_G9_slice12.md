# คำถาม

9. มีวงจรไฟฟ้าที่เชื่อมต่อดวงไฟ 3 ดวง กับถ่านไฟฉายแบบอนุกรมดังต่อไปนี้ เมื่อใช้ลวดตัวนำมาต่อระหว่างจุด P และ Q บนวงจรไฟฟ้าดังภาพต่อไปนี้ ![illustration](../images/TEDET2559_Science_G9_slice12_crop_0.png) จากตัวเลือก ข้อใดบ้างที่เป็นการเปลี่ยนแปลงที่ไม่ถูกต้อง 1 ความต่างศักย์ไฟฟ้าระหว่าง P และ Q เป็น 0 2 ความสว่างของดวงไฟ C ไม่เปลี่ยนแปลง 3 ความสว่างของดวงไฟ A สว่างขึ้น 4 กระแสไฟฟ้าที่ไหลผ่านวงจรไฟฟ้าเพิ่มขึ้น 5 ความต่างศักย์ไฟฟ้าที่ปลายทั้งสองข้างของดวงไฟ B ไม่เปลี่ยนแปลง

# คำอธิบายและวิธีทำ

เมื่อมีการต่อลวดตัวนำระหว่างจุด P และ Q จะส่งผลให้เกิดวงจรลัด (Short Circuit) คร่อมหลอดไฟ C ในวงจรขนานดังกล่าว กระแสไฟฟ้าส่วนใหญ่จะเลือกไหลผ่านลวดตัวนำที่มีความต้านทานต่ำมากแทนที่จะไหลผ่านหลอดไฟ C ทำให้หลอดไฟ C ดับลง (ความสว่างเปลี่ยนไป) ในขณะที่ภาพรวมของความต้านทานรวมของวงจรจะลดลง เพราะหลอดไฟ C ไม่ถูกนำมาคิดในความต้านทานรวม ทำให้กระแสรวมที่ไหลออกจากแหล่งกำเนิดไฟฟ้าเพิ่มขึ้น ส่งผลให้หลอดไฟ A และ B สว่างขึ้นเนื่องจากได้รับกระแสไฟฟ้ามากขึ้น สำหรับข้อความในตัวเลือก: 1) ถูกต้อง เพราะจุด P และ Q เชื่อมต่อกันด้วยลวดตัวนำที่มีความต้านทานเป็นศูนย์ ดังนั้นความต่างศักย์จึงเป็น 0 2) ไม่ถูกต้อง เพราะหลอดไฟ C จะดับ (ความสว่างเปลี่ยนไป) 3) ถูกต้อง เพราะกระแสรวมเพิ่มขึ้น ทำให้ A สว่างขึ้น 4) ถูกต้อง เพราะความต้านทานรวมของวงจรลดลง (เมื่อ C ถูกลัดวงจร) 5) ถูกต้อง เพราะเดิมหลอดไฟ B อยู่ในวงจรอนุกรมกับ A แต่เมื่อมีจุด P และ Q มาเชื่อมต่อแบบขนาน ศักย์ไฟฟ้าที่จุด P เปลี่ยนไป แต่เมื่อพิจารณาเทียบกับศักย์ไฟฟ้าที่ขั้วแหล่งกำเนิด การเปลี่ยนแปลงของกระแสในวงจรหลักที่เพิ่มขึ้นจะส่งผลต่อความต่างศักย์ที่ตกคร่อม B เพิ่มขึ้น อย่างไรก็ตาม คำถามถามหา 'การเปลี่ยนแปลงที่ไม่ถูกต้อง' ซึ่งข้อ 2 ชัดเจนที่สุดว่าการที่ C ดับถือเป็นการเปลี่ยนแปลงความสว่าง
