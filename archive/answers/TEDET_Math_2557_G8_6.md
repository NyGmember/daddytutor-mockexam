# คำถาม

6. ถ้า $a$ และ $b$ เป็นจำนวนเต็มที่ทำให้
$$2[3x - 3y - \{2x - (x - 5y)\}] + 2(3y - 4x) = ax + by$$
โดยที่ $x$ และ $y$ เป็นจำนวนจริงใด ๆ
จงหาค่าของ $a \times b$

# คำอธิบายและวิธีทำ

จัดรูปสมการทางฝั่งซ้ายโดยเริ่มยุบวงเล็บจากด้านในสุดออกไปด้านนอก:

1. พิจารณาวงเล็บปีกกาและวงเล็บด้านในสุด:
   $$\{2x - (x - 5y)\} = 2x - x + 5y = x + 5y$$

2. แทนค่ากลับลงในวงเล็บเหลี่ยม:
   $$3x - 3y - \{2x - (x - 5y)\} = 3x - 3y - (x + 5y)$$
   $$= 3x - 3y - x - 5y$$
   $$= 2x - 8y$$

3. แทนค่ากลับลงในสมการทางฝั่งซ้ายทั้งหมด:
   $$2[2x - 8y] + 2(3y - 4x) = 4x - 16y + 6y - 8x$$
   $$= (4x - 8x) + (-16y + 6y)$$
   $$= -4x - 10y$$

จากโจทย์ กำหนดให้สมการเท่ากับ $ax + by$ สำหรับทุกจำนวนจริง $x, y$:
$$-4x - 10y = ax + by$$

เมื่อเปรียบเทียบสัมประสิทธิ์ จะได้:
$$a = -4$$
$$b = -10$$

หาค่าของ $a \times b$:
$$a \times b = (-4) \times (-10) = 40$$

ดังนั้น ค่าของ $a \times b$ เท่ากับ $40$
