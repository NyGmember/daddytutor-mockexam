# คำถาม

12. จากประโยคสัญลักษณ์ต่อไปนี้

$$\{(6x^2 - 18xy) \div 3x\} - \{(4xy + 8y^2) \div (-2y)\}$$

เมื่อเขียนให้อยู่ในรูปผลสำเร็จจะได้เป็น $$ax + by$$
จงหาค่าของ $$a + b$$

# คำอธิบายและวิธีทำ

ขั้นตอนที่ 1: พิจารณาส่วนประกอบแรกของประโยคสัญลักษณ์คือ (6x^2 - 18xy) / 3x 
นำ 3x หารแต่ละพจน์ในวงเล็บจะได้: (6x^2 / 3x) - (18xy / 3x) = 2x - 6y

ขั้นตอนที่ 2: พิจารณาส่วนประกอบที่สองของประโยคสัญลักษณ์คือ (4xy + 8y^2) / (-2y)
นำ -2y หารแต่ละพจน์ในวงเล็บจะได้: (4xy / -2y) + (8y^2 / -2y) = -2x - 4y

ขั้นตอนที่ 3: นำผลลัพธ์จากทั้งสองส่วนมาลบกันตามโจทย์:
(2x - 6y) - (-2x - 4y)
= 2x - 6y + 2x + 4y

ขั้นตอนที่ 4: รวมพจน์ที่เหมือนกัน:
(2x + 2x) + (-6y + 4y) = 4x - 2y

โจทย์กำหนดให้ผลสำเร็จอยู่ในรูป ax + by ดังนั้นจะได้ a = 4 และ b = -2
ขั้นตอนที่ 5: หาค่า a + b
a + b = 4 + (-2) = 2
