# คำถาม

24. จากระบบสมการ $\begin{cases} \frac{4}{x} - \frac{3}{y} = 3 \\ ax - by = 3 \end{cases}$ และระบบสมการ $\begin{cases} \frac{2}{x} + \frac{6}{y} = -1 \\ ax + by = 5 \end{cases}$
ถ้าระบบสมการทั้งสองมีคำตอบเดียวกัน จงหาค่าของ $3(a+b)$

# คำอธิบายและวิธีทำ

เนื่องจากระบบสมการทั้งสองชุดมีคำตอบ $(x, y)$ เดียวกัน ดังนั้นคำตอบนี้ต้องสอดคล้องกับสมการทุกสมการในระบบ
เราสามารถหาค่าของ $x$ และ $y$ ก่อนได้ โดยใช้สมการสองสมการที่ไม่มีตัวแปร $a$ และ $b$:

$$\frac{4}{x} - \frac{3}{y} = 3 \quad \text{--- (สมการที่ 1)}$$
$$\frac{2}{x} + \frac{6}{y} = -1 \quad \text{--- (สมการที่ 2)}$$

---

### 1. หาค่า $x$ และ $y$:
สมมติให้ $u = \frac{1}{x}$ และ $v = \frac{1}{y}$ จะได้ระบบสมการเชิงเส้น:
$$4u - 3v = 3 \quad \text{--- (สมการที่ 3)}$$
$$2u + 6v = -1 \quad \text{--- (สมการที่ 4)}$$

คูณ (สมการที่ 4) ด้วย $2$:
$$4u + 12v = -2 \quad \text{--- (สมการที่ 5)}$$

นำ (สมการที่ 3) ลบด้วย (สมการที่ 5):
$$(4u - 3v) - (4u + 12v) = 3 - (-2)$$
$$-15v = 5$$
$$v = -\frac{5}{15} = -\frac{1}{3}$$

แทนค่า $v = -\frac{1}{3}$ ลงใน (สมการที่ 3):
$$4u - 3\left(-\frac{1}{3}\right) = 3$$
$$4u + 1 = 3$$
$$4u = 2 \implies u = \frac{1}{2}$$

เนื่องจาก $u = \frac{1}{x}$ และ $v = \frac{1}{y}$ จะได้:
* $x = \frac{1}{u} = 2$
* $y = \frac{1}{v} = -3$

---

### 2. หาค่า $a$ และ $b$:
นำคำตอบ $x = 2$ และ $y = -3$ แทนลงในสมการที่เหลือที่มีตัวแปร $a$ และ $b$:

สมการ $ax - by = 3$:
$$2a - (-3)b = 3 \implies 2a + 3b = 3 \quad \text{--- (สมการที่ 6)}$$

สมการ $ax + by = 5$:
$$2a + (-3)b = 5 \implies 2a - 3b = 5 \quad \text{--- (สมการที่ 7)}$$

บวก (สมการที่ 6) และ (สมการที่ 7):
$$(2a + 3b) + (2a - 3b) = 3 + 5$$
$$4a = 8 \implies a = 2$$

แทนค่า $a = 2$ ลงใน (สมการที่ 6):
$$2(2) + 3b = 3$$
$$4 + 3b = 3$$
$$3b = -1 \implies b = -\frac{1}{3}$$

---

### 3. หาค่าของ $3(a + b)$:
$$3(a + b) = 3\left(2 + \left(-\frac{1}{3}\right)\right)$$
$$3\left(\frac{5}{3}\right) = 5$$

ดังนั้น ค่าของ $3(a+b)$ คือ **5**
