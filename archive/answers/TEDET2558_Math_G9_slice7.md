# คำถาม

7. ให้ $x = -3 + \sqrt{7}$ จงหาค่าของ $x^2 + 6x + 7$

# คำอธิบายและวิธีทำ

จากโจทย์ที่กำหนดให้ $x = -3 + \sqrt{7}$ เราสามารถหาค่าของ $x^2 + 6x + 7$ ได้โดยใช้วิธีจัดรูปสมการ ดังนี้: ขั้นที่ 1: ย้ายข้างสมการจาก $x = -3 + \sqrt{7}$ ให้เป็น $x + 3 = \sqrt{7}$ ขั้นที่ 2: ยกกำลังสองทั้งสองข้างของสมการ $(x + 3)^2 = (\sqrt{7})^2$ จะได้ $x^2 + 6x + 9 = 7$ ขั้นที่ 3: จัดรูปสมการเพื่อหาค่าของ $x^2 + 6x$ โดยการลบ 9 ออกจากทั้งสองข้างของสมการ จะได้ $x^2 + 6x = 7 - 9$ ดังนั้น $x^2 + 6x = -2$ ขั้นที่ 4: นำค่า $x^2 + 6x = -2$ ไปแทนในนิพจน์ที่โจทย์ต้องการหาค่า คือ $x^2 + 6x + 7$ จะได้ $(-2) + 7 = 5$
