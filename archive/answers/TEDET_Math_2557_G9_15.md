# คำถาม

15. ถ้าจำนวนเต็ม $a, b$ และ $c$ สอดคล้องกับการแยกตัวประกอบต่อไปนี้
$$(3x+5)^2 - (x-4)^2 = (4x+a)(bx+c)$$
จงหาค่าของ $a+2b+3c$

# คำอธิบายและวิธีทำ

จากสมการที่โจทย์กำหนด:
$$(3x+5)^2 - (x-4)^2 = (4x+a)(bx+c)$$

เราสามารถแยกตัวประกอบของพหุนามทางซ้ายมือได้โดยใช้สูตรผลต่างกำลังสอง:
$$A^2 - B^2 = (A + B)(A - B)$$

ให้:
* $A = 3x + 5$
* $B = x - 4$

---

### 1. แยกตัวประกอบฝั่งซ้ายของสมการ:
* **หาพจน์ $A + B$:**
  $$A + B = (3x + 5) + (x - 4) = 4x + 1$$

* **หาพจน์ $A - B$:**
  $$A - B = (3x + 5) - (x - 4) = 3x + 5 - x + 4 = 2x + 9$$

ดังนั้น จะแยกตัวประกอบทางซ้ายมือได้เป็น:
$$(3x+5)^2 - (x-4)^2 = (4x+1)(2x+9)$$

---

### 2. เปรียบเทียบเพื่อหาค่าของตัวแปร $a, b$ และ $c$:
เปรียบเทียบกับรูปแบบ $(4x+a)(bx+c)$ ในโจทย์:
$$(4x+1)(2x+9) = (4x+a)(bx+c)$$

จากการเทียบสัมประสิทธิ์ของวงเล็บที่มีตัวแปร $x$ จะได้:
* วงเล็บแรก: $4x + 1 = 4x + a \implies a = 1$
* วงเล็บที่สอง: $2x + 9 = bx + c \implies b = 2$ และ $c = 9$

---

### 3. คำนวณหาค่าของ $a + 2b + 3c$:
$$a + 2b + 3c = 1 + 2(2) + 3(9)$$
$$a + 2b + 3c = 1 + 4 + 27 = 32$$

ดังนั้น ค่าของ $a + 2b + 3c$ คือ **32**
