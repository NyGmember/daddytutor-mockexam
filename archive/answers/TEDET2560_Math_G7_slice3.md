# คำถาม

3. จงหาว่า $(-1) + (-1)^2 + (-1)^3 + \dots + (-1)^{100}$ มีค่าเท่ากับข้อใดต่อไปนี้

(1) $-2$

(2) $-1$

(3) $0$

(4) $1$

(5) $2$

# คำอธิบายและวิธีทำ

พิจารณานิพจน์ $(-1) + (-1)^2 + (-1)^3 + \dots + (-1)^{100}$ ซึ่งเป็นผลบวกของเลขยกกำลังของ $-1$ โดยมีเงื่อนไขดังนี้: 1. ถ้าเลขชี้กำลังเป็นจำนวนคี่ $(-1)^{\text{คี่}} = -1$ 2. ถ้าเลขชี้กำลังเป็นจำนวนคู่ $(-1)^{\text{คู่}} = 1$ ดังนั้น นิพจน์นี้สามารถเขียนใหม่ได้เป็น: $(-1) + 1 + (-1) + 1 + \dots + (-1) + 1$ จะเห็นว่านิพจน์นี้ประกอบด้วยคู่ของ $(-1 + 1)$ ซ้ำกันไปเรื่อยๆ ตั้งแต่พจน์ที่ 1 ถึงพจน์ที่ 100 เนื่องจากมีทั้งหมด 100 พจน์ เมื่อจับคู่พจน์จะเป็น $(\text{-1 + 1}) + (-1 + 1) + \dots + (-1 + 1)$ จำนวน 50 คู่พอดี ซึ่งแต่ละคู่มีค่าเท่ากับ $0$ ดังนั้นผลรวมทั้งหมดคือ $0 + 0 + \dots + 0 = 0$
