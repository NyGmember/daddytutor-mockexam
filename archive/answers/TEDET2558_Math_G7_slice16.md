# คำถาม

16. มีน้ำเกลือเข้มข้น 6% ปริมาณ 200 กรัม จงหาว่าต้องเทน้ำเพิ่มลงไปอีกกี่กรัม จึงจะได้น้ำเกลือเข้มข้น 4%

# คำอธิบายและวิธีทำ

ขั้นตอนการคิด:
1. พิจารณาปริมาณเนื้อเกลือในน้ำเกลือเดิม: 
   มีน้ำเกลือ 200 กรัม ความเข้มข้น 6% หมายความว่ามีเกลืออยู่ 6% ของ 200 กรัม
   เนื้อเกลือ = $\frac{6}{100} \times 200 = 12$ กรัม

2. กำหนดให้เติมน้ำลงไป $x$ กรัม:
   เมื่อเติมน้ำเพิ่มเข้าไป เนื้อเกลือจะยังคงมีค่าเท่าเดิมคือ 12 กรัม แต่ปริมาณน้ำเกลือรวมจะเปลี่ยนไปเป็น $(200 + x)$ กรัม

3. สร้างสมการตามเงื่อนไขความเข้มข้นใหม่ที่ 4%:
   ความเข้มข้นคือ $\frac{\text{เนื้อเกลือ}}{\text{สารละลายทั้งหมด}} \times 100 = 4$
   $\frac{12}{200+x} \times 100 = 4$

4. แก้สมการหาค่า $x$:
   $\frac{1200}{200+x} = 4$
   $1200 = 4(200+x)$
   $1200 = 800 + 4x$
   $1200 - 800 = 4x$
   $400 = 4x$
   $x = 100$

ดังนั้น ต้องเติมน้ำเพิ่มลงไป 100 กรัม
