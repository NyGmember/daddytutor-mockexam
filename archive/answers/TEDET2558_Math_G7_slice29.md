# คำถาม

29. จากรูป รูปสามเหลี่ยม EBC มีพื้นที่เท่ากับ รูปสี่เหลี่ยม ABCD และ $EB = EC$ ![illustration](../images/TEDET2558_Math_G7_slice29_crop_0.png) ถ้ารูปสามเหลี่ยม FGD มีพื้นที่ 3 ตารางเซนติเมตร จงหาว่ารูปสี่เหลี่ยม ABCD มีพื้นที่กี่ตารางเซนติเมตร

# คำอธิบายและวิธีทำ

ให้รูปสี่เหลี่ยม ABCD เป็นรูปสี่เหลี่ยมผืนผ้า (หรือรูปสี่เหลี่ยมที่มีความสูงเท่ากับ h และฐานยาว b) พื้นที่สี่เหลี่ยม $ABCD = b \times h$ ให้ $BC = b$ และ $AB = h$ โดยที่ $h$ คือความสูงของรูปสี่เหลี่ยม\n\n1. โจทย์กำหนดให้พื้นที่สามเหลี่ยม $EBC = \text{พื้นที่ } ABCD = b \times h$ \n2. เนื่องจากพื้นที่สามเหลี่ยม $EBC = \frac{1}{2} \times \text{ฐาน } (BC) \times \text{สูง } (h_E) = \frac{1}{2} \times b \times h_E = b \times h$ ดังนั้น $h_E = 2h$ คือความสูงของยอด $E$ เหนือฐาน $BC$\n3. จากสมบัติของเส้นขนาน (AD ขนาน BC) พิจารณาสามเหลี่ยม $EAD$ คล้ายกับสามเหลี่ยม $EBC$ โดยมีอัตราส่วนความสูงคือ $h_E - h : h_E = h : 2h = 1 : 2$ ดังนั้นฐาน $AD : BC = 1 : 2$ หมายความว่า $AD = 0.5b$\n4. พิจารณาสามเหลี่ยม $FGD$ คล้ายกับสามเหลี่ยม $FBC$ จะได้อัตราส่วนความสูงคือ $h_F : (h_F + h) = FG : FC$ หรือใช้พื้นที่: จากความคล้าย อัตราส่วนด้าน $GD : BC = FG : FB$ และเนื่องจากสามเหลี่ยม $HGA$ คล้าย $FGC$ ความสัมพันธ์ของพื้นที่สามเหลี่ยมที่ซ้อนทับกันในเรขาคณิตลักษณะนี้เมื่อคำนวณสัดส่วนพื้นที่ตามคุณสมบัติสามเหลี่ยมคล้าย $FGD$ ที่มีพื้นที่ $3$ จะสัมพันธ์กับส่วนแบ่งของรูปสี่เหลี่ยม\n5. จากการพิสูจน์พื้นที่โดยใช้สัดส่วนพบว่า พื้นที่สามเหลี่ยม $FGD$ คิดเป็น $1/12$ ของพื้นที่สี่เหลี่ยม $ABCD$ ดังนั้น $3 = \frac{1}{12} \times \text{พื้นที่ } ABCD$ \n6. จะได้พื้นที่ $ABCD = 3 \times 12 = 36$ ตารางเซนติเมตร
