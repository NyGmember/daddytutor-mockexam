# คำถาม

7. เขียนจำนวนนับตั้งแต่ 1 ถึง 25 ลงในตารางขนาด $5 \times 5$ โดยเริ่มจาก 1 ในช่องซ้ายล่าง แล้วเขียนจำนวนถัดไปแต่ละจำนวนในช่องที่มีด้านติดกับช่องเดิมไปเรื่อยๆ ดังที่แสดงให้เห็นเพียงบางจำนวนในตารางด้านล่าง ![illustration](../images/TEDET2560_Math_G7_slice7_crop_0.png) เมื่อเขียนจำนวนจนครบแล้ว A จะแทนจำนวนใด

# คำอธิบายและวิธีทำ

เพื่อให้ได้ผลลัพธ์ เราต้องวิเคราะห์รูปแบบการเดิน (path) ในตาราง 5x5 โดยระบุพิกัดของแต่ละช่องเป็น (แถว, คอลัมน์) จากมุมซ้ายล่างคือ (1, 1) ถึงมุมขวาบนคือ (5, 5). เริ่มต้นที่ 1 อยู่ที่ (1, 1), 2 อยู่ที่ (2, 1), 4 อยู่ที่ (1, 2). จากตารางที่ให้มา เราพบเส้นทางการเคลื่อนที่ที่สม่ำเสมอ: 1(1,1) -> 2(2,1) -> 3(3,1) -> 4(1,2) [ต้องสังเกตตำแหน่ง 4]. จากการตรวจสอบการเรียงตัว: เมื่อเราเริ่มจากซ้ายล่าง และตัวเลขถัดไปอยู่ช่องที่ติดกัน เราจะได้รูปแบบการเวียน: Column 1: 1, 2, 3, 10, 9. Column 2: 4, 5, 11, 12, 8. Column 3: 6, 7, 18, 17, 16. Column 4: 15, 14, 13, 19, 20. Column 5: 25, 24, 23, 22, 21. ตามตารางที่โจทย์ให้มา: (1,1)=1, (2,1)=2, (5,1)=10. สังเกต (3,3)=7, (4,4)=14, (5,4)=22, (5,5)=21. ไล่ลำดับตามโจทย์: 1(1,1), 2(2,1), 3(3,1), 4(1,2), 5(2,2), 6(3,2), 7(3,3), 8(4,3), 9(5,3), 10(2,1-ผิดลำดับ, แก้ไข): แท้จริงแล้วคือ 1(1,1) -> 2(2,1) -> 3(3,1) -> 4(1,2) -> 5(2,2) -> 6(3,2) -> 7(3,3) -> 8(4,3) -> 9(5,3) -> 10(2,1...เลข 10 อยู่ที่ row 2 col 1) สรุปจาก pattern การเดินเลขจนครบช่อง: A คือช่อง (1,5) ซึ่งตรงกับจำนวนสุดท้ายในลำดับคือ 25
