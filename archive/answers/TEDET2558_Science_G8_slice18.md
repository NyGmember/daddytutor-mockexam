# คำถาม

17. กราฟแสดงความสัมพันธ์ระหว่างอุณหภูมิและปริมาณไอน้ำอิ่มตัว

ปริมาณไอน้ำอิ่มตัว (g/m³)
![illustration](../images/TEDET2558_Science_G8_slice18_crop_0.png)

ข้อใดอธิบายเกี่ยวกับกราฟไม่ถูกต้อง

① จาก A – C อากาศที่มีอุณหภูมิจุดน้ำค้างสูงที่สุดคือ A
② ถ้าทำให้อากาศ A 1 m³ เย็นลง 10°C ไอน้ำ 13.1 g จะกลั่นตัว
③ อุณหภูมิจุดน้ำค้างในอากาศ A เท่ากับ 20°C และอุณหภูมิจุดน้ำค้างในอากาศ C เท่ากับ 10°C
④ อุณหภูมิจุดน้ำค้างในอากาศ B และ C เท่ากัน
⑤ ปริมาณไอน้ำปัจจุบันในอากาศ C เท่ากับปริมาณไอน้ำอิ่มตัว

# คำอธิบายและวิธีทำ

หลักการสำคัญของกราฟนี้คือการแสดงค่าความชื้นอิ่มตัว ณ อุณหภูมิต่างๆ สำหรับข้อ (4) ที่ระบุว่า 'อุณหภูมิจุดน้ำค้างในอากาศ B และ C เท่ากัน' นั้นไม่ถูกต้อง เนื่องจากการหาอุณหภูมิจุดน้ำค้าง (Dew Point) ทำได้โดยลากเส้นจากจุดสภาวะของอากาศในกราฟ (ในที่นี้คือจุด A, B, C) ไปทางซ้ายในแนวระดับจนตัดกับเส้นโค้งอิ่มตัว อุณหภูมิที่จุดตัดนั้นคืออุณหภูมิจุดน้ำค้าง:
1. จุด C อยู่บนเส้นโค้งอิ่มตัวพอดี ดังนั้นอุณหภูมิจุดน้ำค้างคือ 10°C
2. จุด A มีค่าปริมาณไอน้ำประมาณ 17.3 g/m³ ซึ่งตรงกับจุดอิ่มตัวที่ 20°C ดังนั้นจุดน้ำค้างคือ 20°C
3. จุด B มีค่าปริมาณไอน้ำเท่ากับ 10 g/m³ (ต่ำกว่า C) ดังนั้นเมื่อลากเส้นไปทางซ้าย จุดตัดบนเส้นโค้งจะอยู่ก่อนถึง 10°C (ประมาณ 8-9°C) จึงสรุปได้ว่าอุณหภูมิจุดน้ำค้างของ B และ C ไม่เท่ากัน
