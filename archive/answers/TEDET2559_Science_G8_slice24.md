# คำถาม

20. ภาพ (ก) แสดงหินสามชนิดและภาพ (ข) แสดงวัฏจักรหิน

(ก)
![illustration](../images/TEDET2559_Science_G8_slice24_crop_0.png)
A : หินบะซอลต์
B : หินไนส์
C : หินทราย

(ข)
![illustration](../images/TEDET2559_Science_G8_slice24_crop_1.png)

จากภาพที่กำหนดให้ ข้อใดบ้างที่อธิบายเกี่ยวกับภาพได้ถูกต้อง
(1) A ปรากฏอยู่ในขั้นตอน Y
(2) B ประกอบอยู่ในขั้นตอน (a)
(3) C สังเกตพบได้ในขั้นตอน X
(4) จาก A B C หินที่เกิดจากแรงกดดันสูงที่สุด คือ A
(5) ในขั้นตอน (b) มีการผุพังอยู่กับที่และการกัดกร่อนเกิดขึ้น

# คำอธิบายและวิธีทำ

พิจารณาวัฏจักรหินตามแผนภาพในภาพ (ข): 
1. X คือ หินตะกอน, Y คือ หินอัคนี (เพราะมาจากแมกมา), (a) คือ กระบวนการแปรสภาพ, (b) คือ การผุพังอยู่กับที่และการกัดกร่อน
- (1) ไม่ถูกต้อง: A (หินบะซอลต์) เป็นหินอัคนีพุ ซึ่งเกิดจากแมกมา (หรือลาวา) ที่เย็นตัวลง ไม่ใช่หินตะกอน (Y)
- (2) ไม่ถูกต้อง: B (หินไนส์) เป็นหินแปร ไม่ใช่หินตะกอน ซึ่งขั้นตอน (a) คือกระบวนการแปรสภาพจากหินตะกอนไปเป็นหินแปร
- (3) ถูกต้อง: C (หินทราย) เป็นหินตะกอน ซึ่งสังเกตพบได้ในขั้นตอน X
- (4) ไม่ถูกต้อง: หินที่เกิดจากแรงกดดันสูงที่สุดคือหินแปร (เช่น หินไนส์ B) ไม่ใช่หินบะซอลต์ (A) ซึ่งเป็นหินอัคนี
- (5) ถูกต้อง: ขั้นตอน (b) คือกระบวนการที่หินชนิดต่างๆ ถูกทำลายผ่านการผุพังและการกัดกร่อนจนกลายเป็นตะกอน
