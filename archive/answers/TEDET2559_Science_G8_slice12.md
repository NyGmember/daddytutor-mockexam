# คำถาม

11. ภาพแสดงหน้าตัดของรากและเส้นทางการเคลื่อนที่ของน้ำ

![illustration](../images/TEDET2559_Science_G8_slice12_crop_0.png)

จากตัวเลือก ข้อใดบ้างที่อธิบายเกี่ยวกับภาพที่กำหนดให้ได้ถูกต้องทั้งหมด

① C เชื่อมโยงไปจนถึงเส้นใบของใบไม้
② เซลล์ที่ประกอบเป็น C มีรูเล็ก ๆ ที่ผนังเซลล์
③ สิ่งที่เคลื่อนที่จาก A ไป C คือน้ำและสารอนินทรีย์
④ ความเข้มข้นเพิ่มสูงขึ้นจาก A > B > C ตามลำดับ
⑤ B มีพื้นที่ผิวกว้าง ช่วยดูดซึมน้ำในดินได้อย่างมีประสิทธิภาพ

# คำอธิบายและวิธีทำ

จากภาพแสดงหน้าตัดของราก (Root cross-section) สามารถวิเคราะห์โครงสร้างและหน้าที่ได้ดังนี้: A คือขนราก (root hair) ซึ่งมีหน้าที่เพิ่มพื้นที่ผิวในการดูดซึมน้ำและแร่ธาตุ, B คือชั้นคอร์เทกซ์ (cortex) ของราก, และ C คือท่อลำเลียงน้ำ (xylem) ซึ่งอยู่บริเวณใจกลางราก พิจารณาจากข้อความแต่ละข้อ: 1) C คือไซเลม ซึ่งทำหน้าที่ลำเลียงน้ำจากรากขึ้นสู่ลำต้นและไปถึงใบจริง 2) เซลล์ที่ประกอบเป็นไซเลม (tracheary elements) จะมีผนังหนาและมีรูเล็กๆ (pits) เพื่อให้เกิดการลำเลียงน้ำระหว่างเซลล์ในแนวข้างได้ถูกต้อง 3) สิ่งที่เคลื่อนที่จาก A เข้าสู่ C คือน้ำและสารละลายแร่ธาตุ (สารอนินทรีย์) ซึ่งถูกต้อง 4) ความเข้มข้นของสารละลายในดินมักจะต่ำกว่าในราก เพื่อให้น้ำออสโมซิสจากดินเข้าสู่รากได้ ดังนั้นความเข้มข้นจะสูงขึ้นจากดินเข้าสู่ภายในราก (A ไป C) ซึ่งทำให้ข้อนี้ผิดในแง่ของลำดับ (จากภาพคือการเคลื่อนที่เข้าสู่ศูนย์กลาง) 5) B คือชั้นคอร์เทกซ์ ทำหน้าที่สะสมอาหารและลำเลียงน้ำเข้าสู่ท่อลำเลียง ไม่ใช่ส่วนที่มีหน้าที่ดูดซึมโดยตรง (หน้าที่ดูดซึมคือขนราก) ดังนั้น ข้อที่ถูกต้องที่สุดคือ 1, 2 และ 3
