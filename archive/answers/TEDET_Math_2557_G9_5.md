# คำถาม

5. เมื่อแยกตัวประกอบ $6x^2 - 7xy - 20y^2$ ในรูปผลคูณของพหุนามดีกรีหนึ่งสองตัว ถ้าผลบวกของพหุนามดีกรีหนึ่งทั้งสองตัวเป็น $ax + by$ จงหาค่าของ $a + b$

# คำอธิบายและวิธีทำ

ต้องการแยกตัวประกอบของพหุนาม $6x^2 - 7xy - 20y^2$

พิจารณาการแยกตัวประกอบในรูป:
$$6x^2 - 7xy - 20y^2 = (px + qy)(rx + sy)$$

โดยที่:
1. $p \times r = 6$
2. $q \times s = -20$
3. $ps + qr = -7$

เมื่อทดลองหาคู่ลำดับที่สอดคล้อง จะได้:
* เลือก $p = 2, r = 3$ (เนื่องจาก $2 \times 3 = 6$)
* เลือก $q = -5, s = 4$ (เนื่องจาก $-5 \times 4 = -20$)

ตรวจสอบพจน์กลาง:
$$ps + qr = (2)(4) + (-5)(3) = 8 - 15 = -7$$
ซึ่งถูกต้องตรงตามโจทย์

ดังนั้น การแยกตัวประกอบของ $6x^2 - 7xy - 20y^2$ คือ:
$$(2x - 5y)(3x + 4y)$$

จากโจทย์ พหุนามดีกรีหนึ่งทั้งสองตัวคือ $2x - 5y$ และ $3x + 4y$
หาผลบวกของพหุนามดีกรีหนึ่งทั้งสองตัว:
$$(2x - 5y) + (3x + 4y) = (2 + 3)x + (-5 + 4)y = 5x - y$$

โจทย์กำหนดให้ผลบวกนี้เท่ากับ $ax + by$
จากการเปรียบเทียบสัมประสิทธิ์ จะได้:
* $a = 5$
* $b = -1$

ดังนั้น:
$$a + b = 5 + (-1) = 4$$
