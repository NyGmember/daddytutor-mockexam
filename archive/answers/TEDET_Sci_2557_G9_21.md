# คำถาม

21. จากภาพเป็นการต่อหลอดไฟ 3 หลอด

[![illustration](../images/TEDET_Sci_2557_G9_21_crop_0.png)]

ข้อใดแสดงการต่อวงจรแบบอนุกรมและแบบขนานของหลอดไฟทั้งหมดได้ถูกต้องตามลำดับ

① A และ B
② A และ C
③ B และ A
④ B และ C
⑤ C และ A

# คำอธิบายและวิธีทำ

พิจารณาลักษณะการต่อหลอดไฟในแต่ละวงจร (A, B และ C) ดังนี้:

1. **วงจร A (การต่อแบบอนุกรม - Series Connection):**
   - หลอดไฟทั้ง 3 หลอดต่อเรียงกันเป็นเส้นเดียวในวงจร กระแสไฟฟ้าจะไหลผ่านหลอดไฟทุกหลอดในทิศทางเดียวกันและมีค่าเท่ากัน หากมีหลอดใดหลอดหนึ่งขาด วงจรจะเปิดและไม่มีกระแสไฟฟ้าไหลผ่านหลอดที่เหลือเลย
   - ดังนั้น **วงจร A คือ การต่อแบบอนุกรมของหลอดไฟทั้งหมด**

2. **วงจร B (การต่อแบบขนาน - Parallel Connection):**
   - หลอดไฟทั้ง 3 หลอดต่อแยกคร่อมขนานกัน โดยขั้วข้างหนึ่งของหลอดทุกหลอดต่อร่วมกันเข้ากับขั้วหนึ่งของแหล่งจ่าย และขั้วอีกข้างหนึ่งต่อร่วมกันเข้ากับอีกขั้วหนึ่งของแหล่งจ่าย ทำให้หลอดไฟทุกหลอดได้รับแรงดันไฟฟ้าเท่ากันและทำงานอย่างเป็นอิสระต่อกัน
   - ดังนั้น **วงจร B คือ การต่อแบบขนานของหลอดไฟทั้งหมด**

3. **วงจร C (การต่อแบบผสม - Series-Parallel Connection):**
   - มีหลอดไฟ 2 หลอดต่ออนุกรมกันที่สาขาด้านบน แล้วสาขานี้มาต่อขนานกับหลอดไฟอีก 1 หลอดที่สาขาตรงกลาง

เมื่อโจทย์ถามหา "การต่อวงจรแบบอนุกรมและแบบขนานของหลอดไฟทั้งหมดตามลำดับ" คำตอบจึงเป็น **A และ B**

ตรงกับตัวเลือก **1**
