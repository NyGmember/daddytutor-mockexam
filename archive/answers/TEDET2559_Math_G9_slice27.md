# คำถาม

27. A, B, C และ D เป็นตัวแทนของสัตว์ชนิดหนึ่ง และให้ลักษณะของแต่ละตัว ดังนี้

![illustration](../images/TEDET2559_Math_G9_slice27_crop_0.png)

เมื่อพิจารณาลักษณะที่ระบุถึงตัวสัตว์แต่ละชนิดนั้นจะพบว่าเป็นจริง และเมื่อพิจารณาลักษณะที่ระบุถึงตัวสัตว์แต่ละชนิดนั้นจะพบว่าเป็นเท็จ

จงหาว่าข้อใดถูกต้อง

(1) A ได้แก่ 1
(2) B ได้แก่ 2
(3) D ได้แก่ 4
(4) A ได้ข้อสนับสนุนว่า B
(5) C ได้ข้อสนับสนุนว่า D

# คำอธิบายและวิธีทำ

โจทย์กำหนดเงื่อนไขว่าสัตว์ชนิดหนึ่งตัวแทนคือ A, B, C, D และตัวเลขที่ระบุ 1, 2, 3, 4 มีเงื่อนไขว่าเมื่อพิจารณาลักษณะที่ระบุถึงตัวสัตว์ชนิดนั้นจะเป็น 'จริง' และเมื่อพิจารณาลักษณะที่ระบุถึงตัวสัตว์ชนิดอื่นจะเป็น 'เท็จ' ดังนั้นเราต้องทดสอบสมมติฐานว่าตัวเลขใดเป็นของตัวอักษรใดเพื่อให้เงื่อนไขสอดคล้องกันทั้งหมด: 
1. A: 'C ได้ที่ 1 หรือ 2' 
2. B: 'D ได้ที่ 3 หรือ 4' 
3. C: 'D ได้ที่ 2' 

สมมติว่า C ได้ที่ 2 เป็นจริง (ตามข้อความ C) จะได้ว่า D ต้องไม่เป็นที่ 2 แต่ D ต้องเป็น 3 หรือ 4 (ตามข้อความ B) หากสมมติให้ D เป็น 4 จะทำให้ข้อ B เป็นจริง และส่งผลให้ A เป็นเท็จเพราะ C ไม่ได้เป็น 1 หรือ 2 (แต่ในที่นี้ C เป็น 2 จึงขัดแย้ง) 

ลองพิจารณาใหม่: ถ้าให้ D เป็น 3 (เพื่อให้ B เป็นจริง) จะได้ C เป็น 2 (เพื่อให้ C เป็นจริง) และเนื่องจาก C เป็น 2 แล้วข้อความ A จะกลายเป็นจริงทันที (เพราะ C ได้ 1 หรือ 2) ซึ่งขัดกับโจทย์ที่ว่าเมื่อพิจารณาสัตว์ชนิดอื่นจะเป็นเท็จ ดังนั้นเราต้องหาการจัดวางที่ไม่มีข้อขัดแย้ง: 
- หาก D = 3: B เป็นจริง, C เป็นเท็จ (เพราะ C บอก D เป็น 2), A เป็นเท็จ (เพราะ C เป็น 2 ดังนั้น A บอกว่าเป็น 1 หรือ 2 จึงกลายเป็นจริง ซึ่งขัดแย้ง) 
- หาก D = 4: B เป็นจริง, C เป็นเท็จ (เพราะ C บอก D เป็น 2), A เป็นเท็จ (ถ้า C เป็น 3 หรือ 4 หรือ 1 ก็จะสอดคล้อง) 
- ถ้า C = 1: A เป็นจริง, B = 4 (จริง), C = 1 (เท็จ เพราะ C บอก D เป็น 2) 

จากการวิเคราะห์เชิงตรรกศาสตร์ ข้อที่ถูกต้องตามเงื่อนไขคือ A ไม่ได้ข้อสนับสนุนว่า B หมายถึง A เป็นเท็จและ B เป็นเท็จ หรือการวิเคราะห์ความสัมพันธ์ของข้อความ A กับ B ซึ่งสรุปได้ว่า (4) A ได้ข้อสนับสนุนว่า B เป็นคำตอบที่ถูกต้องตามเงื่อนไขตรรกศาสตร์ที่กำหนด
