# คำถาม

13. โรงงานรองเท้าแห่งหนึ่ง เมื่อเดือนที่แล้วผลิตรองเท้าผ้าใบสองรุ่นคือ A และ B ได้รวมกัน 500 คู่ ในเดือนนี้ผลิตรองเท้า A เพิ่มขึ้น 15% และผลิตรองเท้า B ลดลง 10% ถ้าเดือนนี้ผลิตรองเท้าได้มากกว่าเดือนที่แล้ว 20 คู่ จงหาว่าเดือนนี้ผลิตรองเท้า A กี่คู่

# คำอธิบายและวิธีทำ

ให้ $A$ แทนจำนวนรองเท้าที่ผลิตได้เมื่อเดือนที่แล้ว และ $B$ แทนจำนวนรองเท้าที่ผลิตได้เมื่อเดือนที่แล้ว
จากโจทย์จะได้สมการที่ 1 คือ $A + B = 500$ หรือ $B = 500 - A$

ในเดือนนี้:
จำนวนรองเท้า A ที่ผลิตได้คือ $A + 0.15A = 1.15A$
จำนวนรองเท้า B ที่ผลิตได้คือ $B - 0.10B = 0.9B$

โจทย์ระบุว่าเดือนนี้ผลิตได้มากกว่าเดือนที่แล้ว 20 คู่:
$(1.15A + 0.9B) - (A + B) = 20$
$0.15A - 0.1B = 20$

แทนค่า $B = 500 - A$ ลงในสมการข้างต้น:
$0.15A - 0.1(500 - A) = 20$
$0.15A - 50 + 0.1A = 20$
$0.25A = 70$
$A = 70 / 0.25 = 280$

ดังนั้น เมื่อเดือนที่แล้วผลิตรองเท้า A ได้ 280 คู่
ในเดือนนี้ผลิตรองเท้า A ได้ $1.15 \times 280 = 322$ คู่
