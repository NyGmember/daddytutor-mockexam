# คำถาม

11. มดตัวหนึ่งจะเดินบนเส้นขอบของทรงสี่หน้าปกติจากจุดยอดหนึ่งไปยังอีกจุดยอดหนึ่ง โดยแต่ละครั้งจะเดินทีละหนึ่งเส้นขอบ

![illustration](../images/TEDET2560_Math_G9_slice11_crop_0.png)

มดเริ่มต้นอยู่ที่จุดยอดจุดหนึ่ง ความน่าจะเป็นที่เดิน 3 ครั้ง แล้วจะกลับมาที่จุดยอดเดิมคือข้อใดต่อไปนี้

(1) $\frac{1}{27}$ (2) $\frac{2}{27}$ (3) $\frac{1}{9}$

(4) $\frac{5}{27}$ (5) $\frac{2}{9}$

# คำอธิบายและวิธีทำ

ให้ $V_0$ เป็นจุดยอดเริ่มต้นที่มดอยู่ ทรงสี่หน้าปกติมีจุดยอด 4 จุด โดยแต่ละจุดเชื่อมต่อกับจุดยอดอีก 3 จุดที่เหลือ

1. การเดินครั้งที่ 1: มดต้องเดินจาก $V_0$ ไปยังจุดยอดข้างเคียงจุดใดจุดหนึ่ง จากทั้งหมด 3 จุดที่เป็นไปได้ ดังนั้นมีโอกาสเดินไปได้ 3 เส้นทาง แต่ละเส้นทางมีความน่าจะเป็น $1/3$ ในการเลือกเดิน

2. การเดินครั้งที่ 2: หลังจากมดไปอยู่ที่จุดยอดใหม่ (สมมติว่าเป็น $V_1$) มดต้องเลือกเดินต่อไปอีก 3 เส้นทางที่เป็นไปได้จาก $V_1$ เพื่อจะกลับมาที่จุดเดิม ($V_0$) ได้ในการเดินครั้งที่ 3 มดจะต้องเดินไปที่จุดยอดที่เชื่อมกับ $V_0$ เท่านั้น

3. วิเคราะห์สถานะ:
- ให้ $P_n$ เป็นความน่าจะเป็นที่มดจะอยู่ที่จุดยอดเริ่มต้น ($V_0$) หลังจากเดิน $n$ ครั้ง
- ให้ $Q_n$ เป็นความน่าจะเป็นที่มดจะอยู่ที่จุดยอดอื่น (ที่ไม่ใช่ $V_0$) หลังจากเดิน $n$ ครั้ง
- เนื่องจากผลรวมของความน่าจะเป็นคือ $P_n + Q_n = 1$ และถ้ามดไม่อยู่ที่ $V_0$ มดต้องอยู่ที่จุดอื่น 3 จุดอย่างเท่าเทียมกัน
- การเปลี่ยนสถานะ:
  - มดจะกลับมาที่ $V_0$ ในครั้งถัดไปได้ ก็ต่อเมื่อตอนนี้อยู่ที่จุดอื่น ($Q_n$) และเลือกเดินเส้นทางที่กลับมา $V_0$ ซึ่งมีโอกาส $1/3$
  - มดจะอยู่ที่จุดอื่นในครั้งถัดไปได้ ถ้าตอนนี้อยู่ที่ $V_0$ (ต้องเดินไปจุดอื่นเสมอ ความน่าจะเป็น 1) หรืออยู่ที่จุดอื่น แล้วเลือกเดินไปที่จุดอื่นอีก 2 เส้นทาง (โอกาส $2/3$)
  - $P_{n+1} = \frac{1}{3} Q_n$
  - $Q_{n+1} = P_n + \frac{2}{3} Q_n$

เริ่มที่ $P_0 = 1, Q_0 = 0$:
- ครั้งที่ 1: $P_1 = \frac{1}{3}(0) = 0$, $Q_1 = 1 + rac{2}{3}(0) = 1$
- ครั้งที่ 2: $P_2 = \frac{1}{3}(1) = \frac{1}{3}$, $Q_2 = 0 + \frac{2}{3}(1) = \frac{2}{3}$
- ครั้งที่ 3: $P_3 = \frac{1}{3}(rac{2}{3}) = \frac{2}{9}$

ดังนั้น ความน่าจะเป็นที่มดจะกลับมาที่จุดยอดเดิมในการเดินครั้งที่ 3 คือ $\frac{2}{9}$
