# คำอธิบายและวิธีทำ

ในการเปลี่ยนเลขฐานสิบเป็นเลขฐานห้า เราใช้วิธีการหารสั้นด้วย 5 ไปเรื่อยๆ จนกระทั่งผลหารเป็นศูนย์ แล้วนำเศษที่ได้มาเรียงลำดับจากล่างขึ้นบน ดังนี้: \n1. นำ 80 หารด้วย 5: 80 ÷ 5 = 16 เศษ 0 \n2. นำผลหารที่ได้คือ 16 หารด้วย 5: 16 ÷ 5 = 3 เศษ 1 \n3. นำผลหารที่ได้คือ 3 หารด้วย 5: 3 ÷ 5 = 0 เศษ 3 \nเมื่ออ่านค่าเศษจากล่างขึ้นบน จะได้ 3, 1, และ 0 ดังนั้น 80 ในฐานสิบ เขียนในระบบตัวเลขฐานห้าได้เป็น 310_5
