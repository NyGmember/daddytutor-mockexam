# คำถาม

29. พิจารณาตารางตรวจสอบประเภทของพืชการจำแนกประเภทของข้าวเจ้า โดยอ้างอิงตารางการจำแนกต่อไปนี้ที่ถูกต้องคือข้อใด (ตัวอย่าง : $A_2 - C_4 - B_4$)

![illustration](../images/TEDET2558_Science_G9_slice30_crop_0.png)

① $A_1 - B_1 - C_1 - D_2$
② $A_1 - B_2$
③ $A_1 - B_1 - C_3$
④ $A_2 - B_3 - C_3$
⑤ $A_2 - B_4$

# คำอธิบายและวิธีทำ

การพิจารณาไดโคโทมัสคีย์ (Dichotomous Key) สำหรับจำแนกข้าวเจ้า ซึ่งเป็นพืชดอกใบเลี้ยงเดี่ยว มีลักษณะสำคัญดังนี้: 1. ข้าวเจ้าเป็นพืชที่มีราก ลำต้น ใบ และมีท่อลำเลียงน้ำและอาหาร (Tracheophyte) ซึ่งตรงกับเงื่อนไข A1 2. ข้าวเจ้าเป็นพืชดอกที่ขยายพันธุ์ด้วยเมล็ด ซึ่งตรงกับเงื่อนไข B1 3. เมล็ดของข้าวเป็นเมล็ดพืชดอกที่มีการปฏิสนธิซ้อน ทำให้ไข่ถูกห่อหุ้มด้วยรังไข่ (Angiosperm) ซึ่งตรงกับเงื่อนไข C1 4. ข้าวเป็นพืชใบเลี้ยงเดี่ยว ซึ่งในระยะงอกจะมีใบเลี้ยงเพียงหนึ่งใบ (Monocotyledon) ซึ่งตรงกับเงื่อนไข D2 ดังนั้น เส้นทางการจำแนกคือ A1 - B1 - C1 - D2
