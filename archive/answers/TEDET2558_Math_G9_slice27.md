# คำถาม

27. จากรูป จุด O เป็นจุดศูนย์กลางของวงกลม ที่มีเส้นผ่านศูนย์กลางยาว 20 เซนติเมตร ABC เป็นรูปสามเหลี่ยมมุมฉากหน้าจั่ว จุด C เป็นจุดศูนย์กลางที่ใช้สร้าง $\widehat{AB}$ ![illustration](../images/TEDET2558_Math_G9_slice27_crop_0.png) จงหาส่วนที่แรเงาพื้นที่กี่ตารางเซนติเมตร

# คำอธิบายและวิธีทำ

ขั้นตอนการหาพื้นที่ส่วนที่แรเงาเป็นดังนี้:1.  **พิจารณาวงกลมใหญ่:**จากโจทย์ จุด O เป็นจุดศูนย์กลางของวงกลม และเส้นผ่านศูนย์กลางยาว 20 เซนติเมตรดังนั้น รัศมีของวงกลม (R) = $20/2 = 10$ เซนติเมตรพื้นที่วงกลมทั้งหมด = $\pi R^2 = \pi (10^2) = 100\pi$ ตารางเซนติเมตร2.  **พิจารณาสามเหลี่ยม ABC และจุด C:**โจทย์ระบุว่า ABC เป็นรูปสามเหลี่ยมมุมฉากหน้าจั่ว และจุด C เป็นจุดศูนย์กลางที่ใช้สร้างส่วนโค้ง AB (ส่วนโค้งด้านล่าง)การที่สามเหลี่ยม ABC เป็นสามเหลี่ยมมุมฉากหน้าจั่ว หมายถึง $\angle ACB = 90^\circ$ และ $AC = BC$เนื่องจาก C เป็นจุดศูนย์กลางของส่วนโค้ง AB (ด้านล่าง) จึงหมายความว่า $AC$ และ $BC$ เป็นรัศมีของส่วนโค้งนั้น ให้ $AC = BC = r_c$จากทฤษฎีบทพีทาโกรัสในสามเหลี่ยมมุมฉาก ABC จะได้ $AB^2 = AC^2 + BC^2 = r_c^2 + r_c^2 = 2r_c^2$ ดังนั้น $AB = r_c\sqrt{2}$3.  **เชื่อมโยงกับวงกลมใหญ่ (จุด O):**จุด A และ B อยู่บนวงกลมที่มีจุด O เป็นศูนย์กลางและมีรัศมี 10 เซนติเมตร ดังนั้น $OA = OB = 10$ เซนติเมตรจากข้อมูลและรูปภาพ การจัดวางที่สอดคล้องกับเงื่อนไขทั้งหมดคือ เมื่อมุมที่จุดศูนย์กลาง O ที่รองรับคอร์ด AB ($\angle AOB$) มีค่า $90^\circ$หาก $\angle AOB = 90^\circ$ แล้ว สามเหลี่ยม OAB จะเป็นสามเหลี่ยมมุมฉากหน้าจั่ว (มี $OA=OB=10$)โดยทฤษฎีบทพีทาโกรัสในสามเหลี่ยม OAB จะได้ $AB^2 = OA^2 + OB^2 = 10^2 + 10^2 = 100 + 100 = 200$ดังนั้น ความยาวของคอร์ด $AB = \sqrt{200} = 10\sqrt{2}$ เซนติเมตรเมื่อนำความยาว AB นี้ไปเทียบกับสมการ $AB = r_c\sqrt{2}$ ที่ได้จากสามเหลี่ยม ABC จะได้ $10\sqrt{2} = r_c\sqrt{2}$ ซึ่งหมายความว่า $r_c = 10$ เซนติเมตรดังนั้น $AC = BC = 10$ เซนติเมตร ซึ่งยืนยันว่าการสมมติให้ $\angle AOB = 90^\circ$ นั้นสอดคล้องกับเงื่อนไขที่ให้มาทั้งหมด4.  **คำนวณพื้นที่ส่วนต่างๆ:***   **พื้นที่สามเหลี่ยม OAB:** เนื่องจาก $\triangle OAB$ เป็นสามเหลี่ยมมุมฉากหน้าจั่ว ($\angle AOB = 90^\circ$)พื้นที่สามเหลี่ยม OAB = $\frac{1}{2} \times OA \times OB = \frac{1}{2} \times 10 \times 10 = 50$ ตารางเซนติเมตร*   **พื้นที่ภาคส่วนของวงกลม OAB (ส่วนน้อย):**มุมที่จุดศูนย์กลาง $\angle AOB = 90^\circ$พื้นที่ภาคส่วน OAB = $\frac{90^\circ}{360^\circ} \times \pi R^2 = \frac{1}{4} \times 100\pi = 25\pi$ ตารางเซนติเมตร*   **พื้นที่ส่วนของวงกลม (segment) ด้านล่างคอร์ด AB (ส่วนที่ไม่แรเงาในวงกลม O):**พื้นที่ส่วนของวงกลม (segment) = พื้นที่ภาคส่วน OAB - พื้นที่สามเหลี่ยม OAB= $25\pi - 50$ ตารางเซนติเมตร5.  **คำนวณพื้นที่ส่วนที่แรเงา:**ส่วนที่แรเงาคือพื้นที่ส่วนของวงกลม (major segment) ที่อยู่เหนือคอร์ด ABพื้นที่ส่วนที่แรเงา = พื้นที่วงกลมทั้งหมด - พื้นที่ส่วนของวงกลม (segment) ด้านล่างคอร์ด AB= $100\pi - (25\pi - 50)$= $100\pi - 25\pi + 50$= $75\pi + 50$ ตารางเซนติเมตร
