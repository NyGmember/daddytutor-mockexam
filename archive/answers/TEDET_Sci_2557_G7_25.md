# คำถาม

25. จากภาพ บุญชูออกเดินทางจากจุด A ไปจุด B แล้วเดินย้อนกลับไปยังจุด C ญานีออกเดินทางหลังจากที่บุญชูเดินทางไปแล้ว 10 วินาที โดยเริ่มเดินจากจุด A ไปจุด C และหลังจากญานีออกเดินทางได้ 10 วินาที จึงพบกับบุญชูที่จุด C

ข้อใดคือความเร็วเฉลี่ยของญานีและบุญชู ตามลำดับ

① 2.5 m/s, 2.5 m/s  
② 2.5 m/s, 5.0 m/s

# คำอธิบายและวิธีทำ

กำหนดให้จุดเริ่มต้น $A$ อยู่ที่ตำแหน่ง $0\text{ เมตร}$ ทิศทางไปทางขวามือ (ทิศตะวันออก) เป็นบวก (+)
- ตำแหน่งของจุด $A = 0\text{ เมตร}$
- ตำแหน่งของจุด $C = +50\text{ เมตร}$
- ตำแหน่งของจุด $B = +100\text{ เมตร}$

1. **พิจารณาการเคลื่อนที่ของญานี:**
   - ญานีเริ่มต้นจากจุด $A$ ไปยังจุด $C$
   - การกระจัดของญานี ($\Delta x_{\text{ญานี}}$) คือ:
     $$\Delta x_{\text{ญานี}} = x_C - x_A = 50\text{ เมตร} - 0\text{ เมตร} = +50\text{ เมตร}$$
   - เวลาที่ญานีใช้ในการเดินทางจนพบกับบุญชูคือ $t_{\text{ญานี}} = 10\text{ วินาที}$
   - ดังนั้น ความเร็วเฉลี่ยของญานี ($\vec{v}_{\text{ญานี}}$) คือ:
     $$\vec{v}_{\text{ญานี}} = \frac{\Delta x_{\text{ญานี}}}{t_{\text{ญานี}}} = \frac{50\text{ เมตร}}{10\text{ วินาที}} = 5.0\text{ m/s}$$

2. **พิจารณาการเคลื่อนที่ของบุญชู:**
   - บุญชูเริ่มเดินจากจุด $A \rightarrow B \rightarrow C$
   - เนื่องจากความเร็วเฉลี่ยคิดจากการกระจัดทั้งหมดหารด้วยเวลารวม ตำแหน่งเริ่มต้นของบุญชูคือ $A$ และตำแหน่งสุดท้ายคือ $C$
   - การกระจัดของบุญชู ($\Delta x_{\text{บุญชู}}$) คือ:
     $$\Delta x_{\text{บุญชู}} = x_C - x_A = 50\text{ เมตร} - 0\text{ เมตร} = +50\text{ เมตร}$$
   - หาเวลาที่บุญชูใช้ในการเดินทางทั้งหมด ($t_{\text{บุญชู}}$):
     - บุญชูเริ่มเดินก่อนญานีเป็นเวลา $10\text{ วินาที}$
     - ญานีเดินอีก $10\text{ วินาที}$ แล้วพบกับบุญชู
     - ดังนั้น เวลารวมของบุญชูคือ:
       $$t_{\text{บุญชู}} = 10\text{ วินาที} + 10\text{ วินาที} = 20\text{ วินาที}$$
   - ดังนั้น ความเร็วเฉลี่ยของบุญชู ($\vec{v}_{\text{บุญชู}}$) คือ:
     $$\vec{v}_{\text{บุญชู}} = \frac{\Delta x_{\text{บุญชู}}}{t_{\text{บุญชู}}} = \frac{50\text{ เมตร}}{20\text{ วินาที}} = 2.5\text{ m/s}$$

ดังนั้น ความเร็วเฉลี่ยของญานีและบุญชู ตามลำดับ คือ **5.0 m/s และ 2.5 m/s**
