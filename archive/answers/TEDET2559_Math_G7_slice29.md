# คำอธิบายและวิธีทำ

เราต้องการหาผลตอบรับของผู้ชมรวมในหนึ่งปีให้มากที่สุด โดยมีเงื่อนไขดังนี้: กำหนดให้ P คือค่าตัวรายปี (หน่วยหมื่นบาท), N คือจำนวนครั้งที่โฆษณาต่อปี, และ R คือผลตอบรับต่อโฆษณาหนึ่งครั้ง จากโจทย์ N = 0.5 * (300 - P) และผลตอบรับรวม = R * N ซึ่งจะได้ว่า ผลตอบรับรวม = R * 0.5 * (300 - P) เราจะคำนวณผลตอบรับรวมสำหรับนางแบบแต่ละคน (หน่วยเป็นหมื่นบาท): 1) นางแบบ A: P=100, R=200 -> N = 0.5 * (300 - 100) = 100 ครั้ง -> ผลรวม = 200 * 100 = 20,000. 2) นางแบบ B: P=60, R=160 -> N = 0.5 * (300 - 60) = 120 ครั้ง -> ผลรวม = 160 * 120 = 19,200. 3) นางแบบ C: P=80, R=170 -> N = 0.5 * (300 - 80) = 110 ครั้ง -> ผลรวม = 170 * 110 = 18,700. 4) นางแบบ D: P=120, R=220 -> N = 0.5 * (300 - 120) = 90 ครั้ง -> ผลรวม = 220 * 90 = 19,800. 5) นางแบบ E: P=350 -> ซึ่งเกินเงื่อนไขงบประมาณรวม 3,000,000 บาท (หรือ 300 หมื่นบาท) เพราะค่าตัว 350 หมื่นบาทเกิน 300 หมื่นบาท ดังนั้นนางแบบ E จึงตัดออกได้เลย เมื่อเปรียบเทียบค่าผลรวมทั้งหมด 20,000, 19,200, 18,700 และ 19,800 พบว่านางแบบ A ให้ผลตอบรับรวมมากที่สุด
