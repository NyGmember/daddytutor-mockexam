# คำถาม

12. คะแนนสอบวิชาคณิตศาสตร์ ($x$) ของนักเรียนห้องหนึ่งเป็นดังตารางต่อไปนี้

![illustration](../images/TEDET_Math_2557_G8_12_crop_0.png)

โดยที่จำนวนคนสะสมเป็นผลบวกของจำนวนคนในช่วงคะแนนนั้น ๆ กับช่วงคะแนนก่อนหน้าทั้งหมด (เช่น ในช่วงคะแนน $50 \le x < 60$ จะมีจำนวนคนสะสม $2+6=8$ คน เป็นต้น)
จงหาค่าของ $a + b + c$

# คำอธิบายและวิธีทำ

ขั้นตอนการคำนวณแบ่งออกเป็น 5 ขั้นตอนหลักดังนี้:

**ขั้นที่ 1: หาจำนวนนักเรียนทั้งหมดในห้องเรียน ($N$)**
จากข้อมูลในตารางแถวแรกสำหรับช่วงคะแนน $40 \le x < 50$:
- จำนวนคนคือ $2$ คน
- คิดเป็นร้อยละ $4$

สูตรการหาความสัมพันธ์ของร้อยละ:
$$\text{ร้อยละ} = \left( \frac{\text{จำนวนคนในช่วงคะแนน}}{\text{จำนวนนักเรียนทั้งหมด } (N)} \right) \times 100$$
แทนค่าจะได้:
$$4 = \frac{2}{N} \times 100$$
$$4N = 200$$
$$N = 50\text{ คน}$$

ดังนั้น จำนวนนักเรียนทั้งหมดในห้องเรียนนี้คือ $50$ คน

---

**ขั้นที่ 2: หาจำนวนนักเรียนในช่วงคะแนน $80 \le x < 90$**
จากตาราง ช่วงคะแนน $80 \le x < 90$ มีร้อยละเป็น $16$
ดังนั้น จำนวนนักเรียนในช่วงคะแนนนี้คือ:
$$\text{จำนวนคน} = 50 \times \frac{16}{100} = 8\text{ คน}$$

---

**ขั้นที่ 3: หาค่าของ $a$ และ $b$**
เนื่องจากจำนวนนักเรียนทั้งหมดคือ $50$ คน ผลรวมของจำนวนนักเรียนในทุกช่วงคะแนนรวมกันต้องเท่ากับ $50$ คน:
$$\text{จำนวนนักเรียนในแต่ละช่วง} = 2 + 6 + a + 10 + 8 + 1 = 50$$
$$27 + a = 50$$
$$a = 23$$

ต่อมา หาค่า $b$ ซึ่งเป็นร้อยละของจำนวนนักเรียนในช่วงคะแนน $60 \le x < 70$:
$$b = \frac{a}{N} \times 100$$
$$b = \frac{23}{50} \times 100 = 46$$

ดังนั้น จะได้ $a = 23$ และ $b = 46$

---

**ขั้นที่ 4: หาค่าของ $c$**
ค่า $c$ คือจำนวนคนสะสมในช่วงคะแนน $70 \le x < 80$ ซึ่งมีค่าเท่ากับผลรวมของจำนวนคนตั้งแต่ช่วงแรกจนถึงช่วงคะแนน $70 \le x < 80$:
$$c = 2 + 6 + a + 10$$
แทนค่า $a = 23$ จะได้:
$$c = 2 + 6 + 23 + 10 = 41$$

*(หรือคิดจากจำนวนนักเรียนทั้งหมดหักออกด้วยจำนวนนักเรียนในช่วงคะแนนหลังจากนั้น: $c = 50 - 8 - 1 = 41$)*

ดังนั้น จะได้ $c = 41$

---

**ขั้นที่ 5: หาค่าของ $a + b + c$**
แทนค่าที่ได้จากการคำนวณทั้งหมด:
$$a + b + c = 23 + 46 + 41 = 110$$

ดังนั้น ค่าของ $a + b + c$ เท่ากับ $110$
