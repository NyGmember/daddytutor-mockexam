# คำถาม

19. ต้องการเขียนจำนวนนับที่มีค่าไม่เกิน 100 ตามหลักเกณฑ์ต่อไปนี้
![illustration](../images/TEDET2559_Math_G7_slice18_crop_0.png)
ทำให้ได้ตารางดังต่อไปนี้
![illustration](../images/TEDET2559_Math_G7_slice18_crop_1.png)
จงหาว่ามีจำนวนนับทั้งหมดกี่จำนวนที่ปรากฏในตารางเพียง 3 ครั้งเท่านั้น

# คำอธิบายและวิธีทำ

พิจารณาตารางที่สร้างขึ้น แถวที่ $i$ และหลักที่ $j$ จะมีค่าเท่ากับ $i \times j$ โดยที่ $i, j$ เป็นจำนวนนับและ $i \times j \le 100$ โจทย์ต้องการทราบจำนวนนับ $n$ ใดที่ปรากฏในตารางเพียง 3 ครั้ง หมายความว่า จำนวน $n$ ดังกล่าวจะต้องเขียนอยู่ในรูปผลคูณ $i \times j = n$ ได้เพียง 3 คู่ลำดับ $(i, j)$ เท่านั้น โดยที่ $1 \le i, j \le 100$ และ $i, j \le 100$ (จากตาราง แถวมีถึง 100 และคอลัมน์ถึง 100) แต่เงื่อนไขคือ $i \times j \le 100$ ดังนั้นตัวประกอบของ $n$ ต้องมีจำนวนคู่บวกที่ทำให้ได้ $n$ เพียง 3 คู่เท่านั้น จำนวนที่มีจำนวนตัวประกอบเป็น 3 คือจำนวนที่เป็น 'กำลังสองสมบูรณ์ของจำนวนเฉพาะ' เท่านั้น (เช่น $p^2$ จะมีตัวประกอบคือ $1, p, p^2$ รวม 3 ตัว) สมมติ $n = p^2$ จะมีคู่ลำดับ $(i, j)$ ที่ได้ $n$ คือ $(1, p^2), (p, p), (p^2, 1)$ ซึ่งมีเพียง 3 คู่พอดี ดังนั้นเราต้องหาว่า $p^2 	ext{ เป็นจำนวนเฉพาะใดบ้างที่ } p^2 	ext{ ไม่เกิน } 100$ ได้แก่: 1. $2^2 = 4$, 2. $3^2 = 9$, 3. $5^2 = 25$, 4. $7^2 = 49$ สำหรับจำนวนเฉพาะถัดไปคือ $11^2 = 121$ ซึ่งเกิน 100 จึงใช้ไม่ได้ นอกจากนี้ ต้องพิจารณาว่าไม่มีค่าอื่นที่ซ้ำในตารางได้ 3 ครั้ง ซึ่งจำนวน $n$ ที่มีตัวประกอบ 3 ตัวพอดีคือเลขกำลังสองของจำนวนเฉพาะตามทฤษฎีจำนวน ดังนั้นจำนวนที่ปรากฏ 3 ครั้งในตารางคือ 4, 9, 25 และ 49 รวมทั้งหมด 4 จำนวน
