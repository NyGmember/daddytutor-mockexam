# คำถาม

6. กระเป๋าเป้ใบหนึ่งมีสมุด 2 เล่ม กับหนังสือ 3 เล่ม กระเป๋าเป้อีกใบหนึ่งมีสมุด 1 เล่ม กับหนังสือ 2 เล่ม ถ้าถุงกระเป๋าเป้สองใบนี้มีราคา โดยกระเป๋าเป้ใบแรกมีค่า 345 บาท และกระเป๋าเป้อีกใบมีค่า 190 บาท จงหาว่าสมุด 1 ใบ และหนังสือ 1 ใบ มีราคาเท่ากันเท่าใด

# คำอธิบายและวิธีทำ

กำหนดให้ x แทนราคาของสมุดหนึ่งเล่ม และ y แทนราคาของหนังสือหนึ่งเล่ม จากโจทย์เราสามารถสร้างระบบสมการเชิงเส้นสองตัวแปรได้ดังนี้: 1) กระเป๋าเป้ใบแรก: 2x + 3y = 345, 2) กระเป๋าเป้อีกใบ: x + 2y = 190. จากสมการ (2) เราสามารถจัดรูปได้เป็น x = 190 - 2y. นำค่า x นี้ไปแทนในสมการ (1): 2(190 - 2y) + 3y = 345 จะได้ 380 - 4y + 3y = 345 ดังนั้น 380 - y = 345 ซึ่งจะได้ y = 380 - 345 = 35 บาท. เมื่อทราบราคาหนังสือ (y) แล้ว ให้นำกลับไปแทนค่าเพื่อหา x: x = 190 - 2(35) = 190 - 70 = 120 บาท. ดังนั้น สมุดหนึ่งเล่มราคา 120 บาท และหนังสือหนึ่งเล่มราคา 35 บาท ผลรวมของราคาของสมุดหนึ่งเล่มและหนังสือหนึ่งเล่มคือ 120 + 35 = 155 บาท
