# คำถาม

15. ถ้า $x$ และ $y$ เป็นจำนวนนับที่มีหนึ่งหลักซึ่ง

$$2(10x + y) = 5(3x + y)$$

จงหาค่าของ $10x + y$

# คำอธิบายและวิธีทำ

จากโจทย์ที่กำหนดให้: 2(10x + y) = 5(3x + y) โดยที่ x และ y เป็นจำนวนนับที่มีหนึ่งหลัก (หมายถึง x และ y เป็นเลขโดดตั้งแต่ 1 ถึง 9) ขั้นตอนการแก้สมการมีดังนี้: 1) กระจายตัวเลขเข้าไปในวงเล็บ: 20x + 2y = 15x + 5y, 2) จัดกลุ่มตัวแปร x และ y ให้อยู่คนละข้างของสมการ: 20x - 15x = 5y - 2y, 3) จะได้ 5x = 3y, 4) เนื่องจาก x และ y เป็นจำนวนนับที่มีหนึ่งหลัก จากสมการ 5x = 3y เราจะได้อัตราส่วน x/y = 3/5 ซึ่งหมายความว่า x ต้องเป็นพหุคูณของ 3 และ y ต้องเป็นพหุคูณของ 5 เนื่องจากเป็นเลขหนึ่งหลัก ค่าที่เป็นไปได้เพียงค่าเดียวคือ x = 3 และ y = 5, 5) แทนค่า x = 3 และ y = 5 ลงในนิพจน์ 10x + y จะได้ 10(3) + 5 = 30 + 5 = 35
