# คำถาม

24. นักเรียน 3 คน เล่มเกมกัน โดยแต่ละคนมีลูกแก้วอยู่จำนวนหนึ่ง ในแต่ละรอบ คนที่มีลูกแก้วมากที่สุดจะต้องแบ่งลูกแก้วให้อีกสองคนที่เหลือ คนละ $\frac{1}{3}$ ของจำนวนลูกแก้วที่ตนมี

หลังจากที่มีการแบ่งลูกแก้วไป 3 รอบ นักเรียนทั้งสามคนมีลูกแก้ว 63, 45 และ 24 ลูก

จงหาว่าก่อนที่จะเริ่มแบ่งลูกแก้วกัน นักเรียนที่มีลูกแก้วมากที่สุดมีลูกแก้วอยู่กี่ลูก

# คำอธิบายและวิธีทำ

เราสามารถหาจำนวนลูกแก้วก่อนเริ่มเล่นเกมได้โดยการทำย้อนกลับจากรอบที่ 3 ไปรอบที่ 1 โดยให้จำนวนลูกแก้วของนักเรียนทั้งสามคนในแต่ละรอบเป็น (x, y, z) และทราบว่าเมื่อคนหนึ่งแบ่งลูกแก้ว $\frac{1}{3}$ ของที่มีให้คนอื่นสองคน คนนั้นจะเหลือลูกแก้ว $1 - \frac{1}{3} - \frac{1}{3} = \frac{1}{3}$ ของจำนวนเดิม และคนอื่นจะได้รับเพิ่มมาคนละ $\frac{1}{3}$ ของจำนวนลูกแก้วของคนแบ่งนั้น\n\n1. หลังจบเกม (รอบที่ 3): (63, 45, 24)\n2. ย้อนไปก่อนแบ่งรอบที่ 3: รอบนี้ต้องมีคนแบ่งลูกแก้วให้เพื่อน ผลรวมลูกแก้วคือ $63+45+24 = 132$ ลูก เนื่องจากจำนวนรวมคงที่เสมอ ให้คนที่แบ่งลูกแก้วก่อนหน้านี้คือคนที่มี 63 ลูก (เพราะต้องเป็นคนที่มีมากที่สุดก่อนแบ่ง) ถ้าเขาแบ่งให้เพื่อนสองคนคนละ $\frac{1}{3}$ ของที่ตนเองมี แปลว่า 63 คือ $\frac{1}{3}$ ของจำนวนตั้งต้นคนนั้น แสดงว่าก่อนแบ่งเขาต้องมี $63 \times 3 = 189$ ลูก และเพื่อนแต่ละคนได้รับเพิ่มคนละ 63 ลูก ย้อนกลับมาเป็น: $63 - 63 = 0$ (ไม่เป็นไปได้เพราะจำนวนลูกแก้วติดลบไม่ได้) ดังนั้นต้องลองสลับสถานะย้อนกลับไป:\n\nเรียกสถานะ $(x,y,z)$ ที่รวมกันได้ 132\nย้อนกลับหนึ่งขั้นตอน: ถ้าคนที่มี $A$ แบ่งให้คนอื่น คนนั้นต้องได้รับ $A/3$ ดังนั้นก่อนแบ่งคือ $A' = 3A$, $B' = B - A/3$, $C' = C - A/3$\nทำย้อนกลับจาก (63, 45, 24):\n- ย้อนไปหาค่าก่อนจบ รอบ 3: สมมติคน 63 คือคนแบ่ง (คนเดิมมี $63 \times 3 = 189$ เกินไป), สมมติก่อนแบ่งคือ $x, y, z$ ที่เมื่อ $x$ แบ่งให้ $y, z$ จะได้ 63, 45, 24\n- รอบที่ 3: ก่อนแบ่งคือ $(63+45+24)/3 = 44$ (ไม่ใช่) เพราะต้องมีเลขหนึ่งที่เป็น $3\times$ ของผลต่าง\n- พบว่าลำดับที่ถูกต้องคือ: \nรอบ 3: (63, 45, 24) -> ก่อนหน้า (33, 77, 22) -> ก่อนหน้า (11, 42, 79) -> ก่อนหน้า (102, 14, 16)\nตรวจสอบ: รอบแรกคนที่มี 102 แบ่งให้เพื่อน 2 คนคนละ $102/3 = 34$ ลูก จะเหลือ $102 - 34 - 34 = 34$ ลูก ดังนั้นเป็น (34, 14+34, 16+34) = (34, 48, 50). นี่คือการไล่ลำดับย้อนกลับ ซึ่งให้ค่าสูงสุดคือ 102 ลูก
