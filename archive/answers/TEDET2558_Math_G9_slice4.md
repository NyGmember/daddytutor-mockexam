# คำถาม

4. จากรูป $\square$ABCD เป็นรูปสี่เหลี่ยมด้านขนาน
![illustration](../images/TEDET2558_Math_G9_slice4_crop_0.png)
ถ้า $\angle$ACD = $90^{\circ}$ และ $\angle$BDC = $38^{\circ}$
จงหาว่า $\angle x + \angle y + \angle z$ เท่ากับกี่องศา

# คำอธิบายและวิธีทำ

จากสมบัติของรูปสี่เหลี่ยมด้านขนาน ABCD จะได้ว่าด้านตรงข้ามขนานกัน (AD // BC และ AB // DC) และทแยงมุมตัดกันที่จุดหนึ่ง ซึ่งเราจะเรียกว่าจุด O\n\n1. พิจารณามุมแย้ง: เนื่องจาก AD // BC ดังนั้นมุมแย้ง $\angle ADB = \angle DBC$ ซึ่งหมายความว่า $\angle DBC = 38^{\circ}$\n2. จาก $\angle ACD = 90^{\circ}$ และ $\angle BCD$ เป็นมุมฉากของสามเหลี่ยมมุมฉาก\n3. ในรูปสามเหลี่ยม BCD: ผลรวมมุมภายในคือ $180^{\circ}$ เราทราบว่า $\angle BCD = 90^{\circ}$ (จากโจทย์ระบุ $\angle ACD = 90^{\circ}$ และเป็นรูปสี่เหลี่ยมด้านขนาน ดังนั้นมุมที่ฐานย่อมส่งผลให้ $\angle BCD$ ต้องเป็น $90^{\circ}$ ด้วย) ดังนั้น $\angle DBC + \angle BDC + \angle BCD = 180^{\circ}$ จะได้ $\angle DBC + 38^{\circ} + 90^{\circ} = 180^{\circ}$ ดังนั้น $\angle DBC = 180^{\circ} - 128^{\circ} = 52^{\circ}$\n4. จากโจทย์มุม $\angle y$ คือมุมที่อยู่ภายในสามเหลี่ยมที่เกิดจากการตัดกันของเส้นทแยงมุม ซึ่งในที่นี้ $\angle y = \angle DBC = 52^{\circ}$\n5. พิจารณามุม $\angle z$: มุม $\angle z$ คือมุมภายนอกของสามเหลี่ยมที่เกิดจากจุดตัดทแยงมุม โดย $\angle z = \angle DAC + \angle ADC$ หรือพิจารณาจากมุมตรงข้าม จะพบว่าในรูปสี่เหลี่ยมด้านขนาน มุมที่เส้นทแยงมุมตัดกันมีสมบัติเฉพาะ\n6. จากมุม $\angle x$: เนื่องจาก AD // BC จะได้ $\angle x = \angle DBC = 52^{\circ}$\n7. ในรูปสามเหลี่ยมเล็กที่เกิดจากจุดตัดทแยงมุม (ให้จุดตัดคือ O): $\angle z$ เป็นมุมภายนอกของสามเหลี่ยมที่ประกอบด้วยมุม $\angle BCD = 90^{\circ}$ มุม $\angle z = 180^{\circ} - (\angle OBC + \angle OCB) = 180^{\circ} - (52^{\circ} + 38^{\circ}) = 90^{\circ}$\nดังนั้น $\angle x + \angle y + \angle z = 52^{\circ} + 52^{\circ} + 90^{\circ} = 194^{\circ}$
