# คำถาม

8. ต้องการหาผลคูณของ $a$ กับ 1.5 แต่ทำผิดพลาดไปโดยได้ใช้ $1.\dot{5}$ แทน 1.5 หากผลลัพธ์ที่ผิดพลาดกับผลลัพธ์ที่ถูกต้องมีผลต่างเท่ากับ $0.\dot{3}$ จงหาค่าของ $a$

# คำอธิบายและวิธีทำ

โจทย์ต้องการหาผลคูณของ $a$ กับ 1.5 แต่ใช้ค่า $1.555...$ หรือ $1.\dot{5}$ แทน ทำให้ผลลัพธ์คลาดเคลื่อนไป $0.333...$ หรือ $0.\dot{3}$

1. **แปลงทศนิยมซ้ำให้อยู่ในรูปเศษส่วน:**
   * สำหรับ $1.5$:
     $$1.5 = \frac{15}{10} = \frac{3}{2}$$
   * สำหรับ $1.\dot{5}$ (ทศนิยมซ้ำ 1 ตำแหน่ง):
     $$1.\dot{5} = 1 + 0.555... = 1 + \frac{5}{9} = \frac{14}{9}$$
   * สำหรับ $0.\dot{3}$:
     $$0.\dot{3} = 0.333... = \frac{3}{9} = \frac{1}{3}$$

2. **ตั้งสมการหาผลต่างของผลลัพธ์:**
   * ผลลัพธ์ที่ถูกต้องคือ: $1.5 \times a = \frac{3}{2}a$
   * ผลลัพธ์ที่ผิดพลาดคือ: $1.\dot{5} \times a = \frac{14}{9}a$
   * ผลต่างของทั้งสองผลลัพธ์คือ $0.\dot{3}$:
     $$\frac{14}{9}a - \frac{3}{2}a = 0.\dot{3}$$
     
     หาครน. ของตัวส่วนคือ 18:
     $$\left(\frac{28}{18} - \frac{27}{18}\right)a = \frac{1}{3}$$
     $$\frac{1}{18}a = \frac{1}{3}$$

3. **แก้สมการหาค่า $a$:**
     $$a = \frac{1}{3} \times 18$$
     $$a = 6$$

ดังนั้น ค่าของ $a$ คือ **6**
