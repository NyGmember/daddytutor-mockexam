# คำถาม

26. ถ้าแยกตัวประกอบของพหุนาม

$$x^2 + xy - 6y^2 + 3x + 19y - 10$$

ได้เป็น $(x + ay + b)(x + cy + d)$
จงหาค่าของ $a + b + c + d$

# คำอธิบายและวิธีทำ

เราต้องการแยกตัวประกอบของพหุนาม $x^2 + xy - 6y^2 + 3x + 19y - 10$ ให้อยู่ในรูป $(x + ay + b)(x + cy + d)$ โดยการจัดกลุ่มพหุนามพหุนามสองตัวแปรนี้

1. พิจารณาส่วนที่เป็นพหุนามกำลังสองของ $x$ และ $y$ คือ $x^2 + xy - 6y^2$ ซึ่งสามารถแยกตัวประกอบได้เป็น $(x + 3y)(x - 2y)

2. ดังนั้น พหุนามเดิมของเราจะอยู่ในรูป $(x + 3y + b)(x - 2y + d) = x^2 + (d - 2y + 3y)x + ...$ เมื่อกระจายพหุนามนี้จะได้:
$(x + 3y + b)(x - 2y + d) = x^2 + x(-2y + d) + 3y(x) + 3y(-2y + d) + bx + b(-2y + d)
= x^2 + xy + x(d + b) + y(3d - 2b) - 6y^2 + bd$

3. เทียบสัมประสิทธิ์กับพหุนามโจทย์ $x^2 + xy - 6y^2 + 3x + 19y - 10$:
   - สัมประสิทธิ์ของ $x$: $d + b = 3$  --- (1)
   - สัมประสิทธิ์ของ $y$: $3d - 2b = 19$ --- (2)
   - ค่าคงที่: $bd = -10$ --- (3)

4. แก้ระบบสมการจาก (1) และ (2):
   จาก (1): $b = 3 - d$
   แทนใน (2): $3d - 2(3 - d) = 19$
   $3d - 6 + 2d = 19$
   $5d = 25$ ดังนั้น $d = 5$
   หาค่า $b$ จาก $b = 3 - 5 = -2$
   ตรวจสอบกับสมการ (3): $bd = (-2)(5) = -10$ (ถูกต้อง)

5. ดังนั้นตัวประกอบคือ $(x + 3y - 2)(x - 2y + 5)$ ซึ่งตรงกับรูป $(x + ay + b)(x + cy + d)$
   จะได้ $a = 3, b = -2, c = -2, d = 5$

6. ค่าของ $a + b + c + d = 3 + (-2) + (-2) + 5 = 4$
