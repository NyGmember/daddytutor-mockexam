# คำถาม

10. โรงงานผลิตน้ำประปาจะใส่คลอรีนในอัตราส่วนแน่นอนค่าหนึ่งลงในน้ำประปาเพื่อฆ่าเชื้อโรค ดังตัวอย่างต่อไปนี้ ![illustration](../images/TEDET2560_Math_G8_slice10_crop_0.png) น้ำประปาจะมีปริมาณกี่ลิตร เมื่อใส่คลอรีน 38 มิลลิกรัม

# คำอธิบายและวิธีทำ

จากโจทย์ โรงงานผลิตน้ำประปาใส่คลอรีนในอัตราส่วนที่แน่นอน เราสามารถหาอัตราส่วนของปริมาณคลอรีนต่อปริมาณน้ำประปาได้ดังนี้ จากตารางกำหนดให้ ปริมาณน้ำประปา 7 ลิตร ใช้คลอรีน 1.4 มิลลิกรัม ดังนั้น อัตราส่วนของคลอรีนต่อน้ำประปาคือ $\frac{1.4}{7} = 0.2$ มิลลิกรัมต่อลิตร เราสามารถตรวจสอบได้ว่าอัตราส่วนนี้คงที่ทุกกรณีหรือไม่: $\frac{2.4}{12} = 0.2$, $\frac{4.8}{24} = 0.2$ และ $\frac{15.6}{78} = 0.2$ ซึ่งเป็นค่าคงที่จริง ให้ $V$ แทนปริมาณน้ำประปา (ลิตร) เมื่อใส่คลอรีน $C$ มิลลิกรัม จะได้ความสัมพันธ์ $C = 0.2 \times V$ จากโจทย์ต้องการทราบปริมาณน้ำประปาเมื่อใส่คลอรีน 38 มิลลิกรัม แทนค่าในสมการจะได้ $38 = 0.2 \times V$ แก้สมการหาค่า $V$ โดยนำ 0.2 ไปหารทั้งสองข้าง: $V = \frac{38}{0.2} = \frac{380}{2} = 190$ ลิตร
