# คำถาม

22. มีแอปเปิลอยู่ 1 ลัง ถ้า A หยิบไป $\frac{1}{3}$ ของทั้งหมด จากนั้น B หยิบไปอีก $\frac{1}{3}$ ของที่เหลือ ท้ายสุด C หยิบไป $\frac{1}{3}$ ของที่เหลือ ทำให้เหลือแอปเปิลในลัง 8 ผล จงหาว่าเดิมมีแอปเปิลอยู่ในลังกี่ผล

# คำอธิบายและวิธีทำ

เพื่อให้หาจำนวนแอปเปิลเริ่มต้นได้ เราสามารถใช้วิธีการแก้สมการเชิงเส้นตัวแปรเดียว โดยกำหนดให้จำนวนแอปเปิลเดิมมีค่าเป็น $x$ ผล

1. เมื่อ A หยิบไป $\frac{1}{3}$ ของทั้งหมด จะเหลือแอปเปิล: $x - \frac{1}{3}x = \frac{2}{3}x$
2. เมื่อ B หยิบไปอีก $\frac{1}{3}$ ของที่เหลือ จะเหลือแอปเปิล: $\frac{2}{3}x - \frac{1}{3}(\frac{2}{3}x) = \frac{2}{3}x(1 - \frac{1}{3}) = \frac{2}{3}x \times \frac{2}{3} = \frac{4}{9}x$
3. เมื่อ C หยิบไปอีก $\frac{1}{3}$ ของที่เหลือล่าสุด จะเหลือแอปเปิล: $\frac{4}{9}x - \frac{1}{3}(\frac{4}{9}x) = \frac{4}{9}x(1 - \frac{1}{3}) = \frac{4}{9}x \times \frac{2}{3} = \frac{8}{27}x$

โจทย์กำหนดให้เหลือแอปเปิลในลัง 8 ผล ดังนั้นเราจะได้สมการ:
$\frac{8}{27}x = 8$

แก้สมการโดยการคูณด้วย $\frac{27}{8}$ ทั้งสองข้าง:
$x = 8 \times \frac{27}{8}$
$x = 27$

ดังนั้น เดิมมีแอปเปิลอยู่ในลังทั้งหมด 27 ผล
