# คำถาม

29. มีลานจอดรถ 7 แห่ง คือ A, B, C, D, E, F และ G มีรถที่ยังไม่มีที่จอดในลานจอด A, B และ C อยู่ 9, 6 และ 8 คัน ตามลำดับ จึงต้องย้ายรถไปยังลานจอด D, E, F และ G ซึ่งยังมีที่จอดรถได้อีก 5, 7, 3 และ 8 คัน ตามลำดับ โดยมีค่าใช้จ่ายในการย้ายรถแต่ละคันระหว่างลานจอดดังแสดงในตารางต่อไปนี้

# คำอธิบายและวิธีทำ

ปัญหาข้อนี้คือปัญหาการขนส่ง (Transportation Problem) ซึ่งเราต้องการเลือกเส้นทางการย้ายรถให้มีค่าใช้จ่ายรวมต่ำที่สุด โดยมีเงื่อนไขดังนี้: 
แหล่งที่มา (Supply): A=9, B=6, C=8 (รวม 23 คัน)
จุดหมายปลายทาง (Demand): D=5, E=7, F=3, G=8 (รวม 23 คัน)
เราต้องจัดสรรจำนวนรถ $x_{ij}$ เพื่อให้ค่าใช้จ่าย $\sum C_{ij} x_{ij}$ น้อยที่สุด:

1. ให้ความสำคัญกับค่าใช้จ่ายต่ำที่สุดก่อน:
- ย้ายจาก A ไป E (ค่าใช้จ่าย 2): ให้เต็มจำนวนที่ E รับได้ 7 คัน (คงเหลือ E=0, A=2)
- ย้ายจาก C ไป D (ค่าใช้จ่าย 3): ให้เต็มจำนวนที่ D รับได้ 5 คัน (คงเหลือ D=0, C=3)
- ย้ายจาก B ไป F (ค่าใช้จ่าย 3): ให้เต็มจำนวนที่ F รับได้ 3 คัน (คงเหลือ F=0, B=3)
- ย้ายจาก B ไป D (ค่าใช้จ่าย 4 - แต่ D เต็มแล้ว): ย้ายจาก B ไป A หรือจุดอื่น แต่เนื่องจาก D เต็ม, F เต็ม, E เต็ม, เราจึงเหลือลาน G ที่ว่างอยู่

ปรับปรุงการจัดสรรตามต้นทุนต่ำสุดใหม่:
- A -> E: 7 คัน (ค่าใช้จ่าย 7 * 2 = 14)
- A -> G: 2 คัน (ค่าใช้จ่าย 2 * 4 = 8)
- B -> F: 3 คัน (ค่าใช้จ่าย 3 * 3 = 9)
- B -> G: 3 คัน (ค่าใช้จ่าย 3 * 8 = 24)
- C -> D: 5 คัน (ค่าใช้จ่าย 5 * 3 = 15)
- C -> G: 3 คัน (ค่าใช้จ่าย 3 * 5 = 15)
รวมค่าใช้จ่าย = 14 + 8 + 9 + 24 + 15 + 15 = 85.

ลองตรวจสอบอีกทางเลือก (ให้ B ไป D ถูกลง):
- C -> D: 5 คัน (15)
- B -> D: 0 คัน (เลือกให้ B ไปที่อื่น)
- B -> F: 3 คัน (9)
- B -> G: 3 คัน (24)
- A -> E: 7 คัน (14)
- A -> G: 2 คัน (8)
- C -> G: 3 คัน (15)
รวมเป็น 85 ดอลลาร์ ซึ่งเป็นการหาค่าที่น้อยที่สุดที่เป็นไปได้ภายใต้เงื่อนไข Supply และ Demand ที่กำหนด
