# คำถาม

23. ต้องการตัดแท่งไม้ A ที่ยาว 1 เมตร และแท่งไม้ B ที่ยาว 2 เมตร ออกเป็นท่อน ๆ ให้แต่ละแท่งมีจำนวนท่อนที่เท่ากัน และให้แต่ละท่อน (ในแต่ละแท่ง) มีความยาวเท่า ๆ กัน แล้วนำท่อนไม้ที่ตัดจากแท่งไม้ A และ B แห่งละ 1 ท่อน มาวางต่อกัน ดังตัวอย่างต่อไปนี้

![illustration](../images/TEDET2560_Math_G8_slice23_crop_0.png)

ถ้าต้องการให้ท่อนไม้ที่ต่อกันมีความยาวมากกว่า 10 เซนติเมตร แต่สั้นกว่า 150 เซนติเมตร
จงหาว่าท่อนไม้ที่ต่อกันจะมีความยาวที่แตกต่างกันได้ทั้งหมดกี่ค่า

# คำอธิบายและวิธีทำ

กำหนดให้แท่งไม้ A ยาว 100 ซม. (1 เมตร) และแท่งไม้ B ยาว 200 ซม. (2 เมตร) ตัดออกเป็นจำนวน n ท่อนที่เท่ากัน โดยให้แต่ละท่อนของ A มีความยาว $a = \frac{100}{n}$ ซม. และแต่ละท่อนของ B มีความยาว $b = \frac{200}{n}$ ซม. เมื่อ $n$ ต้องเป็นจำนวนเต็มบวก
เมื่อนำท่อนไม้จาก A และ B อย่างละ 1 ท่อนมาวางต่อกัน ความยาวรวม $L$ คือ:
$L = a + b = \frac{100}{n} + \frac{200}{n} = \frac{300}{n}$
โจทย์กำหนดเงื่อนไขว่า $10 < L < 150$ ซม.
แทนค่า $L$ ด้วย $\frac{300}{n}$:
$10 < \frac{300}{n} < 150$
หารด้วย 300 จะได้:
$\frac{10}{300} < \frac{1}{n} < \frac{150}{300}$
$\frac{1}{30} < \frac{1}{n} < \frac{1}{2}$
กลับเศษเป็นส่วน (อสมการจะเปลี่ยนทิศทาง):
$2 < n < 30$
ดังนั้น $n$ สามารถเป็นจำนวนเต็มใดก็ได้ตั้งแต่ $3, 4, 5, \dots, 29$ ซึ่งจะมีทั้งหมด $29 - 3 + 1 = 27$ ค่า
เนื่องจากค่า $n$ แต่ละค่าให้ค่า $L = \frac{300}{n}$ ที่แตกต่างกันเสมอ ดังนั้นจึงมีทั้งหมด 27 ค่าที่แตกต่างกัน
