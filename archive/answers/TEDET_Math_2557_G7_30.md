# คำถาม

30. กำหนดให้ $a, b, c, d, e, f$ แทนเลขโดดที่มีค่าไม่ซ้ำกันตั้งแต่ 1 ถึง 6 และสอดคล้องกับเงื่อนไขทั้งสองข้อต่อไปนี้
(ก) $a+b < c+d$
(ข) $c+e < a < f$

จงหาค่าของ $a + 2b + 3c + 4d$

# คำอธิบายและวิธีทำ

กำหนดให้ $a, b, c, d, e, f$ เป็นเลขโดดที่แตกต่างกันจากเซต $\{1, 2, 3, 4, 5, 6\}$
และสอดคล้องกับสองเงื่อนไขต่อไปนี้:
1. **เงื่อนไข (ก):** $a+b < c+d$
2. **เงื่อนไข (ข):** $c+e < a < f$

เราสามารถวิเคราะห์หาค่าของตัวแปรแต่ละตัวได้ดังนี้:

1. **วิเคราะห์เงื่อนไข (ข) เพื่อจำกัดขอบเขตของ $a$:**
   * เนื่องจาก $c$ และ $e$ เป็นจำนวนเต็มบวกที่แตกต่างกัน ค่าที่น้อยที่สุดที่เป็นไปได้ของ $c+e$ คือ $1 + 2 = 3$
   * จาก $c+e < a$ จะได้ว่า $a > 3$ ดังนั้น $a \in \{4, 5, 6\}$
   * จาก $a < f$ และเนื่องจากค่าสูงสุดของเลขโดดคือ 6 ดังนั้น $a$ จะต้องมีค่าน้อยกว่า $f$ ซึ่งเป็นไปได้ดังนี้:
     * หาก $a = 6$ จะไม่มีค่า $f$ ในเซตที่มากกว่า 6 (ใช้ไม่ได้)
     * ดังนั้น $a \in \{4, 5\}$ เท่านั้น

2. **ทดสอบแต่ละกรณีของ $a$:**

   * **กรณีที่ 1: $a = 5$**
     * จาก $a < f$ จะได้ว่า $f = 6$ (เนื่องจาก 6 เป็นค่าเดียวที่มากกว่า 5)
     * จาก $c+e < a \implies c+e < 5$ แสดงว่า $c+e$ เป็นไปได้สองค่าคือ 3 หรือ 4
       * **ถ้า $c+e = 3$** จะได้ $\{c, e\} = \{1, 2\}$
         * ตัวเลขที่เหลือสำหรับ $\{b, d\}$ คือ $\{3, 4\}$
         * ทดสอบเงื่อนไข (ก) $a+b < c+d$ โดยที่ $a = 5$:
           * หาก $c = 1$ จะได้ $5+b < 1+d \implies b - d < -4$ (ไม่มีคู่ใดใน $\{3,4\}$ สอดคล้อง)
           * หาก $c = 2$ จะได้ $5+b < 2+d \implies b - d < -3$ (ไม่มีคู่ใดใน $\{3,4\}$ สอดคล้อง)
       * **ถ้า $c+e = 4$** จะได้ $\{c, e\} = \{1, 3\}$
         * ตัวเลขที่เหลือสำหรับ $\{b, d\}$ คือ $\{2, 4\}$
         * ทดสอบเงื่อนไข (ก) $a+b < c+d$ โดยที่ $a = 5$:
           * หาก $c = 1$ จะได้ $5+b < 1+d \implies b - d < -4$ (ไม่มีคู่ใดใน $\{2,4\}$ สอดคล้อง)
           * หาก $c = 3$ จะได้ $5+b < 3+d \implies b - d < -2$ (ไม่มีคู่ใดใน $\{2,4\}$ สอดคล้อง)
     * ดังนั้น กรณีที่ $a = 5$ ไม่มีคำตอบที่สอดคล้อง

   * **กรณีที่ 2: $a = 4$**
     * จาก $c+e < a \implies c+e < 4$ ดังนั้น $c+e = 3$ เท่านั้น ซึ่งจะได้ $\{c, e\} = \{1, 2\}$
     * จาก $a < f \implies 4 < f$ และเหลือตัวเลือกของ $f$ จากตัวเลขที่เหลือที่ไม่ใช่ $\{1, 2, 4\}$ คือ $\{3, 5, 6\}$
       * **ถ้า $f = 5$:**
         * ตัวเลขที่เหลือคือ $\{3, 6\}$ จะเป็นค่าของ $\{b, d\}$
         * ทดสอบเงื่อนไข (ก) $a+b < c+d$ โดยที่ $a = 4$:
           * หาก $c = 1, e = 2$: จะได้ $4+b < 1+d \implies b - d < -3$ (ไม่มีคู่ใดใน $\{3,6\}$ สอดคล้อง)
           * หาก $c = 2, e = 1$: จะได้ $4+b < 2+d \implies b - d < -2$
             * ถ้าเลือก $b = 3$ และ $d = 6$ จะได้:
               $$4 + 3 < 2 + 6 \implies 7 < 8 \text{ (เป็นจริง)}$$
             * ได้คำตอบคือ: $a=4, b=3, c=2, d=6, e=1, f=5$ ซึ่งสอดคล้องกับทุกเงื่อนไข

3. **คำนวณหาค่าของ $a + 2b + 3c + 4d$:**
   แทนค่าตัวแปรที่หาได้:
   $$a = 4$$
   $$b = 3$$
   $$c = 2$$
   $$d = 6$$
   
   จะได้:
   $$a + 2b + 3c + 4d = 4 + 2(3) + 3(2) + 4(6)$$
   $$= 4 + 6 + 6 + 24$$
   $$= 40$$

ดังนั้น ค่าของ $a + 2b + 3c + 4d$ เท่ากับ **40**
