# คำถาม

21. ภาพแสดงการเกิดหินอัคนีในพื้นที่หนึ่ง และการทดลองโดยใช้ผลึกกำมะถันเพื่อให้เข้าใจขั้นตอนการเย็นตัวของแมกมาในตำแหน่ง A ถึง C ![illustration](../images/TEDET2559_Science_G8_slice25_crop_0.png)

[ขั้นตอนการทดลอง]
![illustration](../images/TEDET2559_Science_G8_slice25_crop_1.png)
1. ใส่ผงกำมะถันลงในหลอดทดลองและให้ความร้อนเพื่อให้ละลาย
2. การทดลองที่ (ก) : เท $\frac{1}{3}$ ของกำมะถันที่ละลายลงในสไตโรโฟมดังภาพ (ก)
3. การทดลองที่ (ข) : เท $\frac{1}{3}$ ของกำมะถันที่ละลายลงบนแผ่นกระจกดังภาพ (ข)
4. การทดลองที่ (ค) : เท $\frac{1}{3}$ ของกำมะถันที่ละลายลงในบีกเกอร์บรรจุน้ำเย็นดังภาพ (ค)
5. จากการทดลองที่ (ก) ถึง (ค) ลองเปรียบเทียบความเร็วในการเย็นตัวของกำมะถัน
6. เมื่อกำมะถันแข็งตัวเรียบร้อยแล้วให้ใช้มีดผ่าก้อนกำมะถันเพื่อสังเกตหน้าตัด

ข้อใดจับคู่การทดลองและชื่อหิน ของคำอธิบายต่อไปนี้ได้ถูกต้อง

การทดลองที่สอดคล้องกับหินอัคนีที่เกิดขึ้นในตำแหน่ง C คือ การทดลองใด [ขั้นตอนการทดลอง] ในภาพ ก - ค หินอัคนีชนิดหนึ่งที่เกิดขึ้นในตำแหน่ง C ที่ประกอบด้วยแร่ไม่มีสีจำนวนมากจนทำให้เกิดหินที่มีสีสว่างมีชื่อเรียกว่าหินอะไร

การทดลอง | ชื่อหิน
(1) | (ก) | หินไรโอไลต์
(2) | (ก) | หินแกรนิต
(3) | (ข) | หินแกบโบร
(4) | (ข) | หินแอนดีไซต์
(5) | (ค) | หินแกรนิต

# คำอธิบายและวิธีทำ

การทดลองในภาพใช้เปรียบเทียบอัตราการเย็นตัวของแมกมา: (ก) เทลงในสไตโรโฟม เป็นการเย็นตัวช้าๆ เนื่องจากสไตโรโฟมเป็นฉนวนความร้อน เปรียบได้กับแมกมาที่เย็นตัวใต้ผิวโลก (หินอัคนีแทรกซอน) (ข) เทลงบนแผ่นกระจก เป็นการเย็นตัวปานกลาง และ (ค) เทลงในน้ำเย็น เป็นการเย็นตัวอย่างรวดเร็ว (หินอัคนีพุ) ตำแหน่ง C ในภาพแสดงถึงตำแหน่งที่อยู่ลึกลงไปใต้เปลือกโลก (หินอัคนีแทรกซอน) ซึ่งเย็นตัวช้ามาก ทำให้ผลึกแร่มีขนาดใหญ่และหินมีสีสว่าง (Felsic) หินอัคนีที่เกิดจากการเย็นตัวช้าและมีองค์ประกอบแร่สีสว่างคือหินแกรนิต ส่วนการทดลองที่สอดคล้องกับการเย็นตัวช้าคือการทดลอง (ก) (เปรียบเทียบการเก็บความร้อนได้ดีที่สุด) ดังนั้น คู่ที่ถูกต้องคือการทดลอง (ก) และหินแกรนิต ซึ่งตรงกับข้อ (2)
