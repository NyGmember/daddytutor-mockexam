# คำถาม

9. จงหาว่าความยาวรอบรูปทั้งสองรูปต่อไปนี้ รวมกันเท่ากับกี่เซนติเมตร

# คำอธิบายและวิธีทำ

ในการหาความยาวรอบรูปของรูปทรงที่มีลักษณะเป็นขั้นบันไดนี้ เราสามารถใช้เทคนิคการมองภาพโดยการเลื่อนเส้นขอบ (shifting sides) เพื่อให้ง่ายขึ้นดังนี้: 1. สำหรับรูปบน: ความยาวรอบรูปประกอบด้วยด้านแนวนอนรวมกัน 2 ด้าน (บนและล่าง) และด้านแนวตั้งรวมกัน 2 ด้าน (ซ้ายและขวา) โดยด้านแนวนอนรวมกันได้ $48 + 48 = 96$ ซม. และด้านแนวตั้งรวมกันได้ $8 + 8 = 16$ ซม. ดังนั้นความยาวรอบรูปบน = $96 + 16 = 112$ ซม. 2. สำหรับรูปล่าง: ในทำนองเดียวกัน ความยาวด้านแนวนอนรวมกันได้ $48 + 48 = 96$ ซม. และด้านแนวตั้งรวมกันได้ $10 + 10 = 20$ ซม. ดังนั้นความยาวรอบรูปล่าง = $96 + 20 = 116$ ซม. 3. ความยาวรอบรูปทั้งสองรูปรวมกัน = $112 + 116 = 228$ ซม. สรุปคือสำหรับรูปทรงลักษณะนี้ ความยาวรอบรูปจะเท่ากับ 2 เท่าของผลรวมความกว้างและความยาวของกรอบสี่เหลี่ยมผืนผ้าที่ครอบรูปนั้นไว้
