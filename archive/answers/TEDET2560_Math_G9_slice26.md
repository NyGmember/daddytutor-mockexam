# คำถาม

26. นำกระเบื้อง 40 แผ่น มาวางเรียงต่อกันเป็นรูปสี่เหลี่ยมมุมฉาก 2 วง โดยวงในมีกระเบื้อง 16 แผ่น และวงนอกมีกระเบื้อง 24 แผ่น ทำให้เกิดเป็นบริเวณสี่เหลี่ยมจัตุรัสที่เว้นช่องตรงกลางไว้ ดังรูป

![illustration](../images/TEDET2560_Math_G9_slice26_crop_0.png)

เมื่อนำกระเบื้อง 700 แผ่น มาวางเรียงต่อกันเป็นรูปสี่เหลี่ยมมุมฉาก 5 วง เกิดเป็นบริเวณสี่เหลี่ยมจัตุรัสที่เว้นช่องตรงกลางไว้ในลักษณะเดียวกับข้างต้น จงหาว่ากระเบื้องบนด้านหนึ่งของวงนอกสุด และกระเบื้องบนด้านหนึ่งของวงในสุดมีจำนวนรวมกันกี่แผ่น

# คำอธิบายและวิธีทำ

ให้ $n_k$ เป็นจำนวนกระเบื้องต่อหนึ่งด้านของวงที่ $k$ นับจากวงในสุด (วงในสุดคือ $k=1$) และ $x$ เป็นจำนวนกระเบื้องในแต่ละด้านของช่องว่างตรงกลาง (รูปสี่เหลี่ยมจัตุรัสขนาด $x \times x$) ดังนั้น วงในสุด (วงที่ 1) จะมีด้านยาว $x+2$ กระเบื้อง จำนวนกระเบื้องของวงที่ $k$ คือ $N_k = (x+2k)^2 - (x+2k-2)^2 = 8(x+2k-1)$ กระเบื้อง โดยที่ $x$ คือจำนวนกระเบื้องด้านละ $x$ ของช่องว่างตรงกลาง จากโจทย์มี 5 วง ผลรวมกระเบื้องทั้งหมดคือ $\sum_{k=1}^5 8(x+2k-1) = 8[5x + 2(1+2+3+4+5) - 5] = 8(5x + 25) = 40x + 200 = 700$ ดังนั้น $40x = 500$ จะได้ $x = 12.5$ เนื่องจาก $x$ ต้องเป็นจำนวนเต็ม สังเกตจากรูป วงในสุด ($k=1$) มีด้านยาว $x+2$ และวงนอกสุด ($k=5$) มีด้านยาว $x+10$ ด้านของวงในสุดมี $x+2$ กระเบื้องและวงนอกสุดมี $x+10$ กระเบื้อง แต่ละวงมีจำนวนกระเบื้องตามสูตร $4 \times (\text{ด้าน}) - 4$ กระเบื้องมุมซ้อน หากวงที่ 1 มี $x+2$ กระเบื้องต่อด้าน จำนวนกระเบื้องวงในสุดคือ $4(x+2)-4 = 16$ จะได้ $x=2$ วงที่ 5 มีด้านยาว $x+2(5) = 12$ กระเบื้องต่อด้าน จำนวนกระเบื้องคือ $4(12)-4 = 44$ ตรวจสอบผลรวม $N_k = 4(x+2k)-4 = 4x+8k-4$ ผลรวม $\sum_{k=1}^5 (4x+8k-4) = 20x + 8(15) - 20 = 20x + 100 = 700$ ดังนั้น $20x = 600$ จะได้ $x = 30$ จำนวนกระเบื้องต่อด้านของวงนอกสุด ($k=5$) คือ $30 + 2(5) = 40$ แผ่น และวงในสุด ($k=1$) คือ $30 + 2(1) = 32$ แผ่น ผลรวมคือ $40 + 32 = 72$
