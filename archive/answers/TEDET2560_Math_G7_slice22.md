# คำถาม

ตัวอย่าง
ระยะทางระหว่าง B กับ C เท่ากับ 52.3 กิโลเมตร
ระยะทางระหว่าง A กับ E เท่ากับ 152.3 กิโลเมตร

(1) 72.3 กิโลเมตร (2) 78.8 กิโลเมตร
(3) 83.5 กิโลเมตร (4) 88.5 กิโลเมตร

# คำอธิบายและวิธีทำ

จากตารางระยะทางระหว่างสถานที่ A, B, C, D และ E บนถนนสายเดียวกัน ซึ่งเรียงลำดับจาก A ไป E เราสามารถวิเคราะห์ความสัมพันธ์ของระยะทางได้ดังนี้: ให้ระยะทางจาก A ไป B คือ $AB$, จาก B ไป C คือ $BC$, จาก C ไป D คือ $CD$ และจาก D ไป E คือ $DE$ ตามลำดับ จากข้อมูลในตาราง: 1. ระยะทางจาก B ไป C ($BC$) = 52.3 กิโลเมตร 2. ระยะทางจาก C ไป E ($CE$) = 68.8 กิโลเมตร 3. ระยะทางจาก D ไป E ($DE$) = 32.6 กิโลเมตร เนื่องจาก $CE = CD + DE$ ดังนั้น $68.8 = CD + 32.6$ จะได้ $CD = 68.8 - 32.6 = 36.2$ กิโลเมตร นอกจากนี้ จากตาราง ระยะทางจาก A ไป D ($AD$) = 119.7 กิโลเมตร และเราทราบว่า $AD = AB + BC + CD$ หรือ $AD = AB + (BC + CD)$ แทนค่าจะได้ $119.7 = AB + 52.3 + 36.2$ ดังนั้น $AB = 119.7 - 88.5 = 31.2$ กิโลเมตร สิ่งที่โจทย์ต้องการคือระยะทางจาก A ไป C ($AC$) ซึ่งคือผลรวมของระยะทาง $AB + BC$ นั่นคือ $AC = 31.2 + 52.3 = 83.5$ กิโลเมตร
