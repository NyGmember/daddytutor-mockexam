# คำถาม

13. ตารางเปรียบเทียบส่วนประกอบของเลือดของคนสี่คน A ~ D ที่ใช้ชีวิตอยู่ต่างสถานที่กันกับคนปกติ

![illustration](../images/TEDET2558_Science_G9_slice13_crop_0.png)

จากข้อความ ข้อใดอธิบายเกี่ยวกับสภาพร่างกายของ A ~ D ได้ถูกต้อง

a. A เลือดไม่แข็งตัว
b. B มีแบคทีเรียรุกล้ำเข้ามาในร่างกายจึงเกิดการอักเสบ
c. C มีอาการโรคโลหิตจาง
d. D อาศัยอยู่ในพื้นที่ภูเขาสูง

(1) a, b
(2) b, c
(3) c, d
(4) a, b, d
(5) b, c, d

# คำอธิบายและวิธีทำ

พิจารณาข้อมูลในตารางเปรียบเทียบกับค่าปกติ:
1. A: มีเกล็ดเลือด 7 หมื่น (ต่ำกว่าเกณฑ์ปกติ 25-40 หมื่น) ส่งผลให้การแข็งตัวของเลือดผิดปกติ แต่ไม่ได้หมายความว่าเลือดจะไม่แข็งตัวเลย ข้อ a จึงไม่ถูกต้อง
2. B: มีเม็ดเลือดขาว 20,000 (สูงกว่าเกณฑ์ปกติ 6,000-8,000) ซึ่งเป็นกลไกที่ร่างกายสร้างเม็ดเลือดขาวเพิ่มขึ้นเพื่อต่อสู้กับเชื้อโรค เช่น แบคทีเรีย ทำให้เกิดการอักเสบได้ ข้อ b จึงถูกต้อง
3. C: มีเม็ดเลือดแดง 750 หมื่น (สูงกว่าปกติ 450-500 หมื่น) ไม่ใช่อาการโลหิตจาง (ซึ่งเกิดจากเม็ดเลือดแดงต่ำ) ข้อ c จึงไม่ถูกต้อง
4. D: มีเม็ดเลือดแดง 150 หมื่น (ต่ำกว่าปกติ 450-500 หมื่น) แสดงว่าเม็ดเลือดแดงน้อย ร่างกายจึงต้องปรับตัวหากอาศัยอยู่ในที่สูงที่มีออกซิเจนน้อยด้วยการเพิ่มเม็ดเลือดแดง ซึ่งนี่แสดงถึงภาวะตรงกันข้าม ข้อ d จึงไม่ถูกต้อง
ดังนั้น ข้อความที่ถูกต้องคือข้อ b เพียงข้อเดียว แต่ในตัวเลือกที่กำหนดให้ (1)-(5) ไม่มีข้อ b เพียงอย่างเดียว เมื่อพิจารณาโจทย์ใหม่ หากเทียบจากตัวเลือกที่มี b อยู่ จะพบว่าตัวเลือก (2), (5) มี b และ c/d หากตรวจสอบใหม่ c ไม่ใช่และ d ไม่ใช่ ดังนั้นโจทย์นี้อาจมีความผิดพลาดของตัวเลือก หรือต้องวิเคราะห์ใหม่ว่าหากผู้ป่วย C มีภาวะผิดปกติอื่น หรือ D คือกรณีอื่น แต่จากข้อมูลในตาราง b คือคำตอบที่ชัดเจนที่สุดในทางชีววิทยาเบื้องต้น
