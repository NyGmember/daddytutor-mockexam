# คำถาม

11. กู้เงินจากธนาคารโดยกู้เงินรวมกับดอกเบี้ยรวม 17 ปี อัตราดอกเบี้ยร้อยละ 57 คิดแบบคงที่และผ่อนชำระคืนเท่ากันทุกเดือน

ถ้ากู้เงินรวมทั้งหมด 300 เมตร และใช้เวลา 1,146 วัน ในการผ่อนชำระคืน 
จงหาว่ากู้เงินทั้งหมดได้รับอัตราดอกเบี้ยเป็นกี่เปอร์เซ็นต์
ต่อปี

# คำอธิบายและวิธีทำ

จากโจทย์ที่กำหนดให้: 
1. จำนวนเงินกู้รวมทั้งหมด (ต้นเงิน + ดอกเบี้ย) คือ 300 หน่วย (ในโจทย์ระบุหน่วยเป็น เมตร ซึ่งในบริบททางธุรกิจน่าจะเป็นการพิมพ์ผิดจาก บาท ให้ใช้ค่า 300 ในการคำนวณ)
2. ระยะเวลาการกู้ทั้งหมดคือ 17 ปี แต่มีการระบุระยะเวลาการผ่อนชำระจริงคือ 1,146 วัน 
3. อัตราดอกเบี้ยรวม 57% ต่อระยะเวลา 17 ปี แบบคงที่ (Simple Interest)

ขั้นตอนการคำนวณ:
- คำนวณดอกเบี้ยทั้งหมด: ดอกเบี้ยรวม = 57% ของเงินต้น ถ้าให้ P คือเงินต้น จะได้ ดอกเบี้ย = 0.57P ดังนั้น เงินรวม (ต้น + ดอกเบี้ย) = P + 0.57P = 1.57P
- จากโจทย์ เงินรวม = 300 ดังนั้น 1.57P = 300 จะได้ P = 300 / 1.57 ≈ 191.08
- ดอกเบี้ยทั้งหมดคือ 300 - 191.08 = 108.92
- ระยะเวลาที่ใช้จริงในการผ่อนคือ 1,146 วัน คิดเป็นปี (สมมติ 1 ปี = 365 วัน): t = 1,146 / 365 ≈ 3.14 ปี
- ใช้สูตรดอกเบี้ยคงที่: ดอกเบี้ย (I) = เงินต้น (P) * อัตราดอกเบี้ย (r) * เวลา (t)
- 108.92 = 191.08 * r * 3.14
- r = 108.92 / (191.08 * 3.14) ≈ 108.92 / 600 ≈ 0.1815
- ดังนั้น อัตราดอกเบี้ยต่อปีคือประมาณ 18.15%
