# คำถาม

2. พิจารณาข้อมูลต่อไปนี้

| | |
|---|---|
| ⓐ | นำผลที่ได้จากการทดลองไปวิเคราะห์และหาความสัมพันธ์ |
| ⓑ | การลงความเห็นเบื้องต้นเกี่ยวกับปัญหาโดยใช้ความรู้หรือประสบการณ์ทางวิทยาศาสตร์ |
| ⓒ | ทำการทดลองเพื่อพิสูจน์สมมติฐาน |
| ⓓ | การค้นพบปัญหาโดยการสังเกตปรากฏการณ์ธรรมชาติแล้วออกไปค้นคว้าในสิ่งที่สงสัย |
| ⓔ | การวินิจฉัยผลการวิเคราะห์ข้อมูลว่าสอดคล้องกับสมมติฐานที่วางไว้ในตอนแรกหรือไม่ |

ข้อใดเรียงลำดับกระบวนการค้นคว้าทางวิทยาศาสตร์ได้ถูกต้อง

① ⓐ $\rightarrow$ ⓓ $\rightarrow$ ⓑ $\rightarrow$ ⓔ $\rightarrow$ ⓒ
② ⓐ $\rightarrow$ ⓔ $\rightarrow$ ⓓ $\rightarrow$ ⓑ $\rightarrow$ ⓒ
③ ⓑ $\rightarrow$ ⓒ $\rightarrow$ ⓐ $\rightarrow$ ⓔ $\rightarrow$ ⓓ
④ ⓓ $\rightarrow$ ⓑ $\rightarrow$ ⓐ $\rightarrow$ ⓔ $\rightarrow$ ⓒ
⑤ ⓓ $\rightarrow$ ⓑ $\rightarrow$ ⓒ $\rightarrow$ ⓐ $\rightarrow$ ⓔ

# คำอธิบายและวิธีทำ

กระบวนการค้นคว้าหรือวิธีการทางวิทยาศาสตร์ (Scientific Method) ประกอบด้วยขั้นตอนที่เป็นระบบตามลำดับดังนี้:

1. **การตั้งปัญหาหรือตั้งคำถาม (ⓓ):** เริ่มต้นจากการสังเกตปรากฏการณ์ธรรมชาติ แล้วนำมาสู่การระบุปัญหาหรือเรื่องที่ต้องการศึกษาค้นคว้า
2. **การตั้งสมมติฐาน (ⓑ):** เป็นการคาดคะเนคำตอบของปัญหาหรือการอธิบายเบื้องต้นอย่างมีเหตุผล โดยอาศัยความรู้และประสบการณ์เดิมที่มี
3. **การทดลองเพื่อตรวจสอบสมมติฐาน (ⓒ):** วางแผนและดำเนินกระบวนการทดลองเพื่อรวบรวมข้อมูลเชิงประจักษ์มาพิสูจน์สมมติฐานที่ตั้งไว้
4. **การวิเคราะห์ข้อมูล (ⓐ):** นำข้อมูลที่ได้จากการทดลองมาประมวลผล วิเคราะห์เชิงความสัมพันธ์ เพื่ออธิบายข้อเท็จจริง
5. **การสรุปผลการทดลอง (ⓔ):** นำผลการวิเคราะห์ข้อมูลมาพิจารณาเปรียบเทียบว่าสอดคล้องหรือขัดแย้งกับสมมติฐานที่ตั้งไว้ตอนแรก เพื่อหาข้อสรุปเชิงวิทยาศาสตร์

ดังนั้น ลำดับกระบวนการสืบเสาะค้นคว้าทางวิทยาศาสตร์ที่ถูกต้องคือ:
$$\text{ⓓ} \rightarrow \text{ⓑ} \rightarrow \text{ⓒ} \rightarrow \text{ⓐ} \rightarrow \text{ⓔ}$$

ซึ่งสอดคล้องกับตัวเลือก **5**
