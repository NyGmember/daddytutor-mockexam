# คำถาม

21. ถ้า $\sqrt{3} + \sqrt{5} = n + a$ โดยที่ $n$ เป็นจำนวนเต็ม และ $0 \le a < 1$ (กล่าวคือ $n$ เป็นส่วนจำนวนเต็ม และ $a$ เป็นส่วนทศนิยมของ $\sqrt{3} + \sqrt{5}$) จงหาค่าของ $b$ ที่สอดคล้องกับสมการต่อไปนี้
$$a^2 + 6a + 1 = \sqrt{b}$$

# คำอธิบายและวิธีทำ

จากที่โจทย์กำหนด:
$$\sqrt{3} + \sqrt{5} = n + a$$
เมื่อ $n$ เป็นจำนวนเต็ม (ส่วนจำนวนเต็ม) และ $0 \le a < 1$ (ส่วนทศนิยม)

---

### 1. หาค่าของ $n$ และรูปของ $a$:
ประมาณค่าของ $\sqrt{3}$ และ $\sqrt{5}$:
* $\sqrt{3} \approx 1.732$
* $\sqrt{5} \approx 2.236$
ดังนั้น:
$$\sqrt{3} + \sqrt{5} \approx 1.732 + 2.236 = 3.968$$

เนื่องจาก $3 \le \sqrt{3} + \sqrt{5} < 4$ จะได้ส่วนจำนวนเต็มคือ:
$$n = 3$$

ดังนั้น ส่วนทศนิยม $a$ คือ:
$$a = (\sqrt{3} + \sqrt{5}) - 3$$

---

### 2. จัดรูปสมการเพื่อหาค่าของ $a^2 + 6a + 1$:
จาก $a = \sqrt{3} + \sqrt{5} - 3$ ย้ายข้างจำนวนเต็มไปอยู่ทางฝั่ง $a$:
$$a + 3 = \sqrt{3} + \sqrt{5}$$

ยกกำลังสองทั้งสองข้างของสมการ:
$$(a + 3)^2 = (\sqrt{3} + \sqrt{5})^2$$

ใช้สูตรกำลังสองสมบูรณ์:
* ฝั่งซ้าย: $(a + 3)^2 = a^2 + 6a + 9$
* ฝั่งขวา: $(\sqrt{3} + \sqrt{5})^2 = 3 + 2\sqrt{15} + 5 = 8 + 2\sqrt{15}$

แทนกลับลงในสมการ:
$$a^2 + 6a + 9 = 8 + 2\sqrt{15}$$

ลบด้วย $8$ ทั้งสองข้างของสมการ เพื่อให้ได้พจน์ $a^2 + 6a + 1$ ตามที่ต้องการ:
$$(a^2 + 6a + 9) - 8 = (8 + 2\sqrt{15}) - 8$$
$$a^2 + 6a + 1 = 2\sqrt{15}$$

---

### 3. หาค่าของ $b$:
จากเงื่อนไขในโจทย์:
$$a^2 + 6a + 1 = \sqrt{b}$$
แทนค่าที่ได้จากการคำนวณลงไป:
$$\sqrt{b} = 2\sqrt{15}$$

ยกกำลังสองทั้งสองข้างเพื่อหาค่า $b$:
$$b = (2\sqrt{15})^2$$
$$b = 4 \times 15 = 60$$

ดังนั้น ค่าของ $b$ คือ **60**
