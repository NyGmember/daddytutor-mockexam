# คำถาม

14. ถ้า $x = 1, y = -1$ เป็นคำตอบของ
$$\frac{2ax+by}{3} = \frac{ax-by}{4} = \frac{3x-y}{2}$$
จงหาค่าของ $6(a-b)$

# คำอธิบายและวิธีทำ

เนื่องจาก $x = 1, y = -1$ เป็นคำตอบของระบบสมการที่กำหนดให้:
$$\frac{2ax+by}{3} = \frac{ax-by}{4} = \frac{3x-y}{2}$$

เราสามารถแทนค่า $x = 1$ และ $y = -1$ ลงในสมการเพื่อหาความสัมพันธ์ของ $a$ และ $b$:

1. **คำนวณค่าตัวเลขในส่วนสุดท้ายของสมการ:**
   $$\frac{3x-y}{2} = \frac{3(1) - (-1)}{2} = \frac{3+1}{2} = \frac{4}{2} = 2$$

2. **ตั้งสมการความสัมพันธ์ของ $a$ และ $b$:**
   * **จากพจน์แรก:**
     $$\frac{2a(1) + b(-1)}{3} = 2 \implies \frac{2a-b}{3} = 2$$
     $$2a - b = 6 \quad \text{--- (สมการที่ 1)}$$
   
   * **จากพจน์ที่สอง:**
     $$\frac{a(1) - b(-1)}{4} = 2 \implies \frac{a+b}{4} = 2$$
     $$a + b = 8 \quad \text{--- (สมการที่ 2)}$$

3. **แก้ระบบสมการหาค่า $a$ และ $b$:**
   * นำสมการที่ 1 บวกกับสมการที่ 2:
     $$(2a - b) + (a + b) = 6 + 8$$
     $$3a = 14 \implies a = \frac{14}{3}$$
   
   * แทนค่า $a = \frac{14}{3}$ ในสมการที่ 2:
     $$b = 8 - \frac{14}{3} = \frac{24-14}{3} = \frac{10}{3}$$

4. **คำนวณหาค่า $6(a-b)$:**
   $$6(a-b) = 6\left(\frac{14}{3} - \frac{10}{3}\right)$$
   $$= 6\left(\frac{4}{3}\right)$$
   $$= 2 \times 4 = 8$$

ดังนั้น ค่าของ $6(a-b)$ คือ **8**
