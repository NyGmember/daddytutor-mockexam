# คำถาม

27. โรงงานน้ำแข็งแห่งหนึ่งผลิตน้ำแข็งได้วันละ $a$ ตัน และมีน้ำแข็งที่ผลิตไว้แล้ว 3,200 ตัน

หากเครื่องจักรทำงานได้ตามปกติ จะสามารถส่งน้ำแข็งให้ร้านค้าวันละ $b$ ตัน ได้เป็นเวลา 40 วัน

แต่ถ้าเครื่องจักรทำงานผิดปกติ ปริมาณการผลิตต่อวันจะเหลือเพียง $\frac{1}{4}$ ของปริมาณการผลิตปกติ และสามารถส่งน้ำแข็งให้ร้านค้าวันละ $b$ ตัน ได้เป็นเวลา 16 วัน เท่านั้น

ถ้าเครื่องจักรทำงานผิดปกติ แต่ต้องการส่งน้ำแข็งให้ร้านค้าได้เป็นเวลา 40 วัน ในปริมาณที่เท่ากันทุกวัน จงหาว่าต้องส่งน้ำแข็งวันละกี่ตัน

# คำอธิบายและวิธีทำ

จากโจทย์ เราสามารถตั้งสมการของปริมาณน้ำแข็งทั้งหมดที่มีได้ดังนี้: 1. กรณีปกติ: ปริมาณน้ำแข็งที่มีคือ น้ำแข็งที่ผลิตได้ใน 40 วัน รวมกับของเดิม คือ 40a + 3,200 ตัน ซึ่งส่งให้ร้านวันละ b ตัน เป็นเวลา 40 วัน ได้ว่า 40a + 3,200 = 40b หรือหารด้วย 40 ทั้งสองข้างจะได้ a + 80 = b ... (สมการที่ 1). 2. กรณีเครื่องจักรผิดปกติ: ผลิตได้วันละ a/4 ตัน เป็นเวลา 16 วัน รวมของเดิม คือ 16(a/4) + 3,200 = 4a + 3,200 ตัน ซึ่งส่งให้ร้านวันละ b ตัน เป็นเวลา 16 วัน ได้ว่า 4a + 3,200 = 16b ... (สมการที่ 2). จาก (สมการที่ 1) คือ b = a + 80 นำไปแทนค่าใน (สมการที่ 2): 4a + 3,200 = 16(a + 80) => 4a + 3,200 = 16a + 1,280 => 1,920 = 12a => a = 160. แทนค่า a ใน (สมการที่ 1) จะได้ b = 160 + 80 = 240. ตอนนี้เราต้องการส่งน้ำแข็งให้ได้ 40 วัน โดยเครื่องจักรผลิตได้วันละ a/4 = 160/4 = 40 ตัน. ปริมาณน้ำแข็งรวมที่ต้องใช้คือ ผลิต 40 วัน (40 * 40 = 1,600) บวกของเดิม 3,200 รวมเป็น 4,800 ตัน. หากต้องการส่งให้ร้านค้าวันละ x ตัน เป็นเวลา 40 วัน จะได้ 40x = 4,800 ดังนั้น x = 4,800 / 40 = 120 ตันต่อวัน
