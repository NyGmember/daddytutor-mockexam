# คำถาม

12. หมากทำการทดลองต่อไปนี้เพื่อเปรียบเทียบความแข็งของแร่ A – D

![illustration](../images/TEDET2558_Science_G8_slice13_crop_0.png)

จงเรียงลำดับแร่ที่มีความแข็งน้อยที่สุดไปมากที่สุด
① A – B – C – D
② A – C – D – B
③ B – C – D – A
④ C – B – A – D
⑤ D – C – B – A

# คำอธิบายและวิธีทำ

ในการเปรียบเทียบความแข็งของแร่ตามมาตราความแข็ง (Mohs hardness scale) เราใช้หลักการว่า 'แร่ที่แข็งกว่าจะสามารถขูดแร่ที่อ่อนกว่าให้เป็นรอยได้' จากข้อมูลการทดลอง เราสามารถวิเคราะห์ความสัมพันธ์ได้ดังนี้: 1. เมื่อนำ A ไปขูดกับ B และ C แล้ว A เกิดรอย แสดงว่าทั้ง B และ C มีความแข็งมากกว่า A (ลำดับ: A < B และ A < C). 2. เมื่อนำ B ไปขูดกับ C แล้วไม่มีรอยเกิดขึ้นที่ C แสดงว่า B มีความแข็งน้อยกว่าหรือเท่ากับ C (ลำดับ: B < C). 3. เมื่อนำ D ไปขูดกับ C แล้วเกิดรอยแตกขึ้นที่ C แสดงว่า D มีความแข็งมากกว่า C (ลำดับ: C < D). จากความสัมพันธ์ข้างต้น เราจะได้ลำดับความแข็งจากน้อยไปมากดังนี้: A < B < C < D โดยที่ A อ่อนที่สุด และ D แข็งที่สุด ซึ่งตรงกับตัวเลือกที่ 1
