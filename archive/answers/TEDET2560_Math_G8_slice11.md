# คำถาม

11. นำทรงกระบอก ทรงกลม และลูกบาศก์จำนวนหนึ่งวางบนเครื่องชั่งสองแขน 3 เครื่อง ทำให้เครื่องชั่ง A และเครื่องชั่ง B สมดุล ดังรูป

![illustration](../images/TEDET2560_Math_G8_slice11_crop_0.png)

ถ้าต้องการให้เครื่องชั่ง C สมดุล จะต้องนำรูปทรงในข้อใดต่อไปนี้วางเพิ่มลงในด้านซ้ายของเครื่องชั่ง C

(1) ทรงกลม 3 ลูก
(2) ทรงกลม 1 ลูก และลูกบาศก์ 1 ลูก
(3) ลูกบาศก์ 2 ลูก
(4) ทรงกลม 2 ลูก และทรงกระบอก 1 แท่ง
(5) ทรงกระบอก 1 แท่ง และลูกบาศก์ 1 ลูก

# คำอธิบายและวิธีทำ

กำหนดให้: C = ทรงกระบอก, S = ทรงกลม, K = ลูกบาศก์
จากเครื่องชั่ง A: 1C + 1S = 2K  ...(1)
จากเครื่องชั่ง B: 4S = 1K + 1C  ...(2)
จากสมการ (1) จะได้ 1C = 2K - 1S
แทนค่า 1C ในสมการ (2): 4S = 1K + (2K - 1S)
4S + 1S = 3K
5S = 3K ดังนั้น 1K = 5/3 S
แทนค่า 1K ลงในสมการ (1): 1C + 1S = 2(5/3 S) = 10/3 S
1C = 10/3 S - 3/3 S = 7/3 S
ปัจจุบันเครื่องชั่ง C มีด้านซ้ายเป็น 1C และด้านขวาเป็น 3K
เปรียบเทียบค่าความหนัก: 
ซ้าย = 7/3 S
ขวา = 3K = 3(5/3 S) = 5S = 15/3 S
ผลต่างระหว่างขวาและซ้าย คือ 15/3 S - 7/3 S = 8/3 S
หากเราลองบวกเพิ่ม 1C + 1K เข้าไปที่ด้านซ้าย: 
(7/3 S) + (7/3 S + 5/3 S) = 19/3 S (ไม่สมดุล)
หากบวก 1C และ 1K ตามข้อ 5: 
1C + 1K = 7/3 S + 5/3 S = 12/3 S = 4S
บวกเข้ากับฝั่งซ้ายที่มีเดิมอยู่ 1C (7/3 S): รวมเป็น 7/3 S + 12/3 S = 19/3 S (ไม่สมดุล)
ลองใหม่: จาก 1C = 2K - 1S, แทนลงใน C เพื่อหาว่าต้องเพิ่มอะไรเพื่อให้เท่ากับ 3K:
ฝั่งซ้ายต้องการเพิ่ม X เพื่อให้ 1C + X = 3K
แทน 1C ด้วย 2K - 1S: (2K - 1S) + X = 3K
X = 1K + 1S 
ดังนั้นต้องเพิ่ม ลูกบาศก์ 1 ลูก และ ทรงกลม 1 ลูก ซึ่งตรงกับตัวเลือกที่ 2
