# คำถาม

12. จากรูป เส้นตรง $y=x+2$ ตัดกับเส้นตรง $y=-x+k$ ที่จุด D [![illustration](../images/TEDET2558_Math_G9_slice21_crop_0.png)] ถ้า $\triangle BDC$ มีพื้นที่เป็น 2 เท่าของ $\triangle AOB$ จงหาค่าของ $(k-2)^2$

# คำอธิบายและวิธีทำ

ขั้นตอนในการหาค่าของ $(k-2)^2$ มีดังนี้:

1.  **หารายละเอียดของเส้นตรงและจุดตัด:**
    *   เส้นตรงแรก: $L_1: y = x + 2$
    *   เส้นตรงที่สอง: $L_2: y = -x + k$

2.  **หาจุดพิกัดของแต่ละจุดที่เกี่ยวข้อง:**
    *   **จุด O:** คือจุดกำเนิด มีพิกัด $(0, 0)$ โดยอัตโนมัติ
    *   **จุด A:** เป็นจุดตัดของเส้นตรง $L_1$ กับแกน X (เมื่อ $y=0$)
        $0 = x + 2 \Rightarrow x = -2$
        ดังนั้น จุด A มีพิกัด $(-2, 0)$
    *   **จุด B:** เป็นจุดตัดของเส้นตรง $L_1$ กับแกน Y (เมื่อ $x=0$)
        $y = 0 + 2 \Rightarrow y = 2$
        ดังนั้น จุด B มีพิกัด $(0, 2)$
    *   **จุด C:** เป็นจุดตัดของเส้นตรง $L_2$ กับแกน Y (เมื่อ $x=0$)
        $y = -0 + k \Rightarrow y = k$
        ดังนั้น จุด C มีพิกัด $(0, k)$
        จากรูปภาพ จุด C อยู่สูงกว่าจุด B บนแกน Y ดังนั้น $k > 2$
    *   **จุด D:** เป็นจุดตัดของเส้นตรง $L_1$ และ $L_2$ 
        $x + 2 = -x + k$
        $2x = k - 2$
        $x = \frac{k - 2}{2}$
        แทนค่า $x$ ในสมการ $y = x + 2$:
        $y = \frac{k - 2}{2} + 2 = \frac{k - 2 + 4}{2} = \frac{k + 2}{2}$
        ดังนั้น จุด D มีพิกัด $\left(\frac{k - 2}{2}, \frac{k + 2}{2}\right)$

3.  **คำนวณพื้นที่ของ $\triangle AOB$:**
    สามเหลี่ยม AOB มีฐานอยู่บนแกน Y และความสูงขนานแกน X
    *   ฐาน OB มีความยาวเท่ากับระยะห่างระหว่าง $(0,0)$ กับ $(0,2)$ คือ $|2-0|=2$ หน่วย
    *   ความสูงคือระยะห่างจากจุด A $(-2,0)$ ไปยังแกน Y คือ $|-2|=2$ หน่วย
    *   พื้นที่ $\triangle AOB = \frac{1}{2} \times \text{ฐาน} \times \text{ความสูง} = \frac{1}{2} \times 2 \times 2 = 2$ ตารางหน่วย

4.  **คำนวณพื้นที่ของ $\triangle BDC$:**
    สามเหลี่ยม BDC มีฐานอยู่บนแกน Y และความสูงขนานแกน X
    *   ฐาน BC มีความยาวเท่ากับระยะห่างระหว่าง $(0,2)$ กับ $(0,k)$ คือ $|k-2|$ หน่วย
        เนื่องจาก $k>2$ ดังนั้น ฐาน BC มีความยาว $k-2$ หน่วย
    *   ความสูงคือระยะห่างจากจุด D $\left(\frac{k - 2}{2}, \frac{k + 2}{2}\right)$ ไปยังแกน Y คือ $x$-พิกัดของจุด D ซึ่งก็คือ $\left|\frac{k - 2}{2}\right|$ หน่วย
        เนื่องจาก $k>2$ ดังนั้น $\frac{k-2}{2} > 0$ ความสูงคือ $\frac{k-2}{2}$ หน่วย
    *   พื้นที่ $\triangle BDC = \frac{1}{2} \times \text{ฐาน} \times \text{ความสูง} = \frac{1}{2} \times (k - 2) \times \frac{k - 2}{2} = \frac{(k - 2)^2}{4}$ ตารางหน่วย

5.  **ใช้เงื่อนไขพื้นที่ที่กำหนดให้:**
    โจทย์ระบุว่าพื้นที่ $\triangle BDC$ เป็น 2 เท่าของพื้นที่ $\triangle AOB$
    พื้นที่ $\triangle BDC = 2 \times$ พื้นที่ $\triangle AOB$
    $\frac{(k - 2)^2}{4} = 2 \times 2$
    $\frac{(k - 2)^2}{4} = 4$

6.  **หาค่าของ $(k-2)^2$:**
    คูณ 4 ทั้งสองข้างของสมการ:
    $(k - 2)^2 = 4 \times 4$
    $(k - 2)^2 = 16$

ดังนั้น ค่าของ $(k-2)^2$ คือ 16
