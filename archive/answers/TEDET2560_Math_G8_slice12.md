# คำถาม

12. รูปสามเหลี่ยมมุมฉากหน้าจั่ว ABC มี AB = AC และจุดยอด A อยู่บนเส้นตรง $\ell$ ลากส่วนของเส้นตรงจากจุดยอด B และ C มาตั้งฉากกับเส้นตรง $\ell$ ที่จุด D และ E ตามลำดับ ดังรูป ![illustration](../images/TEDET2560_Math_G8_slice12_crop_0.png) ถ้า BD = 12 เซนติเมตร และ CE = 5 เซนติเมตร จงหาว่า DE เท่ากับกี่เซนติเมตร

# คำอธิบายและวิธีทำ

พิจารณารูปสามเหลี่ยมมุมฉากหน้าจั่ว $\triangle ABC$ โดยมี $AB = AC$ และ $\angle BAC = 90^\circ$ \n\n1. กำหนดให้ $\angle ABD = \alpha$ เนื่องจาก $\triangle BDA$ เป็นรูปสามเหลี่ยมมุมฉาก จะได้ว่า $\angle BAD = 90^\circ - \alpha$ \n2. เนื่องจาก $\angle BAC = 90^\circ$ และ $\angle DAE = 180^\circ$ (เป็นเส้นตรง) จะได้ว่า $\angle CAE = 180^\circ - 90^\circ - (90^\circ - \alpha) = \alpha$ \n3. พิจารณารูปสามเหลี่ยม $\triangle BDA$ และ $\triangle AEC$: \n   - $AB = AC$ (โจทย์กำหนดให้)\n   - $\angle ADB = \angle CEA = 90^\circ$ (โจทย์กำหนดให้)\n   - $\angle ABD = \angle CAE = \alpha$ (จากการพิสูจน์ข้างต้น) \nดังนั้น $\triangle BDA \cong \triangle AEC$ (ความสัมพันธ์แบบ มุม-มุม-ด้าน) \n4. จากความเท่ากันทุกประการ จะได้ว่า $AD = CE = 5$ เซนติเมตร และ $BD = AE = 12$ เซนติเมตร \n5. ดังนั้น ความยาวของ $DE = DA + AE = 5 + 12 = 17$ เซนติเมตร
