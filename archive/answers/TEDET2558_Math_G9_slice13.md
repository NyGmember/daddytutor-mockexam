# คำถาม

23. ถ้า $a$ และ $b$ เป็นจำนวนจริงซึ่ง

$$a^2 + ab + b^2 = 40$$ และ $$a^2 - ab + b^2 = 24$$

จงหาค่าของ $(a - b)^2$

# คำอธิบายและวิธีทำ

เรามีสมการสองสมการคือ:
(1) $a^2 + ab + b^2 = 40$
(2) $a^2 - ab + b^2 = 24$

เป้าหมายของเราคือการหาค่าของ $(a - b)^2$ ซึ่งขยายความได้เป็น $a^2 - 2ab + b^2$ 

ขั้นตอนแรก: นำสมการ (1) ลบด้วยสมการ (2) เพื่อกำจัด $a^2$ และ $b^2$:
$(a^2 + ab + b^2) - (a^2 - ab + b^2) = 40 - 24$
$a^2 - a^2 + ab - (-ab) + b^2 - b^2 = 16$
$2ab = 16$
$ab = 8$

ขั้นตอนที่สอง: หาค่า $a^2 + b^2$ โดยนำสมการ (1) และ (2) มาบวกกัน:
$(a^2 + ab + b^2) + (a^2 - ab + b^2) = 40 + 24$
$2a^2 + 2b^2 = 64$
หารด้วย 2 ทั้งสองข้าง:
$a^2 + b^2 = 32$

ขั้นตอนที่สาม: คำนวณค่าของ $(a - b)^2$:
$(a - b)^2 = a^2 - 2ab + b^2$
จัดกลุ่มใหม่เป็น $(a^2 + b^2) - 2ab$
แทนค่าที่เราหามาได้:
$(a^2 + b^2) = 32$ และ $ab = 8$
$(a - b)^2 = 32 - 2(8)$
$(a - b)^2 = 32 - 16$
$(a - b)^2 = 16$
