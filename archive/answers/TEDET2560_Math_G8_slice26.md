# คำถาม

26. รถยนต์คันหนึ่งแล่นเข้าสนามแข่งจากเส้นทางที่แสดงด้วยลูกศร ดังรูป ![illustration](../images/TEDET2560_Math_G8_slice26_crop_0.png) ถ้ารถยนต์คันนี้ต้องแล่นตรงหรือเลี้ยวขวาเท่านั้น แล้วทางออกหมายเลขใดไม่มีเส้นทางที่เป็นไปได้

# คำอธิบายและวิธีทำ

พิจารณาเส้นทางการเคลื่อนที่ของรถยนต์ โดยมีเงื่อนไขว่าสามารถ 'เลี้ยวขวา' หรือ 'ตรงไป' ได้เท่านั้น เราจะไล่ตามเส้นทางจากจุดเข้า (ลูกศร) ดังนี้: 1. จากทางเข้า รถยนต์วิ่งตรงไปข้างหน้าจะถึงทางแยกแรก ถ้าเลี้ยวขวาจะวนกลับ หรือถ้าตรงไปเรื่อยๆ จะเข้าสู่ระบบทางแยกภายใน 2. การวิเคราะห์แต่ละทางออก: - ทางออกที่ 5: สามารถไปถึงได้โดยการเลี้ยวขวาจากทางแยกหลัก - ทางออกที่ 4: สามารถไปถึงได้โดยการตรงไปเรื่อยๆ ตามเส้นทางแนวตั้ง - ทางออกที่ 2: สามารถไปถึงได้โดยการเลี้ยวขวาเมื่อถึงทางแยกภายใน - ทางออกที่ 1: สามารถไปถึงได้โดยการเลี้ยวขวาเมื่อถึงทางแยกด้านบนซ้าย - ทางออกที่ 3: เมื่อพิจารณาจากทางเข้า หากรถแล่นตรงไปจนสุดแล้วเลี้ยวขวา จะพบว่าเส้นทางที่นำไปสู่หมายเลข 3 นั้นอยู่ในตำแหน่งที่ต้องมีการ 'เลี้ยวซ้าย' เพื่อเข้าสู่ช่องทางนั้น (เพราะตำแหน่งของทางออกที่ 3 อยู่ทางด้านขวาของเส้นทางหลัก แต่ช่องทางเข้าถึงถูกกั้นด้วยกำแพงหรือทิศทางที่บังคับให้ต้องเลี้ยวซ้ายถ้ามาจากทิศทางปกติ) สรุปได้ว่าหมายเลข 3 เป็นเส้นทางที่ไม่สามารถไปถึงได้ภายใต้เงื่อนไขเลี้ยวขวาหรือตรงไปเท่านั้น
