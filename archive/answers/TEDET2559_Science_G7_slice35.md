# คำถาม

28. ต่อไปนี้เป็นข้อมูลที่สำรวจสมบัติของสาร โดยการใส่กรดเกลือ กรดอะซิติก สารละลายแอมโมเนียลงในหลอดทดลอง 3 หลอด คือ A ถึง C

![illustration](../images/TEDET2559_Science_G7_slice35_crop_0.png)

ข้อใดเป็นสารที่ถูกต้องที่อยู่ใน A B และ C

① A : กรดเกลือ B : กรดอะซิติก C : สารละลายแอมโมเนีย

② A : กรดเกลือ B : สารละลายแอมโมเนีย C : กรดอะซิติก

③ A : กรดอะซิติก B : กรดเกลือ C : สารละลายแอมโมเนีย

④ A : กรดอะซิติก B : สารละลายแอมโมเนีย C : กรดเกลือ

⑤ A : สารละลายแอมโมเนีย B : กรดเกลือ C : กรดอะซิติก

# คำอธิบายและวิธีทำ

การวิเคราะห์สมบัติของสาร: 1. หลอด A และ B เมื่อสัมผัสกันเกิดควันสีขาว (ปฏิกิริยาระหว่างแก๊ส HCl จากกรดเกลือและแก๊ส NH3 จากแอมโมเนีย ได้ NH4Cl) ดังนั้น A และ B ต้องเป็นกรดเกลือ (HCl) และสารละลายแอมโมเนีย (NH3) 2. สารละลายในหลอด C เมื่อทดสอบด้วย BTB แล้วเปลี่ยนเป็นสีเขียว หมายความว่า C มีสมบัติเป็นกลาง (pH ประมาณ 7) หรือสารผสมที่ส่งผลให้ค่า pH เป็นกลาง ในโจทย์ระบุว่าค่า pH ของ A และ C น้อยกว่า 7 ทั้งหมด แสดงว่า A และ C เป็นกรด (กรดเกลือและกรดอะซิติก) และ B คือสารละลายแอมโมเนีย (เบส) 3. จาก (ข) เมื่อนำสารในหลอด B (เบส) มาผสมกับหลอด C พบว่าสีของ BTB เปลี่ยนเป็นสีเขียว (แสดงว่าได้สารละลายที่เป็นกลาง) ดังนั้น C ต้องเป็นกรดที่จะมาสะเทินกับ B ได้ และจาก (ค) C มี pH น้อยกว่า 7 (เป็นกรด) ถ้า C คือกรดอะซิติก และ B คือแอมโมเนีย การผสมกันจะให้สภาพใกล้กลางมากกว่า และจากข้อ (ก) A ต้องเป็นกรดที่ระเหยได้ง่ายคือ กรดเกลือ เมื่อพิจารณาตัวเลือก พบว่าข้อ (2) A เป็นกรดเกลือ, B เป็นสารละลายแอมโมเนีย, C เป็นกรดอะซิติก สอดคล้องที่สุดเพราะกรดเกลือ (A) ทำปฏิกิริยากับแอมโมเนีย (B) เกิดควันสีขาวได้ และกรดอะซิติก (C) มีค่า pH น้อยกว่า 7
