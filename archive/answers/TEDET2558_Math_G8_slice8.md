# คำถาม

8. รูปหลายเหลี่ยมรูปหนึ่งมีผลบวกของขนาดของมุมภายในทุกมุมเท่ากับ $900^\circ$ จงหาว่ารูปหลายเหลี่ยมนี้มีเส้นทแยงมุมทั้งหมดกี่เส้น

# คำอธิบายและวิธีทำ

ขั้นตอนที่ 1: หาจำนวนด้านของรูปหลายเหลี่ยม โดยใช้สูตรผลรวมของมุมภายในของรูป n-เหลี่ยม คือ (n - 2) × 180° จากโจทย์ กำหนดให้ผลรวมของมุมภายในเท่ากับ 900° จะได้สมการดังนี้: (n - 2) × 180 = 900. ขั้นตอนที่ 2: แก้สมการหาค่า n: n - 2 = 900 / 180; n - 2 = 5; n = 7. ดังนั้น รูปหลายเหลี่ยมนี้คือรูป 7 เหลี่ยม (Heptagon). ขั้นตอนที่ 3: หาจำนวนเส้นทแยงมุมโดยใช้สูตร n(n - 3) / 2 เมื่อ n คือจำนวนด้าน. แทนค่า n = 7 ลงในสูตร: จำนวนเส้นทแยงมุม = 7(7 - 3) / 2 = 7(4) / 2 = 28 / 2 = 14. สรุปได้ว่ารูป 7 เหลี่ยมนี้มีเส้นทแยงมุมทั้งหมด 14 เส้น
