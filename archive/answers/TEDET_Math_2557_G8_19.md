# คำอธิบายและวิธีทำ

เราต้องการหาค่าของ $k$ จากสองความสัมพันธ์ที่กำหนดให้:

1. **หาความสัมพันธ์ระหว่าง $x$ และ $y$ จากสมการเลขยกกำลัง:**
   $$\frac{(5^4)^x(5^y)}{5^{2x}} = 5^7$$
   
   ใช้สมบัติของเลขยกกำลังยุบรวมพจน์ทางฝั่งซ้าย:
   $$\frac{5^{4x} \cdot 5^y}{5^{2x}} = 5^7$$
   $$5^{4x + y - 2x} = 5^7$$
   $$5^{2x + y} = 5^7$$
   
   เนื่องจากฐานเท่ากัน จะได้ว่าเลขชี้กำลังต้องเท่ากัน:
   $$2x + y = 7 \implies y = 7 - 2x \quad \text{--- (สมการที่ 1)}$$

2. **จัดรูปพจน์พีชคณิตที่กำหนดให้:**
   $$E = 4x - 3(x - y) + y + 7$$
   $$E = 4x - 3x + 3y + y + 7$$
   $$E = x + 4y + 7$$

3. **เขียนพจน์ให้อยู่ในรูปของ $x$ เพียงอย่างเดียว:**
   แทนสมการที่ 1 ($y = 7 - 2x$) ลงในพจน์ที่เราจัดรูปไว้:
   $$E = x + 4(7 - 2x) + 7$$
   $$E = x + 28 - 8x + 7$$
   $$E = -7x + 35$$

4. **เทียบสัมประสิทธิ์กับรูปแบบที่โจทย์ระบุ ($-7x + k$):**
   $$-7x + 35 = -7x + k$$
   
   จะได้:
   $$k = 35$$

ดังนั้น ค่าของ $k$ คือ **35**
