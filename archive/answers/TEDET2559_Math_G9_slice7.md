# คำถาม

7. จากแบบในการก่อสร้างสะพานข้ามแม่น้ำ ดังรูป
จุด A, B, C และ D เป็นจุดยอดของรูปสี่เหลี่ยมมุมฉาก
และส่วนของเส้นตรง PQ ขนานกับด้านของรูป
สี่เหลี่ยมมุมฉาก
![illustration](../images/TEDET2559_Math_G9_slice7_crop_0.png)
จงหาค่าของ $x^2$

# คำอธิบายและวิธีทำ

จากโจทย์ กำหนดให้ ABCD เป็นรูปสี่เหลี่ยมมุมฉาก ให้จุด A, B, C, D อยู่ที่พิกัด (0, h), (0, 0), (w, 0), และ (w, h) ตามลำดับ ให้จุด P และ Q มีพิกัดเป็น (x_p, y_p) และ (x_q, y_q) เนื่องจาก PQ ขนานกับด้านของรูปสี่เหลี่ยม จะได้ y_p = y_q กำหนดให้ y_p = y_q = k ความยาวของเส้นเชื่อมจุดต่างๆ คือ AP^2 = x_p^2 + (h-k)^2 = 3^2 = 9, BP^2 = x_p^2 + k^2 = 4^2 = 16, CQ^2 = (w-x_q)^2 + k^2 = 5^2 = 25 และ DQ^2 = (w-x_q)^2 + (h-k)^2 = x^2. จากสมบัติของสี่เหลี่ยมมุมฉากและจุดภายใน ผลรวมของกำลังสองของระยะทางจากจุดภายในไปยังจุดยอดคู่ตรงข้ามที่ทแยงกันจะมีค่าเท่ากันเสมอ (British Flag Theorem) นั่นคือ AP^2 + CQ^2 = BP^2 + DQ^2 แทนค่าที่โจทย์กำหนดให้: 3^2 + 5^2 = 4^2 + x^2 จะได้ 9 + 25 = 16 + x^2 ดังนั้น 34 = 16 + x^2 จะได้ x^2 = 34 - 16 = 18.
