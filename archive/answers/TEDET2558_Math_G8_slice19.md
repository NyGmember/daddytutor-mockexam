# คำถาม

29. จงหาจำนวนเฉพาะที่มีสองหลักซึ่งเมื่อนำมาเติมระหว่าง 8 กับ 2 ในรูปข้างล่าง แล้วจะได้จำนวนที่มีสี่หลักซึ่งหารด้วย 72 ได้ลงตัว 

![illustration](../images/TEDET2558_Math_G8_slice19_crop_0.png)

# คำอธิบายและวิธีทำ

ให้จำนวนเฉพาะที่มีสองหลักคือ $x$ เมื่อนำมาเติมระหว่าง 8 กับ 2 จะได้จำนวนสี่หลักคือ $8x2$ ซึ่งเขียนในรูปเลขฐานสิบได้เป็น $8002 + 10x$ หรือเลข $8d_1 d_2 2$ ที่มีค่าเท่ากับ $8000 + 100d_1 + 10d_2 + 2 = 8002 + 10(10d_1 + d_2) = 8002 + 10x$ โดยที่ $10 \le x \le 99$ และ $x$ เป็นจำนวนเฉพาะ โจทย์กำหนดให้จำนวนนี้หารด้วย 72 ลงตัว ดังนั้น $8002 + 10x$ ต้องหารด้วย $72$ ลงตัว ซึ่งหมายความว่าต้องหารด้วย $8$ และ $9$ ลงตัว

1. พิจารณาการหารด้วย 8: $8002 + 10x = 8000 + 2 + 8x + 2x = 8(1000 + x) + 2 + 2x$. ดังนั้น $2 + 2x$ ต้องหารด้วย $8$ ลงตัว นั่นคือ $2(1+x) = 8k$ หรือ $1+x = 4k$ ซึ่งหมายความว่า $x$ ต้องอยู่ในรูป $4k-1$ เช่น $11, 19, 23, 31, 43, 47, 59, 67, 71, 79, 83, 91 (ไม่ใช่), 95 (ไม่ใช่).

2. พิจารณาการหารด้วย 9: ผลบวกเลขโดดของ $8d_1 d_2 2$ ต้องหารด้วย $9$ ลงตัว นั่นคือ $8 + d_1 + d_2 + 2 = 10 + (d_1 + d_2) = 10 + x$ (สังเกตว่าผลบวกเลขโดดของ $x$ กับเลขโดดของ $x$ ตัวเดิมนั้นเท่ากันสำหรับจำนวนสองหลัก $x = 10d_1 + d_2$ คือ $d_1+d_2$) ซึ่งจะได้ $10 + (d_1+d_2) = 10 + (	ext{ผลบวกเลขโดดของ } x) = 9n$ ดังนั้น $1 + (	ext{ผลบวกเลขโดดของ } x)$ ต้องหารด้วย $9$ ลงตัว

ทดสอบค่า $x$ ที่เป็นจำนวนเฉพาะและอยู่ในรูป $4k-1$:
- $x=11: 1+1+1=3$ (หารด้วย 9 ไม่ลงตัว)
- $x=19: 1+1+9=11$ (ไม่ลงตัว)
- $x=23: 1+2+3=6$ (ไม่ลงตัว)
- $x=31: 1+3+1=5$ (ไม่ลงตัว)
- $x=43: 1+4+3=8$ (ไม่ลงตัว)
- $x=47: 1+4+7=12$ (ไม่ลงตัว)
- $x=59: 1+5+9=15$ (ไม่ลงตัว)
- $x=67: 1+6+7=14$ (ไม่ลงตัว)
- $x=71: 1+7+1=9$ (ลงตัว!)
ตรวจสอบ $8712$: $8712 \div 72 = 121$ เป็นจำนวนเต็ม
ดังนั้นจำนวนเฉพาะนั้นคือ 71
