# คำถาม

27. บันไดเลื่อนกำลังเลื่อนขึ้นจากชั้นล่างไปชั้นบน ชายคนหนึ่งเดินขึ้นบันไดเลื่อนด้วยอัตราเร็วคงตัวจนถึงชั้นบนภายใน 20 วินาที แต่ถ้าชายคนนี้เดินลงบันไดเลื่อนด้วยอัตราเร็วคงตัวเดิมจะถึงชั้นล่างภายใน 60 วินาที

จงหาว่าถ้าชายคนนี้ขึ้นบันไดเลื่อนโดยไม่เดิน เขาจะถึงชั้นบนในเวลากี่วินาที

# คำอธิบายและวิธีทำ

กำหนดให้: \n- $v_m$ คืออัตราเร็วของการเดินของชายคนนี้ \n- $v_e$ คืออัตราเร็วของบันไดเลื่อน \n- $S$ คือระยะทางจากชั้นล่างถึงชั้นบน \n\nจากโจทย์:\n1. เมื่อเดินขึ้นบันไดเลื่อน (เดินไปในทิศทางเดียวกับบันได): $v_m + v_e = \frac{S}{20}$ (สมการที่ 1)\n2. เมื่อเดินลงบันไดเลื่อน (เดินทวนทิศทางบันได): $v_m - v_e = \frac{S}{60}$ (สมการที่ 2)\n\nเราต้องการหาเวลาที่ชายคนนี้ยืนเฉยๆ บนบันไดเลื่อน ซึ่งก็คือการหาค่า $t$ เมื่อ $v_e = \frac{S}{t}$ \n\nนำ (สมการที่ 1) ลบ (สมการที่ 2): \n$(v_m + v_e) - (v_m - v_e) = \frac{S}{20} - \frac{S}{60}$ \n$2v_e = \frac{3S}{60} - \frac{S}{60}$ \n$2v_e = \frac{2S}{60}$ \n$v_e = \frac{S}{60}$ \n\nจาก $v_e = \frac{S}{t}$, ดังนั้นจะได้ $\frac{S}{60} = \frac{S}{t}$ ซึ่งหมายความว่า $t = 60$ วินาที
