# คำถาม

2. กำหนดให้ $b - a = 3$ และ $a + c = -4$ 

จงหาค่าของ $\sqrt{(b + c)^2}$

# คำอธิบายและวิธีทำ

จากโจทย์ที่กำหนดให้เรามีสมการสองสมการคือ: 1) $b - a = 3$ และ 2) $a + c = -4$ เป้าหมายของเราคือการหาค่าของ $\sqrt{(b + c)^2}$ ซึ่งจากสมบัติของรากที่สอง $\sqrt{x^2} = |x|$ ดังนั้นค่าของ $\sqrt{(b + c)^2}$ จะเท่ากับ $|b + c|$ ก่อนอื่นเราลองนำสมการทั้งสองมาบวกกันเพื่อกำจัดตัวแปร $a$: $(b - a) + (a + c) = 3 + (-4)$ ซึ่งจะได้เป็น $b + c = -1$ จากนั้นนำค่าที่ได้ไปแทนในนิพจน์ที่โจทย์ต้องการ: $\sqrt{(b + c)^2} = |b + c| = |-1|$ เนื่องจากค่าสัมบูรณ์ของ $-1$ คือ $1$ ดังนั้นผลลัพธ์ของสมการนี้คือ $1$
