# คำถาม

19. ภาพแสดงการเคลื่อนที่ของลูกบอลที่โยนขึ้นไปในแนวเฉียง (ไม่ต้องคำนึงถึงแรงต้านทานของอากาศ)

![illustration](../images/TEDET2558_Science_G9_slice19_crop_0.png)

ข้อใดอธิบายไม่ถูกต้อง
① พลังงานกลของลูกบอลคงที่เสมอ
② จุดที่พลังงานจลน์มากที่สุดคือ A, E
③ ความเร็วของลูกบอลที่ C เท่ากับ 0 และมีพลังงานศักย์มากที่สุด
④ พลังงานจลน์ที่ B และ D เท่ากัน
⑤ ถ้าพลังงานจลน์ที่ A เท่ากับ 100 J พลังงานศักย์ที่ C เท่ากับ 100 J

# คำอธิบายและวิธีทำ

การเคลื่อนที่แบบโพรเจกไทล์ (Projectile motion) ในสภาวะที่ไม่มีแรงต้านทานอากาศ พิจารณาพลังงานดังนี้: 1) พลังงานกล (E = Ek + Ep) ของลูกบอลจะมีค่าคงที่ตลอดการเคลื่อนที่ เนื่องจากไม่มีแรงภายนอกมาทำงาน (ข้อ 1 จึงถูกต้อง) 2) ที่จุด A และ E เป็นจุดที่ลูกบอลอยู่ระดับต่ำสุด (ใกล้พื้นที่สุด) พลังงานศักย์โน้มถ่วง (Ep = mgh) จึงน้อยที่สุด ทำให้พลังงานจลน์ (Ek) มีค่ามากที่สุดตามกฎการอนุรักษ์พลังงาน (ข้อ 2 จึงถูกต้อง) 3) ที่จุด C (จุดสูงสุด) ความเร็วในแนวราบ (ux) ยังคงอยู่ แต่ความเร็วในแนวตั้ง (vy) เป็นศูนย์ ดังนั้นความเร็วรวมที่จุด C ไม่เท่ากับ 0 (ข้อ 3 จึงไม่ถูกต้อง) 4) ที่จุด B และ D มีความสูง h = h' เท่ากัน พลังงานศักย์จึงเท่ากัน ส่งผลให้พลังงานจลน์ที่เหลืออยู่ต้องเท่ากันด้วย (ข้อ 4 จึงถูกต้อง) 5) ตามกฎการอนุรักษ์พลังงาน พลังงานรวมที่จุด A เท่ากับพลังงานรวมที่จุด C ถ้าพลังงานจลน์ที่ A คือ 100 J (สมมติพลังงานศักย์ที่ A=0) พลังงานรวมคือ 100 J เมื่อขึ้นไปที่จุด C พลังงานจลน์ที่จุด C จะลดลงแต่พลังงานศักย์จะเพิ่มขึ้น หากพิจารณาว่าพลังงานรวมคือ 100 J และที่จุด C ลูกบอลยังมีความเร็วในแนวราบอยู่ พลังงานศักย์ที่ C จะต้องน้อยกว่า 100 J ดังนั้นข้อที่กล่าวไม่ถูกต้องคือข้อ 3
