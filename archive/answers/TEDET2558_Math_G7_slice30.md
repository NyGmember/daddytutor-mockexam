# คำถาม

30. พิจารณาลำดับ $(1,1), (1,2), (2,1), (1,3), (2,2), (3,1), \dots$ ซึ่งแสดงรูปแบบของการไล่ลำดับได้ด้วยแผนภาพดังนี้ ![illustration](../images/TEDET2558_Math_G7_slice30_crop_0.png) ถ้า $(a, b)$ เป็นตัวที่ 500 ของลำดับนี้ จงหาค่าของ $a \times b$

# คำอธิบายและวิธีทำ

จากแผนภาพ เราสามารถแบ่งลำดับออกเป็นกลุ่มตามแนวทแยงมุมได้ โดยกลุ่มที่ $n$ จะมีผลรวมของพิกัดคือ $a + b = n + 1$

- กลุ่มที่ 1: (1, 1) -> $a+b = 2$, มี 1 จำนวน
- กลุ่มที่ 2: (1, 2), (2, 1) -> $a+b = 3$, มี 2 จำนวน
- กลุ่มที่ 3: (1, 3), (2, 2), (3, 1) -> $a+b = 4$, มี 3 จำนวน

จำนวนสมาชิกทั้งหมดเมื่อสิ้นสุดกลุ่มที่ $n$ คือผลรวมของจำนวนสมาชิกในกลุ่มที่ 1 ถึง $n$ ซึ่งคือ $S_n = 1 + 2 + 3 + \dots + n = \frac{n(n+1)}{2}$

เราต้องการหาลำดับที่ 500 จึงต้องหาค่า $n$ ที่ทำให้ $S_n$ ใกล้เคียง 500 มากที่สุดแต่ไม่เกิน 500:
$\frac{n(n+1)}{2} \approx 500 \implies n^2 \approx 1000 \implies n \approx 31.6$
ลองแทนค่า $n=31$: $S_{31} = \frac{31 \times 32}{2} = 31 \times 16 = 496$
แสดงว่าลำดับที่ 496 คือพิกัดสุดท้ายของกลุ่มที่ 31 ซึ่งคือ $(31, 1)$ เพราะกลุ่มที่ $n$ จะไล่จาก $(1, n)$ ไปจนถึง $(n, 1)$

ลำดับที่ 497 ถึง 500 จะอยู่ในกลุ่มที่ 32 ($a+b = 33$):
- ลำดับที่ 497: (1, 32)
- ลำดับที่ 498: (2, 31)
- ลำดับที่ 499: (3, 30)
- ลำดับที่ 500: (4, 29)

ดังนั้น $a=4$ และ $b=29$ ค่าของ $a \times b = 4 \times 29 = 116$
