# คำถาม

16. จงหาผลลัพธ์ของ $$\frac{12350^2 - (12345 \times 12348) - 53}{(12345 \times 12350) - 12347^2}$$

# คำอธิบายและวิธีทำ

โจทย์ข้อนี้ต้องการให้เราหาผลลัพธ์ของนิพจน์เศษส่วนที่ประกอบด้วยตัวเลขจำนวนมาก ซึ่งหากคำนวณโดยตรงจะซับซ้อนและใช้เวลานาน วิธีที่มีประสิทธิภาพคือการใช้หลักการทางพีชคณิตเข้ามาช่วยในการจัดรูปสมการให้ง่ายขึ้น

ขั้นตอนที่ 1: กำหนดตัวแปรเพื่อลดความซับซ้อนของตัวเลข
สังเกตว่าตัวเลขในนิพจน์มีความสัมพันธ์กัน เราสามารถกำหนดให้ตัวเลขตัวหนึ่งเป็นตัวแปร $x$ เพื่อให้ตัวเลขอื่น ๆ สามารถเขียนในรูปของ $x$ ได้ง่ายขึ้น
ให้ $x = 12347$
ดังนั้น ตัวเลขอื่น ๆ สามารถเขียนได้ดังนี้:
$12350 = 12347 + 3 = x + 3$
$12345 = 12347 - 2 = x - 2$
$12348 = 12347 + 1 = x + 1$

ขั้นตอนที่ 2: แทนค่าตัวแปรลงในตัวเศษ (Numerator)
ตัวเศษของนิพจน์คือ $12350^2 - (12345 	imes 12348) - 53$
แทนค่าด้วยตัวแปร $x$ จะได้:
$(x+3)^2 - ((x-2)(x+1)) - 53$
ทำการกระจายพหุนามและรวมพจน์:
$= (x^2 + 2(x)(3) + 3^2) - (x^2 + x - 2x - 2) - 53$
$= (x^2 + 6x + 9) - (x^2 - x - 2) - 53$
เปิดวงเล็บและเปลี่ยนเครื่องหมายสำหรับพจน์ในวงเล็บหลัง:
$= x^2 + 6x + 9 - x^2 + x + 2 - 53$
รวมพจน์ที่มี $x^2$, พจน์ที่มี $x$ และพจน์ที่เป็นค่าคงที่:
$= (x^2 - x^2) + (6x + x) + (9 + 2 - 53)$
$= 0 + 7x + 11 - 53$
$= 7x - 42$

ขั้นตอนที่ 3: แทนค่าตัวแปรลงในตัวส่วน (Denominator)
ตัวส่วนของนิพจน์คือ $(12345 	imes 12350) - 12347^2$
แทนค่าด้วยตัวแปร $x$ จะได้:
$((x-2)(x+3)) - x^2$
ทำการกระจายพหุนามและรวมพจน์:
$= (x^2 + 3x - 2x - 6) - x^2$
$= (x^2 + x - 6) - x^2$
รวมพจน์ที่มี $x^2$, พจน์ที่มี $x$ และพจน์ที่เป็นค่าคงที่:
$= (x^2 - x^2) + x - 6$
$= 0 + x - 6$
$= x - 6

ขั้นตอนที่ 4: รวมตัวเศษและตัวส่วนที่จัดรูปแล้ว
เมื่อนำตัวเศษและตัวส่วนที่จัดรูปแล้วมารวมกัน จะได้นิพจน์:
$$ \frac{7x - 42}{x - 6} $$

ขั้นตอนที่ 5: ทำให้เป็นรูปอย่างง่าย
สังเกตว่าตัวเศษสามารถดึงตัวประกอบร่วม 7 ออกมาได้:
$7x - 42 = 7(x - 6)$
ดังนั้น นิพจน์จะกลายเป็น:
$$ \frac{7(x - 6)}{x - 6} $$
เนื่องจาก $x = 12347$ ซึ่งทำให้ $x-6 = 12341$ ไม่เท่ากับศูนย์ เราจึงสามารถตัดพจน์ $(x-6)$ ทั้งในตัวเศษและตัวส่วนได้:
$= 7$

ผลลัพธ์สุดท้ายของนิพจน์คือ 7
