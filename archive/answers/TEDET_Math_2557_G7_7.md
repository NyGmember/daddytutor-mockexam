# คำอธิบายและวิธีทำ
จากโจทย์ กำหนดให้ $a$ และ $b$ เป็นจุดใด ๆ บนเส้นจำนวน

1. วิเคราะห์ค่าของ $a$:
มีค่าสัมบูรณ์เท่ากับ $5$ นั่นคือ $|a| = 5$
จะได้ว่า $a$ สามารถเป็นไปได้ 2 ค่า คือ $a = 5$ หรือ $a = -5$

2. วิเคราะห์ค่าของ $b$:
มีค่าสัมบูรณ์เท่ากับ $2$ นั่นคือ $|b| = 2$
จะได้ว่า $b$ สามารถเป็นไปได้ 2 ค่า คือ $b = 2$ หรือ $b = -2$

3. หาระยะห่างระหว่าง $a$ กับ $b$ ซึ่งเท่ากับ $|a - b|$ ในแต่ละกรณี:
- กรณีที่ 1: $a = 5, b = 2$ จะได้ระยะห่างเท่ากับ $|5 - 2| = 3$
- กรณีที่ 2: $a = 5, b = -2$ จะได้ระยะห่างเท่ากับ $|5 - (-2)| = |5 + 2| = 7$
- กรณีที่ 3: $a = -5, b = 2$ จะได้ระยะห่างเท่ากับ $|-5 - 2| = |-7| = 7$
- กรณีที่ 4: $a = -5, b = -2$ จะได้ระยะห่างเท่ากับ $|-5 - (-2)| = |-5 + 2| = |-3| = 3$

จากกรณีทั้งหมด จะพบว่า:
- ระยะห่างที่มากที่สุดคือ $M = 7$
- ระยะห่างที่น้อยที่สุดคือ $m = 3$

ดังนั้น $M - m = 7 - 3 = 4$
