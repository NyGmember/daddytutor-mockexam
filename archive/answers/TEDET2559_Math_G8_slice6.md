# คำถาม

6. มีรูปสามเหลี่ยม วงกลม และรูปสี่เหลี่ยมซ้อนกัน ดังรูป ![illustration](../images/TEDET2559_Math_G8_slice6_crop_0.png) ถ้าพื้นที่ส่วนที่ทับซ้อนของทั้งสามรูปเท่ากับพื้นที่ ในแต่ละข้อต่อไปนี้ 
- $\frac{1}{3}$ ของพื้นที่รูปสามเหลี่ยม 
- $\frac{2}{7}$ ของพื้นที่วงกลม 
- $\frac{3}{5}$ ของพื้นที่รูปสี่เหลี่ยม 
ถ้าอัตราส่วนของพื้นที่ของรูปสามเหลี่ยม ต่อ วงกลม ต่อ รูปสี่เหลี่ยม เป็น $a:b:c$ จงหาค่าของ $a+b+c$

# คำอธิบายและวิธีทำ

ให้ $S$ แทนพื้นที่ของรูปสามเหลี่ยม, $C$ แทนพื้นที่ของวงกลม และ $R$ แทนพื้นที่ของรูปสี่เหลี่ยม และให้ $x$ แทนพื้นที่ของส่วนที่ทับซ้อนกันของทั้งสามรูป ตามโจทย์กำหนดพื้นที่ส่วนที่ทับซ้อนเท่ากับพื้นที่แต่ละส่วนดังนี้: 1) $x = \frac{1}{3}S$ ดังนั้น $S = 3x$ 2) $x = \frac{2}{7}C$ ดังนั้น $C = \frac{7}{2}x$ 3) $x = \frac{3}{5}R$ ดังนั้น $R = \frac{5}{3}x$. อัตราส่วนพื้นที่ของสามเหลี่ยม ต่อ วงกลม ต่อ สี่เหลี่ยม คือ $S : C : R = 3x : \frac{7}{2}x : \frac{5}{3}x$. เพื่อให้อัตราส่วนเป็นจำนวนเต็ม เรานำ ค.ร.น. ของส่วน (คือ 6) มาคูณตลอดทั้งอัตราส่วน: $3(6) : \frac{7}{2}(6) : \frac{5}{3}(6) = 18 : 21 : 10$. ดังนั้น $a = 18$, $b = 21$, และ $c = 10$. ค่าของ $a + b + c = 18 + 21 + 10 = 49$.
