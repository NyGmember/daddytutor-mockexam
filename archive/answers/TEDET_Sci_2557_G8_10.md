# คำถาม

10. จากกราฟ แสดงความสัมพันธ์ระหว่างมวลของแมกนีเซียมออกไซด์กับมวลของแมกนีเซียมที่ทำปฏิกิริยาแล้วเกิดแมกนีเซียมออกไซด์ โดยแมกนีเซียมทำปฏิกิริยากับออกซิเจน

[![illustration](../images/TEDET_Sci_2557_G8_10_crop_0.png)]

พิจารณาข้อความต่อไปนี้

> * **ก.** แมกนีเซียมและออกซิเจนทำปฏิกิริยา ในอัตราส่วนมวล 3 : 5
> * **ข.** เมื่อแมกนีเซียมและออกซิเจนทำปฏิกิริยากัน จะเป็นไปตามกฎทรงมวล
> * **ค.** ถ้าแมกนีเซียม 0.9 g ทำปฏิกิริยากับออกซิเจน จะเกิดแมกนีเซียมออกไซด์ 1.5 g
> * **ง.** ในการทำปฏิกิริยาระหว่างแมกนีเซียม 1.5 g ให้สมบูรณ์ จำเป็นต้องใช้มวลของออกซิเจน 2.5 g

ข้อใดอธิบายเกี่ยวกับกราฟข้างต้นได้ถูกต้อง

① ก, ข  
② ก, ค  
③ ข, ค  
④ ข, ง  
⑤ ค, ง

# คำอธิบายและวิธีทำ

วิเคราะห์ข้อมูลจากกราฟแสดงความสัมพันธ์ระหว่างมวลของแมกนีเซียมและมวลของแมกนีเซียมออกไซด์ที่เกิดขึ้น:
- เมื่อใช้ **แมกนีเซียม ($\text{Mg}$)** มวล $3\text{ g}$ จะเกิด **แมกนีเซียมออกไซด์ ($\text{MgO}$)** มวล $5\text{ g}$
- เมื่อใช้ **แมกนีเซียม ($\text{Mg}$)** มวล $6\text{ g}$ จะเกิด **แมกนีเซียมออกไซด์ ($\text{MgO}$)** มวล $10\text{ g}$
ดังนั้น อัตราส่วนโดยมวลของปฏิกิริยาคือ:
$$\text{Mg} : \text{MgO} = 3 : 5$$

หาปริมาณของออกซิเจน ($\text{O}$) ที่เข้ามาทำปฏิกิริยาโดยใช้กฎทรงมวล:
$$\text{มวลของ Mg} + \text{มวลของ O} = \text{มวลของ MgO}$$
$$3\text{ g} + \text{มวลของ O} = 5\text{ g} \rightarrow \text{มวลของ O} = 2\text{ g}$$
อัตราส่วนโดยมวลของสารที่ทำปฏิกิริยากันพอดีคือ:
$$\text{Mg} : \text{O} = 3 : 2$$

พิจารณาข้อความต่าง ๆ:
* **ก. ไม่ถูกต้อง:** แมกนีเซียมและออกซิเจนทำปฏิกิริยาในอัตราส่วนโดยมวลเท่ากับ $3 : 2$ (อัตราส่วน $3 : 5$ คือ แมกนีเซียมต่อแมกนีเซียมออกไซด์)
* **ข. ถูกต้อง:** การเกิดปฏิกิริยาเคมีระหว่างแมกนีเซียมและออกซิเจนในระบบปิดจะเป็นไปตามกฎทรงมวล (มวลรวมก่อนและหลังทำปฏิกิริยามีค่าคงที่)
* **ค. ถูกต้อง:** 
  - จากอัตราส่วนโดยมวล $\text{Mg} : \text{MgO} = 3 : 5$
  - ถ้าใช้แมกนีเซียม $0.9\text{ g}$ จะเกิดแมกนีเซียมออกไซด์เท่ากับ:
    $$0.9\text{ g} \times \frac{5}{3} = 1.5\text{ g}$$
* **ง. ไม่ถูกต้อง:** 
  - จากอัตราส่วนโดยมวล $\text{Mg} : \text{O} = 3 : 2$
  - ถ้าต้องการทำปฏิกิริยากับแมกนีเซียม $1.5\text{ g}$ ให้สมบูรณ์ ต้องใช้มวลของออกซิเจนเท่ากับ:
    $$1.5\text{ g} \times \frac{2}{3} = 1.0\text{ g}$$ (ไม่ใช่ $2.5\text{ g}$)

ดังนั้น ข้อความที่ถูกต้องคือ **ข และ ค**

ซึ่งสอดคล้องกับตัวเลือก **3**
