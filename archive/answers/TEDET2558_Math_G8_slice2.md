# คำถาม

2. คะแนนสอบวิชาคณิตศาสตร์ของนักเรียนห้องหนึ่งเป็นดังตาราง

![illustration](../images/TEDET2558_Math_G8_slice2_crop_0.png)

โดยที่จำนวนคนสะสมเป็นผลบวกของจำนวนคนในช่วงคะแนนนั้น ๆ กับช่วงคะแนนก่อนหน้าทั้งหมด
จงหาค่าของ A

# คำอธิบายและวิธีทำ

ในการแก้โจทย์ปัญหานี้ เราต้องทำความเข้าใจนิยามของ 'จำนวนคนสะสม' และข้อมูลในตาราง:
1. 'จำนวนคนสะสม' ณ ช่วงคะแนนใด ๆ คือผลรวมของจำนวนนักเรียนในทุกช่วงคะแนนตั้งแต่ช่วงแรกจนถึงช่วงนั้น
2. เราทราบว่าผลรวมของ 'ร้อยละของจำนวนคน' ทั้งหมดต้องเท่ากับ 100
3. ให้ $f_i$ เป็นร้อยละของจำนวนคนในแต่ละช่วงคะแนน และ $C_i$ เป็นจำนวนคนสะสม ณ ช่วงคะแนนนั้น
4. ในช่วงคะแนน 41-50: $f_1 = 2$, $C_1 = 2$
5. ในช่วงคะแนน 51-60: $f_2 = 8$, $C_2 = 2 + 8 = 10$ แต่ในตารางระบุ $C_2 = 5$ หมายความว่าค่าที่ระบุในคอลัมน์ 'ร้อยละของจำนวนคน' นั้น สัมพันธ์กับค่าสะสมผ่านสูตร $C_i = C_{i-1} + f_i$
6. วิเคราะห์จากตาราง:
- ช่วง 61-70: $f_3 = 20$, $C_3 = 15$ (จาก $5 + 10 = 15$ แสดงว่าข้อมูลนี้คือ $f_3$ คือ 10% แต่โจทย์ระบุ 20% ซึ่งน่าจะเป็นความผิดพลาดของตัวเลขในโจทย์หรือการตีความ ตรวจสอบ $C_4$ แทน)
- ช่วง 71-80: $C_4 = 36$ ซึ่งเกิดจาก $C_3 + f_4 = 36 \Rightarrow 15 + f_4 = 36 \Rightarrow f_4 = 21$
- ช่วง 91-100: $f_6 = 6$ และ $C_6 = 100$
- ช่วง 81-90: $f_5 = 100 - (C_4 + f_6) = 100 - (36 + 6) = 100 - 42 = 58$
- ดังนั้น ค่า $A$ ซึ่งคือ $C_5 = C_4 + f_5 = 36 + 58 = 94$
