# คำถาม

18. จากภาพแสดงบริเวณภายในของถ้ำหินปูนแห่งหนึ่ง ข้อใดอธิบายไม่ถูกต้อง

[![illustration](../images/TEDET_Sci_2557_G8_18_crop_0.png)]

① A, B, C เกิดขึ้นที่บริเวณที่มีหินปูน  
② A, B, C เกิดขึ้นหลังจากมีถ้ำหินปูน  
③ A คือหินย้อย B คือเสาหิน และ C คือหินงอก  
④ ถ้า B ถูกน้ำใต้ดินกัดเซาะจนมีขนาดเล็กลงแล้วจะเกิดเป็น A และ C  
⑤ A, B, C มีลักษณะพื้นผิวที่เกิดจากการตกตะกอนและถ้ำหินปูนมีลักษณะพื้นผิวที่เกิดจากการกัดกร่อน

# คำอธิบายและวิธีทำ

วิเคราะห์โครงสร้างภายในถ้ำหินปูนและกระบวนการเกิด:

1. **ลักษณะทางกายภาพของโครงสร้างต่าง ๆ:**
   - **A คือ หินย้อย (Stalactite):** เกิดจากน้ำใต้ดินที่มีสารละลายแคลเซียมไฮโดรเจนคาร์บอเนต หยดลงมาจากเพดานถ้ำ เมื่อน้ำระเหยไปจะเกิดการตกตะกอนของสารแคลเซียมคาร์บอเนตสะสมพอกพูนย้อยลงมาเป็นแท่งหิน
   - **C คือ หินงอก (Stalagmit):** เกิดจากน้ำหยดจากเพดานถ้ำตกลงสู่พื้นถ้ำ แล้วสารละลายแคลเซียมคาร์บอเนตตกตะกอนสะสมพอกพูนสูงขึ้นจากพื้นถ้ำ
   - **B คือ เสาหิน (Stone Pillar / Column):** เกิดจากการที่หินย้อย (A) และหินงอก (C) เติบโตขยายยาวจนกระทั่งมาบรรจบเชื่อมต่อเป็นต้นเดียวกันเป็นเสาค้ำยันถ้ำ

2. **พิจารณาตัวเลือก:**
   - **ตัวเลือก ① และ ② ถูกต้อง:** โครงสร้างเหล่านี้เกิดในพื้นที่หินปูนและเกิดขึ้นภายหลังจากที่กรดคาร์บอนิกละลายหินปูนสร้างโพรงถ้ำขึ้นมาก่อนแล้ว
   - **ตัวเลือก ③ ถูกต้อง:** เป็นการเรียกชื่อตำแหน่งที่ถูกต้องตามหลักธรณีวิทยา
   - **ตัวเลือก ④ ไม่ถูกต้อง และเป็นคำตอบของข้อนี้:** หินย้อย (A) และหินงอก (C) ไม่ได้เกิดมาจากการที่เสาหิน (B) ถูกกัดเซาะให้เล็กลงจนแยกออกจากกัน แต่ในทางตรงกันข้าม เสาหิน (B) ต่างหากที่เกิดจากการเชื่อมต่อกันระหว่างหินย้อย (A) และหินงอก (C) ที่ยาวมาพบกัน
   - **ตัวเลือก ⑤ ถูกต้อง:** ตัวถ้ำเกิดจากกระบวนการกัดเซาะ/ละลายของกรด (Erosion/Dissolution) ส่วนหินงอกหินย้อยเกิดจากการตกตะกอนทางเคมี (Precipitation)

ดังนั้น ข้อความที่อธิบายไม่ถูกต้องคือ **ถ้า B ถูกน้ำใต้ดินกัดเซาะจนมีขนาดเล็กลงแล้วจะเกิดเป็น A และ C**

ซึ่งสอดคล้องกับตัวเลือก **4**
