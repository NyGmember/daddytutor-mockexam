# คำอธิบายและวิธีทำ

ขั้นตอนการวิเคราะห์การเหนี่ยวนำไฟฟ้าสถิต:

1. วิเคราะห์ข้อ (ก): เมื่อนำแท่งอีโบไนต์ที่มีประจุลบมาสัมผัสกับทรงกลม A ที่เป็นกลางทางไฟฟ้า จะเกิดการถ่ายโอนอิเล็กตรอนจากแท่งอีโบไนต์ไปยังทรงกลม A ทำให้ทรงกลม A มีประจุลบสุทธิ ดังนั้นข้อ ก. จึงถูกต้อง

2. วิเคราะห์ข้อ (ข): เมื่อนำทรงกลม A (มีประจุลบ) เข้าใกล้ทรงกลม B (ที่แตะกับ C) ประจุลบจาก A จะผลักอิเล็กตรอนอิสระในระบบ B-C ให้เคลื่อนที่หนีไปให้ไกลที่สุด คือย้ายจาก B ไปยัง C ดังนั้น อิเล็กตรอนไม่ได้เคลื่อนจาก C ไป B แต่เคลื่อนจาก B ไป C (เพื่อไปสะสมที่ C) ดังนั้นข้อ ข. จึงผิด

3. วิเคราะห์ข้อ (ค): เมื่อแยกทรงกลม B และ C ออกจากกันขณะที่ A ยังอยู่ใกล้ ทรงกลม B จะเหลือประจุบวก (เพราะเสียอิเล็กตรอนไป C) ส่วนทรงกลม C จะมีประจุลบ (ได้รับอิเล็กตรอนมา) เมื่อแยกออกมาแล้ว ทรงกลม C จะมีประจุลบและทรงกลม A ก็มีประจุลบ ดังนั้นประจุของ A และ C จึงเหมือนกันคือเป็นประจุลบ ข้อ ค. จึงถูกต้อง

สรุป: ข้อที่ถูกต้องคือ ก. และ ค.
