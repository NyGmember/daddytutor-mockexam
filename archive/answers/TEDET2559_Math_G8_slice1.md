# คำถาม

1. กำหนดให้ $A \odot B = (A - B) \div B$ จงหาผลลัพธ์ของการคำนวณต่อไปนี้

$$(6.75 \odot (-\frac{3}{4})) \odot (-0.25)$$

[FIGURE_0]

# คำอธิบายและวิธีทำ

ขั้นตอนที่ 1: กำหนดการดำเนินการ A ⊙ B = (A - B) ÷ B ซึ่งหมายถึง (A - B) / B หรือ (A/B) - 1. เราจะทำการคำนวณในวงเล็บก่อน คือ (6.75 ⊙ (-3/4)). แปลง 6.75 เป็นเศษส่วนคือ 27/4 และ -3/4 คือ -0.75 ดังนั้น (6.75 ⊙ (-0.75)) = (6.75 - (-0.75)) / (-0.75) = (6.75 + 0.75) / (-0.75) = 7.5 / (-0.75) = -10. ขั้นตอนที่ 2: นำผลลัพธ์ที่ได้ไปคำนวณต่อกับ (-0.25) นั่นคือ (-10) ⊙ (-0.25). แทนค่าลงในสูตร: ((-10) - (-0.25)) / (-0.25) = (-10 + 0.25) / (-0.25) = (-9.75) / (-0.25). ในการหารจำนวนลบด้วยจำนวนลบ ผลลัพธ์จะเป็นบวก: 9.75 / 0.25 = 975 / 25 = 39.
