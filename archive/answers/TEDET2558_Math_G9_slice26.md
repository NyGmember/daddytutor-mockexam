# คำถาม

17. ถ้า 7 เป็นสัมประสิทธิ์ของ $x$ ในการกระจายของ
$(ax+3)(bx-4)$ และ 5 เป็นสัมประสิทธิ์ของ $x$
ในการกระจายของ $(x+a)(2x-b)$
จงหาค่าของ $a+b$

# คำอธิบายและวิธีทำ

โจทย์กำหนดเงื่อนไขเกี่ยวกับสัมประสิทธิ์ของ $x$ จากการกระจายพหุนามสองชุด เราจะต้องทำการกระจายพหุนามแต่ละชุดเพื่อหาสัมประสิทธิ์ของ $x$ แล้วนำมาสร้างเป็นระบบสมการเพื่อหาค่า $a$ และ $b$ จากนั้นจึงหาผลรวม $a+b$

**ขั้นที่ 1: พิจารณาพหุนามแรก $(ax+3)(bx-4)$**
ทำการกระจายพหุนามนี้:
$(ax+3)(bx-4) = (ax)(bx) + (ax)(-4) + (3)(bx) + (3)(-4)$
$= abx^2 - 4ax + 3bx - 12$
จัดรูปพจน์ที่มี $x$:
$= abx^2 + (-4a + 3b)x - 12$
สัมประสิทธิ์ของ $x$ ในการกระจายนี้คือ $(-4a + 3b)$
โจทย์กำหนดให้สัมประสิทธิ์ของ $x$ คือ 7 ดังนั้นเราจะได้สมการที่ 1:
$-4a + 3b = 7$ (สมการที่ 1)

**ขั้นที่ 2: พิจารณาพหุนามที่สอง $(x+a)(2x-b)$**
ทำการกระจายพหุนามนี้:
$(x+a)(2x-b) = (x)(2x) + (x)(-b) + (a)(2x) + (a)(-b)$
$= 2x^2 - bx + 2ax - ab$
จัดรูปพจน์ที่มี $x$:
$= 2x^2 + (2a - b)x - ab$
สัมประสิทธิ์ของ $x$ ในการกระจายนี้คือ $(2a - b)$
โจทย์กำหนดให้สัมประสิทธิ์ของ $x$ คือ 5 ดังนั้นเราจะได้สมการที่ 2:
$2a - b = 5$ (สมการที่ 2)

**ขั้นที่ 3: แก้ระบบสมการเชิงเส้น**
เรามีระบบสมการ:
1) $-4a + 3b = 7$
2) $2a - b = 5$

จากสมการที่ 2 เราสามารถจัดรูปเพื่อแสดง $b$ ในรูปของ $a$ ได้ดังนี้:
$b = 2a - 5$

นำค่า $b$ นี้ไปแทนในสมการที่ 1:
$-4a + 3(2a - 5) = 7$
$-4a + 6a - 15 = 7$
$2a - 15 = 7$
$2a = 7 + 15$
$2a = 22$
$a = 11$

เมื่อได้ค่า $a=11$ แล้ว นำกลับไปแทนในสมการ $b = 2a - 5$ เพื่อหาค่า $b$:
$b = 2(11) - 5$
$b = 22 - 5$
$b = 17$

**ขั้นที่ 4: หาค่าของ $a+b$**
$a + b = 11 + 17$
$a + b = 28$

ดังนั้น ค่าของ $a+b$ คือ 28
