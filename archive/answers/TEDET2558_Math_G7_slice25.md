# คำถาม

25. เฟือง A, B และ C วางสบกัน ดังรูป โดยเฟือง A มีฟัน 30 ซี่ เฟือง B มีฟัน 45 ซี่ และเฟือง C มีฟัน 50 ซี่ ![illustration](../images/TEDET2558_Math_G7_slice25_crop_0.png) ถ้าเฟือง A หมุนไป 15 รอบ จงหาว่าเฟือง C จะหมุนไปกี่รอบ

# คำอธิบายและวิธีทำ

หลักการทำงานของเฟืองที่ขบกัน (connected gears) คือ จำนวนฟันของเฟืองคูณกับจำนวนรอบที่หมุน จะต้องมีค่าเท่ากันเสมอ เนื่องจากในแต่ละรอบที่หมุน ฟันของเฟืองแต่ละอันจะต้องผ่านจุดที่ขบกันเป็นจำนวนเท่าๆ กัน ดังนั้นเราสามารถเขียนความสัมพันธ์ได้ว่า $N_A \times R_A = N_B \times R_B = N_C \times R_C$ โดยที่ $N$ คือจำนวนฟัน และ $R$ คือจำนวนรอบ จากโจทย์ เฟือง A มีจำนวนฟัน $N_A = 30$ และหมุนไป $R_A = 15$ รอบ ดังนั้นผลรวมของจำนวนฟันที่ผ่านจุดขบกันคือ $30 \times 15 = 450$ ฟัน เนื่องจากเฟือง A ขบกับเฟือง B และเฟือง B ขบกับเฟือง C การส่งผ่านแรงจะดำเนินต่อเนื่องกันไป ดังนั้นเฟือง C จะต้องมีจำนวนฟันที่ผ่านจุดขบกันเท่ากับ 450 ฟันเช่นกัน โดยเฟือง C มีจำนวนฟัน $N_C = 50$ เราสามารถหาจำนวนรอบของเฟือง C ($R_C$) ได้จากสมการ $N_A \times R_A = N_C \times R_C$ แทนค่าจะได้ $30 \times 15 = 50 \times R_C$ ซึ่งก็คือ $450 = 50 \times R_C$ ดังนั้น $R_C = 450 / 50 = 9$ รอบ
