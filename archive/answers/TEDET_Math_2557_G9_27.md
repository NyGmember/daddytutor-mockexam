# คำถาม

27. จงหาว่ามีคู่อันดับ $(x,y)$ ของจำนวนเต็มบวกทั้งหมดกี่คู่ที่ $xy$ มีค่าเป็น 10 เท่าของ $x + y$

# คำอธิบายและวิธีทำ

จากเงื่อนไขในโจทย์ $x, y$ เป็นจำนวนเต็มบวก และสอดคล้องกับสมการ:
$$xy = 10(x + y)$$

---

### 1. จัดรูปสมการ:
$$xy = 10x + 10y$$
$$xy - 10x - 10y = 0$$

บวก $100$ ทั้งสองข้างของสมการ เพื่อแยกตัวประกอบแบบกำลังสอง:
$$xy - 10x - 10y + 100 = 100$$
$$(x - 10)(y - 10) = 100$$

---

### 2. หาคู่อันดับ $(x,y)$ ที่เป็นจำนวนเต็มบวก:
ให้ $A = x - 10$ และ $B = y - 10$ จะได้สมการ:
$$A \times B = 100$$

เนื่องจาก $x$ และ $y$ ต้องเป็นจำนวนเต็มบวก ($x \ge 1, y \ge 1$):
* $A = x - 10 \ge -9$
* $B = y - 10 \ge -9$

หาก $A$ และ $B$ เป็นจำนวนเต็มลบ:
* เนื่องจาก $A, B \ge -9$ ค่าผลคูณที่เป็นบวกได้มากที่สุดของสองจำนวนในเซต $\{-9, -8, \dots, -1\}$ คือ $(-9) \times (-9) = 81$ ซึ่งไม่มีทางได้ $100$
* ดังนั้น $A$ และ $B$ ไม่สามารถเป็นจำนวนเต็มลบได้

ดังนั้น $A$ และ $B$ ต้องเป็นจำนวนเต็มบวกเท่านั้น
จำนวนคู่อันดับ $(x,y)$ ที่เป็นจำนวนเต็มบวกจึงเท่ากับจำนวนของตัวหารที่เป็นบวกทั้งหมดของ $100$

---

### 3. หาจำนวนตัวหารของ $100$:
เขียน $100$ ในรูปของตัวประกอบเฉพาะ:
$$100 = 2^2 \times 5^2$$

จำนวนตัวหารที่เป็นบวกทั้งหมดของ $100$ คือ:
$$(2 + 1)(2 + 1) = 3 \times 3 = 9 \text{ ตัว}$$

ซึ่งได้แก่ คู่อันดับ $(A, B)$ และค่า $(x, y)$ ที่สมนัยกัน ดังนี้:
1. $A = 1, B = 100 \implies (x, y) = (11, 110)$
2. $A = 2, B = 50 \implies (x, y) = (12, 60)$
3. $A = 4, B = 25 \implies (x, y) = (14, 35)$
4. $A = 5, B = 20 \implies (x, y) = (15, 30)$
5. $A = 10, B = 10 \implies (x, y) = (20, 20)$
6. $A = 20, B = 5 \implies (x, y) = (30, 15)$
7. $A = 25, B = 4 \implies (x, y) = (35, 14)$
8. $A = 50, B = 2 \implies (x, y) = (60, 12)$
9. $A = 100, B = 1 \implies (x, y) = (110, 11)$

ดังนั้น มีคู่อันดับ $(x,y)$ ของจำนวนเต็มบวกทั้งหมด **9** คู่
