# คำอธิบายและวิธีทำ

กำหนดให้ความยาวด้านของรูปสี่เหลี่ยมจัตุรัส $ABCD$ เท่ากับ $40$ หน่วย 
และมีจุด $E, F, G, H$ เป็นจุดกึ่งกลางของด้าน $AB, BC, CD, DA$ ตามลำดับ
แบ่งรูปสี่เหลี่ยมจัตุรัสใหญ่ออกเป็น 4 ส่วนย่อยขนาด $20 \times 20$ หน่วย:
1. จัตุรัสย่อยบนซ้าย $AEPH$
2. จัตุรัสย่อยบนขวา $HPDG$
3. จัตุรัสย่อยล่างซ้าย $EBPF$
4. จัตุรัสย่อยล่างขวา $PFCG$

เราสามารถหาพื้นที่ของส่วนที่แรเงานอกเหนือจากระนาบแกนได้โดยพิจารณาแยกทีละจัตุรัสย่อยดังนี้:

### 1. พื้นที่แรเงาในจัตุรัสย่อยบนซ้าย ($AEPH$):
* มีครึ่งวงกลมที่มี $AB$ เป็นเส้นผ่านศูนย์กลาง (รัศมี $R = 20$) และมีจุดศูนย์กลางที่ $E$
* ส่วนโค้งของครึ่งวงกลมนี้ลากจากจุด $A$ ไปยังจุด $P$ ซึ่งเป็นรูปพัดเสี้ยวหนึ่งในสี่ของวงกลมรัศมี $20$
* ส่วนที่แรเงาคือบริเวณระหว่างเส้นทแยงมุม $AP$ และส่วนโค้งวงกลม $AP$
* คำนวณได้ดังนี้:
  $$\text{พื้นที่แรเงาส่วนที่ 1} = \text{พื้นที่เซกเตอร์ } EAP - \text{พื้นที่สามเหลี่ยม } EAP$$
  $$\text{พื้นที่แรเงาส่วนที่ 1} = \frac{1}{4}\pi (20)^2 - \frac{1}{2}(20)(20) = 100\pi - 200$$

### 2. พื้นที่แรเงาในจัตุรัสย่อยบนขวา ($HPDG$):
* บริเวณที่แรเงาคือรูปสามเหลี่ยมมุมฉาก $HPG$ ที่มี $HP = 20$ และ $PG = 20$
* คำนวณได้ดังนี้:
  $$\text{พื้นที่แรเงาส่วนที่ 2} = \frac{1}{2} \times 20 \times 20 = 200$$

### 3. พื้นที่แรเงาในจัตุรัสย่อยล่างซ้าย ($EBPF$):
* มีส่วนโค้งของครึ่งวงกลมเส้นผ่านศูนย์กลาง $AB$ ลากจากจุด $P$ ไปจุด $B$ เป็นพัดเสี้ยวหนึ่งในสี่ของวงกลมรัศมี $20$
* บริเวณที่แรเงาคือพื้นที่ด้านนอกส่วนโค้งนี้แต่อยู่ภายในจัตุรัสย่อย $EBPF$
* คำนวณได้ดังนี้:
  $$\text{พื้นที่แรเงาส่วนที่ 3} = \text{พื้นที่จัตุรัส } EBPF - \text{พื้นที่เซกเตอร์ } EPB$$
  $$\text{พื้นที่แรเงาส่วนที่ 3} = 20^2 - \frac{1}{4}\pi (20)^2 = 400 - 100\pi$$

### 4. พื้นที่แรเงาในจัตุรัสย่อยล่างขวา ($PFCG$):
* มีครึ่งวงกลมที่มี $FP = 20$ เป็นเส้นผ่านศูนย์กลาง (รัศมี $r = 10$ และมีจุดศูนย์กลางอยู่ที่จุดกึ่งกลางของ $FP$)
* มีเส้นทแยงมุม $PC$ ตัดผ่านครึ่งวงกลมนี้
* จากรูป ส่วนที่แรเงาคือเซกเมนต์ของวงกลมขนาดเล็กที่อยู่เหนือเส้นทแยงมุม $PC$
* โดยที่เส้นทแยงมุม $PC$ ตัดกับแนวโค้งของครึ่งวงกลมเล็กที่จุดพิกัดทำให้เกิดมุมที่จุดศูนย์กลางของเซกเตอร์เป็นมุมฉาก ($90^\circ$)
* คำนวณพื้นที่เซกเมนต์นี้ได้ดังนี้:
  $$\text{พื้นที่แรเงาส่วนที่ 4} = \text{พื้นที่เซกเตอร์เล็ก} - \text{พื้นที่สามเหลี่ยมของเซกเตอร์เล็ก}$$
  $$\text{พื้นที่แรเงาส่วนที่ 4} = \frac{1}{4}\pi (10)^2 - \frac{1}{2}(10)(10) = 25\pi - 50$$

---

### รวมพื้นที่แรเงาทั้งหมด:
$$\text{พื้นที่แรเงาทั้งหมด} = (100\pi - 200) + 200 + (400 - 100\pi) + (25\pi - 50)$$
$$\text{พื้นที่แรเงาทั้งหมด} = 350 + 25\pi$$

เมื่อเทียบรูปกับ $a + b\pi$ ตารางหน่วย:
* $a = 350$
* $b = 25$

คำนวณหาค่า $a + b$:
$$a + b = 350 + 25 = 375$$

ดังนั้น ค่าของ $a + b$ เท่ากับ **375**
