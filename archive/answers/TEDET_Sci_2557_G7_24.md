# คำถาม

24. จากภาพ นักเรียนคนหนึ่งเดินทางจากจุด A ไปยังจุด C โดยผ่านจุด B ใช้เวลา 20 วินาที

# คำอธิบายและวิธีทำ

จากโจทย์ นักเรียนเดินทางจาก $A \rightarrow B \rightarrow C$ โดยใช้เวลาทั้งหมด $t = 20\text{ วินาที}$

1. **หาอัตราเร็ว (Speed):**
   - อัตราเร็วคือ ระยะทางทั้งหมดตามแนวทางการเคลื่อนที่จริง หารด้วยเวลาที่ใช้
   - ระยะทางที่เคลื่อนที่จริง $s = AB + BC = 80\text{ เมตร} + 60\text{ เมตร} = 140\text{ เมตร}$
   - ดังนั้น อัตราเร็วเฉลี่ย ($v$) คือ:
     $$v = \frac{s}{t} = \frac{140\text{ เมตร}}{20\text{ วินาที}} = 7\text{ m/s}$$

2. **หาความเร็ว (Velocity):**
   - ความเร็วคือ ขนาดของการกระจัดจากจุดเริ่มต้นตรงไปยังจุดสุดท้าย หารด้วยเวลาที่ใช้
   - จุดเริ่มต้นคือ $A$ และจุดสุดท้ายคือ $C$ การกระจัดคือเส้นตรงลากตรงจาก $A \rightarrow C$
   - จากภาพ ขนาดของการกระจัด $\Delta x = 100\text{ เมตร}$
   - ดังนั้น ขนาดของความเร็วเฉลี่ย ($\vec{v}$) คือ:
     $$\vec{v} = \frac{\Delta x}{t} = \frac{100\text{ เมตร}}{20\text{ วินาที}} = 5\text{ m/s}$$

ดังนั้น:
- อัตราเร็ว = $7\text{ m/s}$
- ความเร็ว = $5\text{ m/s}$

ซึ่งสอดคล้องกับตัวเลือก **4**
