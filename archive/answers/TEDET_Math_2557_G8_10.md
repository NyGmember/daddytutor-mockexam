# คำถาม

10. ถ้า $a$ และ $b$ เป็นจำนวนเต็มที่ทำให้
$$(6x^2y + 15xy^2) \div 3xy + (3xy - 18y^2 - 24y) \div (-3y) = x + ay + b$$
โดยที่ $x$ และ $y$ เป็นจำนวนจริงใด ๆ ที่ไม่เป็นศูนย์
จงหาค่าของ $a + b$

# คำอธิบายและวิธีทำ

เราต้องการหาค่าของ $a + b$ จากสมการที่กำหนดให้:
$$(6x^2y + 15xy^2) \div 3xy + (3xy - 18y^2 - 24y) \div (-3y) = x + ay + b$$

1. **หารพจน์แรกด้วย $3xy$:**
   $$\frac{6x^2y + 15xy^2}{3xy} = \frac{6x^2y}{3xy} + \frac{15xy^2}{3xy}$$
   $$= 2x + 5y$$

2. **หารพจน์ที่สองด้วย $-3y$:**
   $$\frac{3xy - 18y^2 - 24y}{-3y} = \frac{3xy}{-3y} - \frac{18y^2}{-3y} - \frac{24y}{-3y}$$
   $$= -x + 6y + 8$$

3. **นำผลลัพธ์ทั้งสองส่วนมาบวกกัน:**
   $$\text{ฝั่งซ้ายของสมการ} = (2x + 5y) + (-x + 6y + 8)$$
   $$= (2x - x) + (5y + 6y) + 8$$
   $$= x + 11y + 8$$

4. **เทียบสัมประสิทธิ์กับฝั่งขวาของสมการ ($x + ay + b$):**
   $$x + 11y + 8 = x + ay + b$$
   
   จะได้:
   * $a = 11$
   * $b = 8$

5. **หาค่าของ $a + b$:**
   $$a + b = 11 + 8 = 19$$

ดังนั้น ค่าของ $a + b$ คือ **19**
