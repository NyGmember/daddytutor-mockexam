# คำอธิบายและวิธีทำ

To solve this problem, we apply Newton's second law ($F = ma$) to each object. Let $a$ be the uniform acceleration. 

1. Analyzing the objects: 
- Object A: Mass $m$, Friction $f_A = \mu mg$
- Object B: Mass $2m$, Friction $f_B = \frac{1}{2}\mu(2mg) = \mu mg$
- Pendulum: Mass $3m$, Gravity $g$

2. Forces on the system (as one unit): 
Total Force = $3mg - f_A - f_B = 3mg - \mu mg - \mu mg = 3mg - 2\mu mg$. 
Total mass = $m + 2m + 3m = 6m$. 
Using $F_{net} = Ma$: $3mg - 2\mu mg = 6ma 
\Rightarrow 3g - 2\mu g = 6a 
\Rightarrow 2\mu g = 3g - 6a 
\Rightarrow \mu = \frac{3g - 6a}{2g} = \frac{3}{2}(1 - \frac{a}{g})$. This matches statement (ข).

3. Checking statement (ก): 
Net force on A ($F_A$) = $T_A - f_A = T_A - \mu mg$. 
Net force on B ($F_B$) = $T_B - T_A - f_B = T_B - T_A - \mu mg$. 
Since $F = ma$, $F_A = ma$ and $F_B = (2m)a = 2ma$. 
Thus, $F_B = 2F_A$. Statement (ก) is correct.

4. Checking statement (ค): 
From $F_B = 2ma$, we have $T_B - T_A - \mu mg = 2ma$. 
From $F_A = ma$, we have $T_A - \mu mg = ma \Rightarrow T_A = ma + \mu mg$. 
Substituting $T_A$ into the equation for $F_B$: $T_B - (ma + \mu mg) - \mu mg = 2ma \Rightarrow T_B = 3ma + 2\mu mg$. 
Compare $2T_A$ and $3T_B$: $2(ma + \mu mg) = 2ma + 2\mu mg$ while $3(3ma + 2\mu mg) = 9ma + 6\mu mg$. They are not equal. Statement (ค) is incorrect. 

Therefore, statements (ก) and (ข) are correct.
