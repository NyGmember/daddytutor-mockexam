# คำถาม

23. กำหนดจุด $A(2,2)$ และ $B(4,1)$ ดังรูป

[![illustration](../images/TEDET_Math_2557_G9_23_crop_0.png)]

เมื่อโยนลูกเต๋า 2 ลูกหนึ่งครั้ง แล้วให้ $a$ และ $b$ เป็นแต้มบนหน้าของลูกเต๋าแต่ละลูกที่ได้ ถ้าความน่าจะเป็นที่เส้นตรง $y = \frac{b}{a}x$ ตัดกับส่วนของเส้นตรง $AB$ มีค่าเท่ากับ $\frac{n}{m}$ จงหาค่าของ $m + n$ (เมื่อ $\frac{n}{m}$ อยู่ในรูปเศษส่วนอย่างต่ำ)

# คำอธิบายและวิธีทำ

เส้นตรง $y = \frac{b}{a}x$ เป็นเส้นตรงที่ผ่านจุดกำเนิด $O(0,0)$ และมีค่าความชันเท่ากับ $k = \frac{b}{a}$

เพื่อให้เส้นตรง $y = kx$ ตัดกับส่วนของเส้นตรง $AB$ ที่เชื่อมระหว่างจุด $A(2,2)$ และ $B(4,1)$ ค่าความชัน $k$ ของเส้นตรงนี้จะต้องอยู่ระหว่างความชันของเส้นตรง $OB$ และความชันของเส้นตรง $OA$ นั่นคือ:
$$\text{ความชันของ } OB \le k \le \text{ความชันของ } OA$$

---

### 1. หาความชันของ $OA$ และ $OB$:
* ความชันของ $OA$ เชื่อมระหว่าง $(0,0)$ และ $A(2,2)$:
  $$\text{ความชันของ } OA = \frac{2 - 0}{2 - 0} = 1$$
* ความชันของ $OB$ เชื่อมระหว่าง $(0,0)$ และ $B(4,1)$:
  $$\text{ความชันของ } OB = \frac{1 - 0}{4 - 0} = \frac{1}{4} = 0.25$$

ดังนั้น เงื่อนไขที่เส้นตรงจะตัดกับส่วนของเส้นตรง $AB$ คือ:
$$\frac{1}{4} \le \frac{b}{a} \le 1$$

---

### 2. หาจำนวนเหตุการณ์ที่สอดคล้องทั้งหมดจากลูกเต๋า 2 ลูก:
การโยนลูกเต๋า 2 ลูก ผลลัพธ์ทั้งหมดที่เป็นไปได้มี $6 \times 6 = 36$ แบบ
ให้ $a$ เป็นแต้มจากลูกเต๋าลูกแรก และ $b$ เป็นแต้มจากลูกเต๋าลูกที่สอง ($a, b \in \{1, 2, 3, 4, 5, 6\}$):

* **กรณี $a = 1$:**
  $\frac{1}{4} \le b \le 1 \implies b = 1$ (มี 1 แบบ: $(1,1)$)

* **กรณี $a = 2$:**
  $\frac{2}{4} \le b \le 2 \implies 0.5 \le b \le 2 \implies b \in \{1, 2\}$ (มี 2 แบบ: $(2,1), (2,2)$)

* **กรณี $a = 3$:**
  $\frac{3}{4} \le b \le 3 \implies 0.75 \le b \le 3 \implies b \in \{1, 2, 3\}$ (มี 3 แบบ: $(3,1), $(3,2), (3,3)$)

* **กรณี $a = 4$:**
  $\frac{4}{4} \le b \le 4 \implies 1 \le b \le 4 \implies b \in \{1, 2, 3, 4\}$ (มี 4 แบบ: $(4,1), (4,2), (4,3), (4,4)$)

* **กรณี $a = 5$:**
  $\frac{5}{4} \le b \le 5 \implies 1.25 \le b \le 5 \implies b \in \{2, 3, 4, 5\}$ (มี 4 แบบ: $(5,2), (5,3), (5,4), (5,5)$)

* **กรณี $a = 6$:**
  $\frac{6}{4} \le b \le 6 \implies 1.5 \le b \le 6 \implies b \in \{2, 3, 4, 5, 6\}$ (มี 5 แบบ: $(6,2), (6,3), (6,4), (6,5), (6,6)$)

---

### 3. รวมจำนวนเหตุการณ์ที่สอดคล้อง:
จำนวนเหตุการณ์ที่สอดคล้องทั้งหมด $= 1 + 2 + 3 + 4 + 4 + 5 = 19$ แบบ

ดังนั้น ความน่าจะเป็นที่เส้นตรงจะตัดกับส่วนของเส้นตรง $AB$ คือ:
$$\frac{n}{m} = \frac{19}{36}$$

เนื่องจาก $\frac{19}{36}$ อยู่ในรูปเศษส่วนอย่างต่ำแล้ว จะได้:
* $n = 19$
* $m = 36$

หาค่า $m + n$:
$$m + n = 36 + 19 = 55$$
